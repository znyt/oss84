　　无尽高空之上，茫茫云海之间，一道剑光疾驰而过，刺穿了云海，掀起一阵狂风。
<br/>　　剑光之中，一名青年盘膝闭目而坐。
<br/>　　青年，自然便是苏莫。
<br/>　　他一边赶路，一边静气凝神，融合神魂和武魂。
<br/>　　这个过程很缓慢，他需要不断的尝试，不断的去沟通，不断的让两者形成彼此依存。
<br/>　　距离与冷江约战之日，已经近在咫尺了，他必须要抓紧时间。
<br/>　　在苏莫赶路之时，苍穹神宫的几位九宫之主，也踏上了征程，走向这全新的世界。
<br/>　　他们不知道未来会怎么样，也许是康庄大道，崛起于古周星，扬名于古灵星河。
<br/>　　但也有可能陨落于这陌生的世界，不过，他们并不后悔。
<br/>　　几人都是天才，自然不想束缚再小小的途山镇，那不是他们的路。
<br/>　　不过，有一点在几人心中都是一样的，那就是他们来自于苍穹神宫，无论未来如何，无论他们走到多远，苍穹神宫永远都是他们的根。
<br/>　　时间匆匆而逝。
<br/>　　不知道过了多久之后，苏莫睁开了眼眸。
<br/>　　他的前方，巨大的陨落星辰出现在了视线之中，庞大的体积仿佛隔绝了天与地。
<br/>　　苏莫叹了口气，今日就是约战之期，然而他还是没有进化战魂。
<br/>　　不是他遇到了困难，而是时间实在太紧迫了。
<br/>　　阎魔鬼圣将武魂进化战魂的要点，全部告诉了他，这些知识应该不会有假，但短短几日之内，想要进化战魂，突破到武帝之境，还是太难了。
<br/>　　他已经有了将要融合的感觉，但始终无法成功。
<br/>　　“就算是没有突破，也要一战！”苏莫心中呢喃自语，根本不愿意退缩。
<br/>　　他虽然平常为人低调，但骨子里比任何人都有要傲，这是这么多年来同代无敌，而铸就的无敌之心。
<br/>　　面对同代武者的挑战，他骨子里不允许自己退缩。
<br/>　　很快，苏莫便进入了神武学府，走进了南苑之中。
<br/>　　战界，是神武学府弟子比斗的地方，入口就在通宇大殿，苏莫早已打听清楚了。
<br/>　　回到学府以后，他一刻不停，立刻来到通宇大殿，然后从光门中进入了战界。
<br/>　　眼前空间转换，苏莫发现自己来到了一片大海之上。
<br/>　　大海一望无际，海水涛涛，滚滚涌动，传出轰隆隆的声响。
<br/>　　抬眼看去，只见远处的海面上坐落着十几座岛屿。
<br/>　　这些岛屿呈环形围绕在一起，形成了一个巨大的圆形。
<br/>　　而在圆形的中央，一片方圆万里的海面上，此刻正有两人在战斗。
<br/>　　狂暴的战斗，掀起方圆万里的海面上狂风大作，巨浪滔天，发出惊天动地的爆响。
<br/>　　而在周围的十几座岛屿上，同样有不少人，均是在观看战斗，时而有嘈杂的声音响起。
<br/>　　这里便是神武学府弟子的比斗之所，十几座岛屿为看台，中央方圆万里的海域，便是战台。
<br/>　　苏莫心中惊讶，没想到除了他之外，还有人在战界比斗。
<br/>　　随即，苏莫向其中一座岛屿飞去，他还没飞到岛屿上，便感受到了一缕凌厉的目光，落在了自己的身上。
<br/>　　这道目光凌厉万分，杀机森然，让人通体生寒。
<br/>　　苏莫转头，寻着目光望了过去，便看到了另一座岛屿上的冷江。
<br/>　　四目相对，迸射出无形的火光。
<br/>　　冷江冷冷一笑，露出一口洁白的牙齿。
<br/>　　苏莫面无表情，对方的杀机毫不掩饰，看来是准备出手杀他了！
<br/>　　在冷江的身边，站着华服青年何传岩，其同样是冷冷一笑，眸子露出戏虐之色。
<br/>　　他们已经来了几个时辰，一直没见苏莫现身，本还担心苏莫不会来了。
<br/>　　现在见苏莫终于来了，心中也就放心了！
<br/>　　唰！
<br/>　　片刻之间，苏莫身形飞落在了岛屿之上。
<br/>　　站在岛上一座小山坡下，苏莫目光望向中央海面之上的战斗。
<br/>　　战斗的双方是两名青年，实力都很强，修为应该都在武帝境三重左右，战况极为激烈。
<br/>　　不过，两人似乎势均力敌，谁也难以拥有压倒性的优势。
<br/>　　“苏莫，以汝现在的修为，基本不可能获胜！”阎魔鬼圣的声音，在苏莫的脑海中响起，仿佛极不看好苏莫。
<br/>　　他也是担心苏莫被杀死，如此的话，他怕是也要完了。
<br/>　　在这神武学府之中，他根本不可能逃出去。
<br/>　　“那你有能让我获胜的办法吗？”苏莫问道，对方作为武圣强者，说不定有什么厉害的绝技，能让他翻盘。
<br/>　　“吾的确有办法，不过现在已经来不及了！”阎魔鬼圣叹道。
<br/>　　苏莫闻言，嘴角忍不住抽搐了几下，对方有办法让他获胜，居然不早说，现在才跟他说来不及了。
<br/>　　“什么办法？你倒是说说看！”苏莫阴沉着脸说道。
<br/>　　“算了，不提也罢！若是此次你能活下来，吾可以教你！”阎魔鬼圣道。
<br/>　　“卧槽！”苏莫心中微怒，这个老家伙是在耍他吧！
<br/>　　不过，虽然心中有些怒气，他也不敢将对方怎么样，他以后还需要对方的指导呢。
<br/>　　嗖！嗖！
<br/>　　这时，两道破空声响起，一男一女两道身影从另一座小岛上飞了过来，落在了苏莫的身前。
<br/>　　来人正是聂乾和韩盈。
<br/>　　“哎！苏兄，你还是来了！”聂乾叹了口气，面上露出了一丝担忧。
<br/>　　在他看来，苏莫和冷江一战，几乎是必败无疑，胜利的希望非常的渺茫。
<br/>　　“苏兄，你有几分把握能保持不败？”韩盈轻声问道。
<br/>　　韩盈的话问的非常有意思，她并不是问苏莫有几分把握胜利，而是有几分把握不败！
<br/>　　因为苏莫战胜冷江没有可能，只要能维持不败，就可以了。
<br/>　　到时候时间一长，便能抽身而退。
<br/>　　“没有把握！”苏莫无奈的摇了摇头。
<br/>　　“这……！”两人闻言愕然，没有把握还敢来？
<br/>　　“苏兄，你要懂得变通啊！没必要与冷江死磕！”聂乾告诫道。
<br/>　　“是啊！留得青山在，不怕没材烧，以你的天赋，估计一年内就能赶超冷江了！”韩盈亦是赞同道。
<br/>　　两人都不希望苏莫与冷江一战，免得白白丢了性命。
<br/>　　苏莫闻言叹了口气，他也知道两人是好意，但他虽然没有不败的把握，但保命的把握还是有的。
<br/>　　换言之，他就是败了，对方想杀他，也不会太容易。
<br/>　　“两位不必担心！你们自便，我先调息片刻！”
<br/>　　苏莫摇了摇头，便不再多言，自顾自的盘膝而坐，准备继续尝试神魂和武魂融合。
<br/>　　他现在感悟已经颇深，神魂和武魂之间，仿佛已经彼此相融，但又仿似隔着一层无形的壁障，让他始终无法迈出那一步。
<br/>　　眸光扫过远处的十几座小岛，苏莫又看到了一个熟悉的身影。
<br/>　　只见其中一座小岛上，一棵古树之下，一身白衣飘飘的木璃导师，静静的伫立于此。
<br/>　　苏莫心中清楚，对方来此，应该是来观看他与冷江一战的。
<br/>(本章完)
