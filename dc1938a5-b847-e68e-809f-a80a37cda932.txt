<br/>				陈默的小舅名叫付慧文，已经四十多岁了。不过对陈默他们家很好，就像陈默去上大学的费用，前两年都是借两个舅舅家里的钱，所以现在外甥有了能力，当然要找娘舅。娘舅亲、娘舅亲，在农村可不是说着玩的，绝对的亲人。<br/>　　<br/>　　自小陈默和陈辉他们兄弟两个，还有老姐都和两个舅舅亲近。只要有时间，两个舅舅都会过来看陈默他们几个，而且每次过来都会带些好吃的。所以那个时候，盼着舅舅来家里，就是几个孩子最喜欢的事情。<br/>　　<br/>　　早上还不到七点，小舅就骑着一辆摩托车来到了地头的食堂，开始给何老他们做早饭。一点小米粥，几个小馒头，一碟咸菜，两个青菜，还有两个土鸡蛋，还有一些其他的糕点什么的，虽然不是很丰盛，但是在陈默看来也不错了，都是农村自家人吃的东西，绝对的无污染的纯绿色食物。<br/>　　<br/>　　何老和吴老的一日三餐，陈默都包了。一个月10万元，当然要包饭菜的。再说了自己种了那么多的蔬菜，也算是物尽其用。<br/>　　<br/>　　所以小舅的任务就是给吴老和何老以及两个跟班做一天三餐，不仅要做好，还要做好吃。当然，这个做好和做好吃，不能和那些米其林或者五星级之类的饭店相比较，就是些农村的家常菜。现在人少，小舅一个人就成，等以后要是人多了，那么就让小舅管人，然后招人。<br/>　　<br/>　　每个月，陈默直接就给小舅六千元的工资，开多了害怕小舅拒绝。而且工资是从丁兰花这里支取，那么多了也担心其他人有看法。所以就定了个六千元，等发奖金的时候多发就是了。<br/>　　<br/>　　小舅先前给镇上的一家饭店做饭，也就拿个三千元的死工资，其他什么都没有。陈默这里可是每个月六千，而且还有奖金什么的，当然要过来了，再说外甥叫他，他还不能来帮忙么？<br/>　　<br/>　　陈默找到小舅后，也是交代一些事情。从今天开始，给两个老人准备的伙食要改改。早上可以清淡一些，也就是点青菜，小米粥之类的，转着花样做。但是中餐和晚餐就要有肉食，必须要熬制骨头汤或者鸡汤之类。<br/>　　<br/>　　药已经开始从今天吃，那么这两个人的饭量必定会增加不少，而且混合着少量的溪水，能改善体质等等，营养就要跟的上才行。等自己从西市回来后，准备配置几个药膳，给两个老人吃，这样营养什么的就能跟上。至于药材，乾坤珠内现在种植的可是很多的。<br/>　　<br/>　　另外就是现在还和地里工人的大食堂凑在一起，主要是没有地方，但是也就今天一天。陈默让集装箱厂家，今天送来一套厨房，包括房子和里面的厨房用具等等，全部都是齐全的。<br/>　　<br/>　　房子的钱已经全部都结清了，所以接货就成。<br/>　　<br/>　　然后带着大海和小舅，确定了一下厨房要放在那里，就放在院子的宾客用房的旁边，那里有很大的空地，都是水泥地，以前按照临时用房的尺寸的的地基，防备着房子不够了增加，只要放上集装箱房子就好。<br/>　　<br/>　　大海负责让人将水管电线全部供应到位，等房子放好以后就能直接接上。至于说水电工，让大海找明叔就成。<br/>　　<br/>　　小舅负责清点房子里的物件，看看和清单上是否一致。另外就是需要添置什么，写一个清单，和大海开车去镇上买，钱从丁兰花哪里支取就成。<br/>　　<br/>　　还有就是大黄和大灰的吃饭的事情，全部都交给小舅了，让他做饭的时候做多一些，然后给两只狗吃，另外就是留下了一袋羊奶，冲给狗喝。<br/>　　<br/>　　交代完成后，陈默就拉着青菜直接去了西市。这次采摘的青菜，他多准备了些，因为不一定生意就能谈成，所以万一要是不成，也要考虑其他出路。<br/>　　<br/>　　正开车在路上走的时候，接到了自己的弟弟陈辉的电话。所以将车停到路边，直接和弟弟陈辉通过电话聊了起来。<br/>　　<br/>　　陈辉在电话中感谢了哥哥陈默打给他的生活费，并且询问了一下哥哥的近况，倒是没有想到自己的哥哥能够回到家乡创业，虽然父亲陈建国说过一嘴，但是没有陈默解释的清楚。<br/>　　<br/>　　陈辉除了鼓励以外也就没有其他说的了，既然老哥选择了这条路，那么作为兄弟的只有支持。另外想着哥哥正在创业，可能手头不是很方便，所以每个月的生活费可以减少些。<br/>　　<br/>　　上个月陈默给陈辉打过去了一千元，可以说完全足够他两个月的花费了，所以现在快到打生活费的时间了，就给哥哥说一下，上次打过来的一千元还有将近600元，根本就没有花完，所以这个月就可以不打生活费了，也好减轻一下老哥的负担。<br/>　　<br/>　　“怎么，你一个月就花500元？”陈默有些赫然，想起自己在大一的时候，每个月生活费用就只有300元，每天也就是青菜豆腐，一个馒头外加免费菜汤。<br/>　　<br/>　　感觉那个时候眼睛都是绿的，见了谁手上有吃的，眼光都能让人害怕。宿舍里其他三个人，都是被陈默的眼光吓过。好在自从打工以后，这个情况就好多了，正是长身体的时候，肯定能吃啊。<br/>　　<br/>　　“老哥，一个月500元当然不够了，但是你兄弟我可以打工的啊，所以能省很多钱的呢。”陈辉笑着说道，口气中也隐隐有些自豪，自小数学就比较好，所以能找个辅导初高中数学的家教，每个月能赚不少钱，补贴一下自己的肚皮是没有问题。<br/>　　<br/>　　“好吧，你厉害行了吧！但是钱我还是会打的，现在我也不缺那点钱，你放心吧，创业的资金我都准备好的。”陈默说道，对已兄弟的关心，心里很暖，但是省钱却没有必要。决定了，等下给弟弟转过去一万元，也算是一份心意。<br/>　　<br/>　　他本来并不准备给弟弟打这么多钱，每个月就基本的生活费就可以了，男孩子么，只要吃饭足够了就成。而且钱打过去多了，有时候一些事情就不可控了，所以还是尽量按月打过去，然后够陈辉吃饭学费就成。<br/>　　<br/>　　“哈！老哥，你这是发财了还是怎么了，我怎么听着有股浓浓的土豪味道？”陈辉也是个快乐的二哈青年，有的时候说话就带着善意的调侃。<br/>　　<br/>　　“你猜对了！我现在就是个土豪，电话里说着不方便，等你回来后咱们详细聊！”陈默笑着说道。<br/>　　<br/>　　“那就这样吧！等我回来咱们再聊，你也详细给我说说你的事情，我很好奇的。老爸说什么事情都是一句话，你回来自己看！我去，就不能给我好好解释解释，还说电话费贵，少说话。”陈辉着实的埋怨了一下老爸。<br/>　　<br/>　　“哈哈，老爸的脾气你又不是不知道。不过你这么说老爸，我是不是给老爸他也说一下你调侃他的事情啊！”<br/>　　<br/>　　“老哥，你过分了啊，等我回去你等着，我要好好跟你较量一下，看你还给老爸打小报告！”陈辉说道。<br/>　　<br/>　　“嘁！你以为我怕你啊，就你那个小体格，还不到一米八，每次都在我手底下坚持不了两回，就知道放大话！”陈默鄙视了一番自己的弟弟陈辉。<br/>　　<br/>　　“老哥你够了啊！揭人不揭短，骂人不骂娘！你这样说，兄弟都没得做了！”陈辉说道。<br/>　　<br/>　　“哈哈！好了好了，不要贫嘴了，我还有事，不说了！”<br/>　　<br/>　　“那行，等我放暑假回家的时候好好聊！”<br/>　　<br/>　　“好，再见！”<br/>　　<br/>　　“保重，老哥！”<br/>　　<br/>　　陈默挂断电话，发动汽车开往西市。<br/>　　<br/>　　如果感觉不错，请推荐和收藏一下，顺便有月票的投个月票，谢谢了！<br/>　　<br/>　　7<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
