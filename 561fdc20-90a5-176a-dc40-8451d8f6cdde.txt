　　武胜国度，秦皇国度等，不过只是小地方，这北邙城，才算是真正九域中的繁华之地。
<br/>　　北邙城，这还是楚风眠第一次踏入其中。
<br/>　　曾经前世的楚风眠，九域中的各大城池，他也都曾经去过，但是那个时候去的楚风眠，只是一个旁观者。
<br/>　　而如今，这一次来到北邙城，楚风眠的心中却是截然不同的感觉。
<br/>　　他如今已经有着实力，不再是一个旁观者，未来他更是要从这北邙城中崛起。
<br/>　　这北邙学院，便是他下一个目标，楚风眠要在这北邙学院中，彻底崛起。
<br/>　　“这北邙城，虽然只是万年间崛起的城池，却是已经极为繁华了。”
<br/>　　楚风眠一行人，走出传送法阵，便是看着北邙城中的种种。
<br/>　　这北邙城，在万年之前，甚至还只是一个偏僻小城，但是如今却是已经成为了北域中最为庞大的城池。
<br/>　　这一切的原因，便是北邙学院造就的。
<br/>　　诺大的北邙城，其实只有这一个宗门，便是北邙学院。
<br/>　　楚风眠看着，就在这北邙城中，一眼望去便是有着无数青年才俊，御风境，甚至是圣者，都是比比皆是。
<br/>　　在一般偏远国度中足以称王称霸的胜者，在这北邙城，却是算不得什么，虽然不能说是满大街都是，一眼望去也可以看见几个。
<br/>　　不过生死境强者，却是就极为罕见了，真正能够从神力境，度过生死大劫踏入生死境的武者，终究还是不多。
<br/>　　生死境的强者，放在北邙学院中，都算是有些地位的了。
<br/>　　“这些人，都是怎么回事？”
<br/>　　楚风眠看着，不少的年轻武者，都是站在一处巨大无比的宫殿前，似乎是在排队，不知道是因为什么。
<br/>　　这些年轻武者，其中许多人都是资质极高之人，算得上是一些天才了。
<br/>　　看来应该都是来参加北邙学院考核的人。
<br/>　　“哦，这些人是在排队报名的，两日之后，便是参加考核的日子。”
<br/>　　拓跋鼎在一旁，听到楚风眠的话，开口道。
<br/>　　“不过少主不需要去跟这些人一起排队，我已经找好的关系，两日后直接去参加便是，不需要去核实身份。”
<br/>　　“恩。”
<br/>　　楚风眠点了点头。
<br/>　　留了这拓跋鼎一命，倒是的确有着不好的好处，起码有着一个地头蛇，楚风眠也不至于白白浪费很多时间。
<br/>　　楚风眠思考了一下，开口询问道。
<br/>　　“在这北邙城中，有没有什么买卖奇物的地方？”
<br/>　　如今楚风眠的实力，虽然已经达到瓶颈，但是他还必须要寻找一些对他有利的奇物，来帮助他突破神力境的时候动用。
<br/>　　有着天元神丹相助，楚风眠突破起来，应该没有那么苦难。
<br/>　　但是提前准备好，却是有备无患，可以保险一点。
<br/>　　“这北邙城中，购买奇物的地方，的确不少，不过对少主而言需要的，却是不多。”
<br/>　　拓跋鼎迟疑了一会，开口道。
<br/>　　“在这北邙城中，有这一处黑市，其中每日都有大量的交易，许多东西都是杀人易货的来的，但是却不知道来历，许多珍贵奇物都是因此贱卖了。”
<br/>　　“少主可以前去看看，也许可以碰到称心如意的东西。”
<br/>　　“可以去看看。”
<br/>　　楚风眠点了点头道。
<br/>　　这黑市，其实便是鱼龙混杂之地，其中无数奇物，鱼目混珠，也许会碰到很好的奇物，也许也会被骗去全身家当。
<br/>　　有的人在黑市中，便是眼力好，直接买到了极好的奇物，一下子发了大财，而有的则是直接赔的家底都没有了。
<br/>　　在其中凭借的，都是眼力。
<br/>　　楚风眠前世熟读了不少古籍，对眼力还是有着自信的，倒是可以前去碰碰运气。
<br/>　　“走。”
<br/>　　楚风眠直接动身，毫不犹豫。
<br/>　　虽然说是千里迢迢前来这北邙城的，但是以楚风眠等人的实力，就算是一年不眠不休都不会累，自然不用浪费时间去休息。
<br/>　　楚风眠一行人直接动身，前去这北邙城的黑市。
<br/>　　黑市，位于北邙城一处极为隐蔽的角落之中。
<br/>　　一路上道路错综复杂，若是么有拓跋鼎这个地头蛇，楚风眠根本不可能找得到。
<br/>　　“想要进入黑市，每人一千九鼎丹。”
<br/>　　来到黑市前，一尊看守便是直接拦住了楚风眠一行人，拓跋鼎熟悉无比，早已经是准备好了三千九鼎丹，直接便是送了上去。
<br/>　　楚风眠一行人，一步入到黑市之中，便是听见了不少的叫卖声音。
<br/>　　这黑市中，倒是并没有太多的秩序，其中许多摆摊的人，实力也都是极强的武者。
<br/>　　其中几人，楚风眠一眼望过去，便是知道这几人都是平日里杀人成性之人，身上不知道是沾染了不少血气。
<br/>　　他们杀人，便是将得到的财富在这黑市上卖，直接换成财富。
<br/>　　楚风眠扫视了一圈，倒是其中一个摊位前停了下来，指着其中一枚足有拳头大小的内丹，开口道。
<br/>　　“这是什么妖兽的内丹？”
<br/>　　楚风眠看着这内丹，眼神中露出了几分茫然，以他的阅历，居然是有些看不出这内丹，但是是属于什么妖兽，
<br/>　　这令楚风眠自己都是极为意外，他前世身为炼药宗师，对各种内丹自然都是极为熟悉的。
<br/>　　但是这内丹，楚风眠却是看不出来，似乎有些熟悉，却是不知道其中奥秘。
<br/>　　“你要买这内丹？不买别问。”
<br/>　　这摊主，是一尊神力境五重的老者，听到楚风眠的询问，不由的冷哼了一声说道。
<br/>　　“免得浪费时间。”
<br/>　　听到这老者的话，拓跋鼎勃然大怒道。
<br/>　　“你一个老家伙，我家少主不过是问你一句，你居然……”
<br/>　　虽然黑市买卖全凭自愿，但是这话也未免太过了。
<br/>　　拓跋鼎刚想继续说下去，却是被楚风眠一把打断。
<br/>　　“这内丹多少钱，我买了。”
<br/>　　“哦？你要买？”
<br/>　　听到楚风眠的话，那老者脸上露出几分惊讶。
<br/>　　他本以为楚风眠只是过来随便问这的，自然懒得理会，但是听到楚风眠这句话，他的眼神中不由的露出了几分贪婪。
<br/>　　“一亿九鼎丹，你要是买了，我就告诉你这是什么内丹。”
