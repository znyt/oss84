　　“天道在上，从今往后，龙族愿镇守洪荒四海，执掌洪荒风调雨顺，吾龙祖愿身化洪荒，弥补破碎洪荒天地！”
<br/>　　话音回荡在东方苍穹之上，一股无上伟力出现，天音缥缈之时，那龙祖巨大的九爪神龙之躯和元神，顿时消散在这天地之间。
<br/>　　顿时，东方天地之间的劫煞之气，少了三分之一，而东方大地，也被重新塑造，
<br/>　　“哈哈！天道在上，自此后，我凤族愿永镇南方不死火山！”
<br/>　　“天道在上，吾麒麟一族愿化为祥瑞之兽，造福天地，吾麒麟王当身化天地，补缺洪荒！”
<br/>　　凤族与麒麟王见龙祖的下场，顿时，面带惨笑，对视一眼，蓦然冲天而起，大喝一声，此声回荡在天地之间，麒麟王陨落，凤祖带着一丝残破真灵直奔南方不死火山，此后无量量劫不出。
<br/>　　顿时，南方天地与北方天地仿佛被重塑一番，破碎的大地恢复原样，天地间劫煞之气，也瞬间减少一大半。
<br/>　　在三人对天发誓之后，那苍穹之上的巨大淡漠眼眸也渐渐消失不见。
<br/>　　而此刻，躲在虚空之中的几人，脸上早已冷汗不止。
<br/>　　那可是天道之眼啊，被一方天道所注视，可不是那么好玩的，一个不小心，就会没命啊。
<br/>　　不过，此刻天道之眼淡化后，他们都轻呼一口气，释放心中压力。
<br/>　　“龙，凤，麒麟，先天百族完了，哼！活该尔等应劫。”
<br/>　　对于东方大地，南方大地与北方大地的变化，百族的落寞，鸿钧几人自然是有所察觉，但他们此刻也顾不得那么多。只是心中冷哼一声，便向着西方大地飞去，因为还有更大的事等着他们。
<br/>　　龙凤麒麟三族的大劫是过了，但他们的劫还没有过。
<br/>　　道魔之争，事关证道之机，不可不争！
<br/>　　……
<br/>　　西方，灵山之上，魔窟之中，魔祖罗睺正坐在大殿之上，吸收着天地之间的劫煞之气修炼，身旁四柄凶剑缓缓旋转，散发着凶戾的剑气，在这大殿之中肆掠，将空间绞碎。
<br/>　　良久，罗睺睁开双眼，眼中闪过诡异的红光。
<br/>　　“还差一点，就差一点了，本尊就突破了，可恶！看来，本尊突破到契机还没有到来。想必那契机就在鸿钧老祖身上吧。哼！”
<br/>　　说到此处，罗睺停了一下，一想到鸿钧，他的脑海中顿时就有一股强烈的敌意，他亦不知为何，想来这就是天生宿敌的原因吧。
<br/>　　“既然来了，那就不要缩头缩尾的，那样，会让本尊瞧不起你们！”
<br/>　　罗睺眼瞳一缩，看向大殿之外的虚空之中，冷声道。
<br/>　　罗睺话音刚落，就听到空中传来一阵道韵歌声，只见一个道人踏空作歌，于虚空之中走了出来。
<br/>　　“高卧九重云，蒲团了道真。
<br/>　　天地玄黄外，吾当掌教尊。
<br/>　　盘古生太极，两仪四象循。
<br/>　　一道传三友，二教阐截分。
<br/>　　玄门都领袖，一气化鸿钧。”
<br/>　　“罗睺，你为了一己之私设计引起龙凤麒麟三族大战，搞得洪荒天地破碎，死伤无数，罪孽深重。今日，就是你魔头授首之日。”
<br/>　　“不错，就你这魔头，我等焉能让你成道！”此时，一道大喝从虚空传来。
<br/>　　“乾坤道友说得是，吾阴阳老祖，今日定要杀你这魔头！”又是一声大喝，两人从虚空之中走了出来。
<br/>　　“三位道友所言极是，魔祖罗睺，当陨！”又是一个道人从虚空走出，却是穹天老祖。
<br/>　　“哼！鸿钧，你们就不要在此虚情假意了，想要杀我，还找如此烂的理由。什么天下苍生，搞得自己有多高尚是的。哼！本尊最看不起尔等这些虚伪小人！
<br/>　　废话少说，你以为你们四人就杀得了我吗？哼！不是本尊看不起你们，就凭你们？！”
<br/>　　“也罢，今日，就让尔等好好看看，本尊是如何屠道证圣的！”
<br/>　　魔祖罗睺好似胜券在握，对鸿钧等四人的话不以为意，张狂依旧，性情如魔。
<br/>　　“非铜非铁亦非钢，曾在须弥山下藏。
<br/>　　不用阴阳颠倒炼，岂无水火淬锋芒？
<br/>　　诛仙利，戮仙亡，陷仙四处起红光。
<br/>　　绝仙变化无穷妙，大罗神仙血染裳。”
<br/>　　“诛仙四剑，给吾现！现！现！现！”
<br/>　　“轰隆隆！”
<br/>　　罗睺话音刚落，悬挂于周身的四剑旋转起来，分别飞向洪荒西方四极。他大手一挥，一张古朴阵图沉落于脚下蔓延开来，不一会儿就笼罩了整个洪荒西方。
<br/>　　诛仙、戮仙、陷仙、绝仙四柄凶剑分别飞到阵图的四门。
<br/>　　“嗡！”
<br/>　　如此，号称杀伐第一的先天至宝诛仙四剑及剑阵便成功布置起来！
<br/>　　庞大的煞气、杀气、凶气、戾气、魔气、黑气等负面之气瞬间便封锁了整个洪荒西部大地。
<br/>　　大阵之中，灵气隔绝，其内红光闪烁，煞气惊天，魔气纵横，凶威难测，不愧是先天第一杀伐至宝诛仙剑阵！
<br/>　　虚空之中，魔祖罗睺凭立虚空，而剑阵之中，却是被诛仙剑阵所包围的鸿钧等四人。
<br/>　　“哼！本尊诛仙剑阵，非四人不可破，尔等虽然有四人，但，也休想一时将之破去。”
<br/>　　诛仙剑阵之中，鸿钧等人面色凝重的看着眼前的大阵，他们能够感受到这大阵之中的无尽凶威，一个不小心，他们就会命丧于此处。
<br/>　　同时，他们也知道那罗睺说的不错，这诛仙剑阵以一人之力是万万无法破除的。就算是它他们四人一起，一时间也无法快速破阵。
<br/>　　所以，他们要小心再小心，不能大意。
<br/>　　“三位道友，吾观这诛仙剑阵玄妙无双，内蕴无限杀机。吾观之，此阵有四门，那魔头所言非虚，非有四位同等级之人，恐怕难以破之。幸好我等刚好有四人。吾等每人各自破一门，如何？”
<br/>　　鸿钧老祖看着眼前的大阵，微微皱眉，神色凝重道。
<br/>　　“嗯！”
<br/>　　其余三人都眉头一皱，知道鸿钧老祖所言不虚，此阵的确如此。
<br/>　　几人立即冲入大阵四方阵门，鸿钧冲到诛仙阵门处，阴阳老祖冲到陷仙阵门，乾坤老祖冲到戮仙剑阵处，穹天老祖冲到绝仙阵门处。各自破一门。
<br/>　　“诛仙剑阵，戮仙弑神，绝灭一切，转！”
