        只见一个八岁左右，长得可爱又水灵的小女孩，提着一篮子娇艳欲滴的红玫瑰，举到了傅夜沉的眼皮子底下。
<br/>
<br/>        “我很老，嗯？”傅夜沉一脸较真地凝眉瞪着小女孩。
<br/>
<br/>        童筱颖忍俊不禁地拍了拍傅夜沉的手臂，故作训斥地说道：“别吓着孩子了。”
<br/>
<br/>        但小女孩似乎一点都不怕傅夜沉，反倒是两眼放光般仰头看着傅夜沉，毫不吝啬地赞美：“大哥哥，你长得真帅，和小姐姐很般配噢！”
<br/>
<br/>        “你这一篮子的玫瑰花，多少钱？”傅夜沉顿时眉开眼笑地问。
<br/>
<br/>        小女孩立即伸出一根食指，一边比划着，一边回答道：“带篮子一起，给你个优惠价，一千块。”
<br/>
<br/>        “……”童筱颖汗颜，这个小丫头真会开价。
<br/>
<br/>        傅夜沉则毫不犹豫地掏出自己的皮夹，随手抽了一叠钞票递给了小女孩。
<br/>
<br/>        明知道是被“宰”了，童筱颖却并未阻止傅夜沉，她心想着，大哥不差这点钱，说不定还能帮到这个小女孩。
<br/>
<br/>        小女孩从傅夜沉的手中接过钞票后，随手将玫瑰花附带篮子一起挂在了傅夜沉的臂弯上后，开始一本正经地数钱。
<br/>
<br/>        她只数了十张钞票出来，剩余的五张全部退还给了傅夜沉。
<br/>
<br/>        “大哥哥，谢谢你。”小女孩高兴地向傅夜沉鞠了个躬，随即又看向一旁的童筱颖，抬起手来，向童筱颖勾了勾食指。
<br/>
<br/>        童筱颖见状，会意地笑了笑，然后蹲下身去。
<br/>
<br/>        小女孩挪步上前，附在童筱颖的耳边呢喃细语起来。
<br/>
<br/>        童筱颖听了后，忍俊不禁地摸了摸小女孩的头顶。
<br/>
<br/>        小女孩将钱塞进了衣兜里，高高兴兴地转身跑开了。
<br/>
<br/>        傅夜沉候在一旁，好奇地问道：“那个小丫头跟你说了什么？”
<br/>
<br/>        “秘密。”童筱颖莞尔一笑。
<br/>
<br/>        傅夜沉并不打算追问，而是再次牵起童筱颖的手，另一只手中则提着那一篮子的玫瑰花，微笑道：“这篮子的玫瑰花，晚上正好用来给你泡玫瑰牛奶浴。”
<br/>
<br/>        还可以这样？！
<br/>
<br/>        童筱颖真的发现，是自己的贫穷限制了想象。
<br/>
<br/>        花一千块钱来泡澡……
<br/>
<br/>        傅大少爷果然是含着金汤匙出生的，除了分分钟都在赚钱以外，几乎秒秒钟都在奢侈地花钱，估计他是从没过过那种没钱苦逼的贫穷日子。
<br/>
<br/>        不过，既然他都开口说了，要拿这些玫瑰花瓣泡牛奶浴，那么她就照做便是。
<br/>
<br/>        “好啊！”童筱颖抿嘴笑了笑。
<br/>
<br/>        毕竟，钱是他赚的，他想怎么花就怎么花，他开心就好，她不会多一句嘴。
<br/>
<br/>        而刚刚那个小女孩，附在她耳边，对她轻轻地说：“大哥哥的眼睛告诉我，他很爱你。”
<br/>
<br/>        他爱她，她都是从别人口中得知的。
<br/>
<br/>        什么时候，她才能亲耳听到他对她说那三个字。
<br/>
<br/>        就像，那个时候，她在等着昼景对她说那三个字一样。
<br/>
<br/>        然而，等到她终于盼来昼景说“我爱你”的时候，却是她和昼景的婚姻走到尽头的时候。
<br/>
<br/>        这一次，会不会又重蹈覆辙？
<br/>
<br/>        她的心里不知不觉中对他傅夜沉有了些许的憧憬，却又隐隐地为自己拉起警戒线。
<br/>
<br/>        童筱颖下意识地仰头看着傅夜沉。
<br/>
<br/>        他对她而言，其实是最熟悉的陌生人。
<br/>
<br/>        她认识他有三年了，但是她却不了解他，一点都不了解。
<br/>
<br/>        他到底是一个什么样的男人，她看不透，也猜不着。
<br/>
<br/>        因为，她看到的，好像跟真正的事实不大一样。
<br/>
<br/>        傅夜沉牵着童筱颖慢慢地继续往前走，直到走到一间别致的餐厅外才停下了脚步。
<br/>
<br/>        这家餐厅的外形是一棵仿真型的大松树，类似于圣诞树那样一共有三层，每层都有窗户，外墙像真的松树叶，上面点缀着琳琅满目的五彩蛇形灯，每层的飞檐下还吊着星星灯，美轮美奂。
<br/>
<br/>        奇怪的是，这家餐厅，居然没有挂店名。
<br/>
<br/>        “这里的饭菜很有特色，我带你去尝尝。”傅夜沉将童筱颖的手搭在了自己的臂弯上，拉回了童筱颖一直在打量的目光。
<br/>
<br/>        此时，恭候在门口的服务生，主动迎上前来，对他俩迎了迎手，微笑着说道：“冷先生、冷太太，欢迎光临。”
<br/>
<br/>        诶？认识吗？
<br/>
<br/>        童筱颖一脸诧异。
<br/>
<br/>        傅夜沉嘴角微扬，矜持地带着童筱颖走了进去。
<br/>
<br/>        随后，又有一个领头的年长女服务生，满面笑容地朝他们走来，毕恭毕敬地对傅夜沉说道：“冷先生，好久不见，想必这位是冷太太吧！冷太太长得真漂亮。”
<br/>
<br/>        “谢谢。”傅夜沉倒是一点都不害臊地替童筱颖谢过了别人的赞美。
<br/>
<br/>        童筱颖听着，却尴尬不已。
<br/>
<br/>        她明明是第一次来吧！
<br/>
<br/>        他们怎么就一个个都称呼她为“冷太太”了？
