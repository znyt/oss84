　　这是他的使命。
<br/>　　有了他在前面撑着，四宝也能全身而退，去帮小伙伴们了。
<br/>　　这个不足十人组成的队伍，像一艘船，在山谷中滑行。总算是艰难地到了谷口。
<br/>　　在山谷外面的空地上，雪魔率领的太阴山大军，还在跟摇光仙子等一干空界的强者厮杀缠斗，无数精魄从死亡的鬼兵体内飞出，无数尸兵倒下，化作腥臭的肉泥。
<br/>　　空界这边，也死了不少几个强者。
<br/>　　惨烈至极的战斗……
<br/>　　叶少阳却是松了一口气，看了道风一眼，道：“好险，从来没这么刺激过！”
<br/>　　这是他有生以来，经历过的第二次最为惨烈的混战。第一次是在悬空观。
<br/>　　不过这一次，对手明显要强大的多。叶少阳回头想了一下，都有点毛骨悚然，不敢相信自己居然从这么多空界强者的手下死里逃生……
<br/>　　“还没结束呢。”道风眼望丛林远处，虽然有丛林阻挡，但是能够看到升腾在树林上方的煞气，那边，必然有情况发生。
<br/>　　叶少阳法力不如他，自然看不到这些尚在远处漂浮的煞气，他以为道风说的是自己回到人间之后，所要面对的那些，咬了咬牙，说道：“不管怎么样，努力去做就是了，我没退路，也不需要什么退路。”
<br/>　　道风指着前面说道：“现在就没退路了。”
<br/>　　叶少阳等人一怔，举目望去，这才发现了树林中的异象：树木摇晃，丛林深处传来哗哗的响声，像是有什么东西正在穿过丛林奔跑来。
<br/>　　叶少阳脑海中出现了一些电影里的画面，下一秒钟，有大象、犀牛之类的大型动物冲出树林……
<br/>　　不过最终冲出树林的不是什么动物，而是一群飞在半空中的浑身黝黑的人形生物。
<br/>　　“飞僵！”叶少阳一眼就认了出来。
<br/>　　数十只飞僵，一起冲过树林，朝山谷奔来，在他们身后，是更多僵尸，穿过丛林，冲了过来。
<br/>　　摇光峰上，突然飞身下来一道魅影，悬在半空中，冲这些僵尸厉声吼道：“总算是来了，空界各派首脑，有一半都在山谷中，速速随我冲进去追杀！”
<br/>　　叶少阳听见这声音，有一种十分熟悉的感觉，转头看去，看到了一张熟悉的脸。
<br/>　　王曼思……当然不是王曼思，王曼思已经不在了，眼前这个长的一模一样的姑娘，是王曼思的本尊：女魃！灵界尸族的三大僵尸王之一！
<br/>　　叶少阳大脑卡壳了几秒钟，猛然明白过来：尸族想要浑水摸鱼！
<br/>　　他们一定早就发现了发生在星宿海的战斗，一直按兵不动，等着自己的捉鬼联盟和风之谷与空界诸势力苦战到现在，大大消耗了空界的力量，然后大举进攻，女魃说的没错，空界只是有五分之二的门派首脑都在这里，包括摇光仙子、黎山老母这样在空界最举足轻重的人物。
<br/>　　假如能够一举消灭这些大佬，那么就算一时间无法占领空界，也会让整个空界元气大伤，为尸族进犯空界铲平了许多障碍……
<br/>　　女魃看了一眼叶少阳，邪魅一笑，道：“总算是见面了，不过你该庆幸，我们这次有共同的敌人。”
<br/>　　说完也不看叶少等人，率领着尸族大军，朝摇光仙子等人冲过去。
<br/>　　眼前局势陡变，那些从山谷里追出来的空界强者，看到这一幕，顿时都傻眼了，看看叶少阳一干人，又看看女魃，一时间有点懵。
<br/>　　摇光仙子呵斥道：“尸族进犯，空界危在旦夕，还管他们干什么！”
<br/>　　这帮人一想也是没错，他们跟叶少阳其实没什么仇，只是为了杀转世鬼童芮冷玉。至于道风也是一样，道风杀了他们很多人，必然是仇人，不过这属于私仇，毕竟道风现在也是在逃命，对空界的整体并没有什么威胁，但是尸族大军却不一样，他们是真正的死敌，是威胁整个空界安危的一方势力，两者权衡之下，自然还是先应付尸族大军。
<br/>　　有一些精英弟子，四散飞走，各自回宗门报信去了，其余的强者联合起来，抵挡尸族大军的进攻。
<br/>　　这里最尴尬的就是雪魔和他的太阴山大军了，被夹在中间，一时间不知所措。
<br/>　　雪魔冲着女魃喊道：“你们来的正好，一起灭了他们！”
<br/>　　女魃冷笑道：“青冥界内战，不需要太阴山插手，你却来的不是地方！”
<br/>　　雪魔一听，怔了一下，冷笑道：“你还想跟我太阴山作对不成？难道不怕承受鬼王的怒火？”
<br/>　　“等你的鬼王能来到空界再说吧。”女魃纵身飞向山谷，丢下一句话：“一起灭了！”
<br/>　　她并非完全不顾忌太阴山，只是看准了雪魔这就是一支孤军，灭了他们，其实并不会真的让太阴山上那位震怒。一切都是棋子而已，她要通过这一作法，向那一位表达一种态度：尸族永不为奴，想要合作，那就是真正的合作，不要妄想操控尸族，把他们当做棋子！
<br/>　　混战开始……
<br/>　　只有捉鬼联盟和风之谷若干人置身事外，叶少阳等人望着激战在一起的三方人马，都是一脸的不可思议。叶少阳转头朝道风看去，道：“怎么办？”
<br/>　　“不趁这个机会冲出去，你还等什么？”
<br/>　　叶少阳怔住，“我们……”
<br/>　　道风冷笑。他知道叶少阳在想什么，从本质上来说，人间与青冥界是一脉相连，空界很多强者，实际上都是人间法师的祖宗。虽然空界很傲慢，一向看不起人间，但归根究底也算是自己人。眼看着空界真的遇到危难，叶少阳这种圣母表的心理又爆发了。
<br/>　　“各自求生而已，少阳，你选择保护鬼童的一刻，你就不再是人间法师了，任何事都轮不到你了。”
<br/>　　轮不到自己……
<br/>　　道风的一句话，也是毫不留情的帮叶少阳认清了现实，是的，自己就算回到人间，也基本上是人人喊打的地步，现在还去操心别人，真是有点太可笑了。
<br/>　　“走吧！”叶少阳对身边人说道。
<br/>　　上古邪神在前面开路，一行人进入树林，刚进去，突然一道人影从对面飞来。叶少阳抬头一看，当场石化。
<br/>　　芮冷玉！
