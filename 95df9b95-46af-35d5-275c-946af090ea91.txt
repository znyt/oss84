　　第102章
<br/>　　骨碌碌人头滚动，在白色的雪地画出红色的痕。
<br/>　　“吼！”
<br/>　　赤虎和野德等人放声大吼，不顾一切向程大雷冲来。桥头有秦蛮和赵子龙，这些人冲过来的行为，就像蚁群冲向火球一样。可他们仍前赴后继，没有丝毫犹豫。
<br/>　　两柄铁枪横扫，枪尖所指之处，血花绽放。
<br/>　　最后的结果——
<br/>　　死。
<br/>　　呼尔勒一行人没有一个逃命，全部死在桥头，尸体横七竖八，致命伤全部在前面。
<br/>　　哒哒哒的马蹄响，远处陆哼带领一帮人奔过来，在相距石桥五十步时，忽然勒马停下。
<br/>　　呼尔勒竟被程大雷杀死了！
<br/>　　北蛮部最有威信的小王子，即将咆哮四方的寒地苍狼，竟死在这里。
<br/>　　死在一个山贼手中。
<br/>　　程大雷浴血而立，脚下的雪多白，衬得他脸上的血便有多红。
<br/>　　忽然咧嘴一笑，弯腰将地上的人头捡起来。呼尔勒警惕的看着对方，下意识攥紧自己身上的短刀。
<br/>　　“陆将，送你了。”
<br/>　　人头从空中划过，一直滚到陆哼马蹄前。呼尔勒仍旧一脸凶恶，可如今的他已经身首异处，不管再凶恶，也奈何不得这人世半分。
<br/>　　陆哼一时之间呆住。
<br/>　　戎族年年犯边，烧杀劫掠，美其名曰打草谷。陆哼身为边关守将，经常有和他们交手的时候，总得说来却是输多赢少，坦白点也可以说没赢过。
<br/>　　可今日这十几具尸体，以及一颗小王子的人头，足以让陆哼的军阶再上一层。务实点说，能得千亩良田的赏赐。
<br/>　　“喂，走啦！”
<br/>　　程大雷高呼一声，几人骑马离开，茫茫雪地很快便没了他们踪影。
<br/>　　今日事，若非赵子龙及时出现，蛤蟆寨将会一败涂地。程大雷觉得功劳全在自己，如果不是自己在千钧一发之际召唤出来他，怎能逆风翻盘。当然，在其他人眼中并不这么认为。
<br/>　　据赵子龙自己所讲，他姓赵，名子龙，却不叫赵云。从小无父无母，在张家长大，与刘关张三人亲若兄弟。尤其是对刘悲，分外崇拜。
<br/>　　程大雷摸摸下巴，怎么看都觉得刘悲和眉目如画、风姿潇洒扯不上关系。
<br/>　　难道这人格魅力就这么邪乎么。
<br/>　　这次是知道刘关张在蛤蟆寨落草，特意前来投奔……刘悲。
<br/>　　程大雷很无语。
<br/>　　大家对赵子龙都很感激，尤其是秦蛮和林少羽。大家都是习武之人，见赵子龙年纪轻轻便如此了得，各人都是打心眼里羡慕。
<br/>　　相比之下，林少羽这少年游侠就显得差点意思了。
<br/>　　“赵兄弟，今天多谢你舍身相救，不然我们还真收拾不了他们。”秦蛮。
<br/>　　“勇者之事，甚于生死。”
<br/>　　“真没想到，你年纪轻轻，竟就有如此了得的枪法，你怎么做到的？”林少羽。
<br/>　　“心怀不惧，方能翱翔于天际。”
<br/>　　“我家四弟，就是胆子大，莫说只是十来个人，便是千军万马他也敢闯。”张肥自豪道。
<br/>　　“枪若惊雷，照一身肝胆。”
<br/>　　咦！
<br/>　　程大雷骑在马上，回头看了赵子龙一眼，总觉得这娃的话有些奇怪和熟悉。
<br/>　　赵子龙从呼尔勒那里抢了一匹马来，此刻正好程大雷挡在他的马前，他抬起头看着程大雷。
<br/>　　“抱歉，你挡路了。”
<br/>　　“……”程大雷。
<br/>　　原来这娃是个台词咖，该不会他除了审美观崩坏外，脑袋也有些……不大好吧。
<br/>　　几匹马踏过长路，在雪地上留下一串串蹄印。
<br/>　　程大雷骑马行在回蛤蟆寨的路上，几人火热的聊天他并未参与，脑海中还不断出现与呼尔勒一伙儿战斗的情景。
<br/>　　戎族骑术，天下无双。
<br/>　　今天程大雷深刻体会到这八个字的含义，他忍不住去想：假若戎族再如三十年前般入侵，帝国该如何抵抗戎族骑兵？
<br/>　　这一刻，程大雷已踏入帝国军人的思考领域。
<br/>　　三十年前的那场战争，是整个帝国军人的耻辱。那场战争中，帝国军人领教到戎族骑兵的恐怖。在那场沉痛的失败后，无数人都在思索，如何抵抗戎族骑兵？
<br/>　　有人主张结步兵方阵，以重甲士兵硬抗戎族骑兵的冲锋。
<br/>　　有人主张据城坚守，戎族虽擅长奔袭，但并无攻城器械。
<br/>　　也有人主张以骑对骑……可帝国并无好马，也无善御马之人。
<br/>　　……
<br/>　　这些法子有用也有限，终不能让帝国在戎族年年犯边的过程中占得上风。
<br/>　　直到如今，帝国没有对抗戎族骑兵的有效办法。
<br/>　　今天是幸得赵子龙及时出现，否则后果不堪设想。虽然赵子龙不是赵云，但看这潜力以后肯定是传说级，至少也是个高仿版赵云。
<br/>　　既然有了高仿版赵云，那么日后未必没有山寨版吕布，水货版诸葛，莆田系樊梨花……而摆在自己眼前的问题，只是如何把他们召唤出来。
<br/>　　一行七人走在路上，程大雷从怀里掏出一本秘籍丢给张肥。
<br/>　　“喏，给你的。”
<br/>　　张肥接过来一楞，只见秘籍封皮写着三个字：『三手矛』
<br/>　　“大当家，这是你从哪里得来的？”张肥接过后大吃一惊：“这可是我张家家传绝学，只是后来失传了。”
<br/>　　“咦，我被神仙传艺的事情你没听说过？”
<br/>　　这本秘籍自然是程大雷抽奖得来的，一看就知道是为张肥量身准备。程大雷已经翻过一遍，这本秘籍中记录的招数一共只有三手：一手矛捅天，一手矛扎地，一手矛赶猪。
<br/>　　虽然招式简单，名字也蠢，秘籍却是绝世级的，程大雷忍不住去想，会不会越是招数少的武学也就越厉害。
<br/>　　可自己的三板斧怎么就感觉那么不上档次。
<br/>　　张肥得到秘籍，短时间能提升到优秀级，可自己的战斗力该怎么提升呢。
<br/>　　真要遁入山林，修炼他个十年八年的，程大雷也没这个时间。上次山寨等级提了一级，自己的战斗力也随之升级。是不是代表，每将山寨升级，自己也可以随之升级。
<br/>　　但二级山寨要升三级，究竟需要满足那些条件呢？
<br/>　　人口，建筑，还是说必须完成某个具体的任务？
<br/>　　关于这些，系统一点提示都没有，只能靠他自己琢磨。
<br/>
