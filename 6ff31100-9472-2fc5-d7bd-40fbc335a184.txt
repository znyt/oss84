　　“这星域这么大，你们巫族，或者是其他族群的武者，都是如何赶路的？”苏莫好奇的问道。
<br/>　　“一般距离较近的星辰，都布置有星域传送阵，但距离较远的星辰，只能靠飞行！”
<br/>　　巫娴倒是没有隐瞒，非常识趣的为苏莫讲解，道：“一般只有武尊以上级别的武者，才会随意的踏进星空，武尊以下的武者很少会飞进星空，因为太耗时间了！”
<br/>　　巫娴现在的心境，早已平静下来了，只要苏莫暂时不杀她，她就有很大的活命机会。
<br/>　　因为她知道，巫梵和巫尹两位尊者，定然会救她。
<br/>　　苏莫闻言点了点头，若是一般的武皇境武者，飞进星空赶路，怕是一次赶路都要数年，甚至是数十年的时间了。
<br/>　　眼眸扫视四方，苏莫很快便锁定了一颗星辰。
<br/>　　那是一颗淡青色的星辰，在他的目视之中，算是这片星空中的中等大小的星辰，最主要的是，这颗星辰貌似距他最近。
<br/>　　随即，苏莫便和秦不死改变了方向，向这颗青色的星辰飞去。
<br/>　　他们不可能一直在星空中漂流，必须找个星辰落脚。
<br/>　　有了目标之后，两人的飞行便没有那么枯燥了，毕竟心中有了期盼。
<br/>　　时光如流水，眨眼间变过去了约莫三个月的时间。
<br/>　　看似相距不远的青色星辰，苏莫和秦不死足足飞行了三个月，依旧没有到达，可见距离之远。
<br/>　　不过，这颗星辰的体积，在两人的视线中已经放大了十数倍，想必他们距此星已经不远了。
<br/>　　“希望不会是废星！”说心中暗道，不过既然是青色的星球，是废星的可能性很低。
<br/>　　那大片的青色，想来应该是连绵山脉中的林木。
<br/>　　三个月的飞行，撑着巨大的玄力气罩，苏莫身上的资源也即将耗尽，斩杀试炼者获得的那些神丹和灵草都已经耗光，还唯有一部分灵石了。
<br/>　　这些灵石，还够他消耗一个月的时间。
<br/>　　而秦不死身上的资源，也所剩不多了。
<br/>　　不过，苏莫也不担心，因为他还有巫森的空间玉镯，其内还有十数万的武者，这都是他的资源。
<br/>　　苏莫和秦不死继续飞行，时间不断的流逝，终于在一个月之内，他的灵石耗尽之前飞近了这颗星辰。
<br/>　　当近距离看清这颗星辰，苏莫和秦不死都不免心中震撼。
<br/>　　此星辰之大，简直难以想象，星辰上那无垠的大地比之苍穹世界五大洲加起来，还要庞大数十倍，那连绵的山脉如同一条条庞大的青龙，延绵千万里。
<br/>　　此星辰之外，笼罩着一层厚实的罡风，呜呜作响，将整个星辰都笼罩在了其内。
<br/>　　这是九天罡风层，苏莫并不陌生，苍穹世界之外，同样也有九天罡风层。
<br/>　　只不过，苍穹世界的九天罡风层之外，是彻彻底底的虚无之地，充满着毁灭的空间之力。
<br/>　　很快，苏莫和秦不死两人，便飞到了九天罡风层之前。
<br/>　　“苏莫，这九天罡风层并不强，以我们的实力足以进去！”秦不死打量了片刻，沉声说道。
<br/>　　“走！”苏莫点了点头，毫不犹豫，立刻和秦不死冲进了九天罡风层之中。
<br/>　　嘭！嘭！嘭！
<br/>　　苏莫的玄力气罩并未撤去，进入九天罡风层之后，无尽的罡风如同一柄柄的天刀，斩击在了玄力气罩上，闷响声不断。
<br/>　　这罡风着实不弱，估计相当于普通的武帝一重武者的攻击，猛烈的风刃将玄力气罩斩击的不断震颤，剧烈的扭曲。
<br/>　　苏莫立刻将玄力气罩稍稍缩小，增强防御力，免得被罡风击破，毕竟，气罩越大防御越弱。
<br/>　　嘭嘭嘭嘭！！
<br/>　　在不断的闷响声中，约莫三个呼吸之后，苏莫和秦不死便穿过了厚实的九天罡风层，进入了这颗青色星辰的高空中。
<br/>　　“好浓郁的灵气！”感受到空气中浓郁的灵气，苏莫不禁眼眸一亮。
<br/>　　这颗星辰上蕴含的灵气，其浓郁程度，是苍穹世界的数十倍都不止，堪称修炼圣地。
<br/>　　随即，苏莫和秦不死两人，立刻向这颗青色星辰的大地之上飞去。
<br/>　　嗖！嗖！
<br/>　　两人速度极快，快速远离了九天罡风层，而苏莫一直在暗暗的注意着九天罡风层的变化。
<br/>　　那巫梵和巫尹，若是跟踪他们的话，必定也会进入九天罡风层，就算隐藏的再深，也会露出踪迹吧。
<br/>　　苏莫的预料果然不错，就在他们远离九天罡风层万余里之后，距离青色星辰的大地已经很近之时，他清晰的注意到，九天罡风层中的罡风一阵絮乱，然后瞬间便恢复如常。
<br/>　　“果然来了！”苏莫顿时心中一凛，虽然他没有发现巫梵和巫尹两人的身影，但他能肯定，那两人一定也进来了。
<br/>　　苏莫立刻头疼了起来，该如何甩掉这两人呢？
<br/>　　面对武尊级别的武者，他实在不知道该如何对付，他现在唯一的底气，便是手中的人质。
<br/>　　可是，他总不能常年手中抓着一名人质吧！而且总有懈怠的时候，若是被两人寻到了机会，他就彻底的完了。
<br/>　　嘭！嘭！
<br/>　　不多时，苏莫和秦不死终于落了下来，脚步踏在了厚实的土地上。
<br/>　　这里是一片一望无际的平原，荒草丛生，充满苍茫的气息，没有半个人影。
<br/>　　据闻，无尽星空中种族无数，不知道这颗星辰上有什么种族？不会没有人类吧？
<br/>　　苏莫心中有些忐忑，若是这颗星辰上是什么奇怪的种族，那他和秦不死来到这里，也会有巨大的危险。
<br/>　　不过，他们现在已经没有了退路，只能向前冲了。
<br/>　　“走！”
<br/>　　随即，苏莫轻喝一声，身形一动，和秦不死两人向远处飞去。
<br/>　　他们两人依旧精神高度集中，不敢有丝毫的松懈，每时每刻都在严防遭到偷袭。
<br/>　　长时间的精神紧绷，两人早已心神疲惫不堪，但只能咬牙支撑。
<br/>　　飞驰在天空中，两人一刻不停，约莫飞行了两个时辰之后，前方出现了一座城池。
<br/>　　苏莫和秦不死顿时心中一震，抬目望去，这是一座雄伟壮阔的古城，仿若蛰伏在地的太古巨兽。
<br/>　　灰白色的城墙拔地而起，高达万丈不止，延绵万里，一眼望不到尽头。
