　　百年时间转瞬即逝，洪荒还是一如既往地在发展着，历史洪流如车轮般滚滚向前。
<br/>　　洪荒除了多了一个圣人外，并没有其他别的事。该来的一样来了，不该发生的，还是没有发生。
<br/>　　虽然洪荒表面上风平浪静，没甚大的争斗，但实则这只是明面表现出来的现象而已。真正的洪荒，永远都是暗潮汹涌，谁也不知道下一次战争会发生在何时。
<br/>　　巫妖两族虽然暂时沉寂了下去，但谁都知道，这两族的绝不会就此罢休，他们都在为下一次大战而在不断积累实力。
<br/>　　而且洪荒之中，除了巫妖两族外，又有一些别的势力在悄悄崛起了，显然在知道了气运的作用后，众人都忍不住想要通过建立势力来夺取天地气运。其中最为强大的势力为东王公与西王母联手建立起的散仙联盟紫府。
<br/>　　东王公与西王母仗着在紫霄宫中被道祖封为天下散修男、女仙之首的名声，在洪荒之中大张旗鼓地招揽手下，扩张势力。
<br/>　　这多少会有些侵犯了巫、妖两族的利益。不过，两族的首领此刻都在忙着修炼，并没有出来驱赶。就这样，紫府更加疯狂了。
<br/>　　至于其他的，大多都是一些小的势力，折腾不起多大的浪花。当然，还有一些比较低调的势力，如，太华山劫魔殿。这些年万劫魔尊仗着魔修手段收拢了不少人物，但他并没有高调出手，而是一如既往地低调行事，这导致在洪荒之中，很少有人知道有劫魔殿这个势力。
<br/>　　当然，也有比如东海三清岛，众人都知道那里是三清的地盘，也没有人敢去那里放肆。
<br/>　　……
<br/>　　昆仑山，这天却是异常热闹。
<br/>　　昆仑山的山门口此时站着一些童子之类的修士，他们在迎接那些受到邀请的修士。
<br/>　　这些童子是在昆仑山中常年受元始三人论道而开启灵智之修，之后他们便一直生活在昆仑山中，元始三人也没有多管。
<br/>　　现如今昆仑开讲大道，元始的徒弟们又回到了东海永恒岛，管不到这里。他们此时正好用来迎接受邀的洪荒大神们。
<br/>　　不久之后，便有人来了。
<br/>　　“玉清圣人此番聚众论道，却是一番大造化，大功德。此行大善！”却是镇元子大仙与红云道人联手而来。
<br/>　　随后，妖族的妖帝帝俊、东皇太一、妖师鲲鹏联手而来了。紧接着，又有娲皇女娲与羲皇伏羲两兄妹一同到来。他们进来后，看了看，来到帝俊三人旁边，坐了下来。
<br/>　　之后，幽冥血海的冥河道人也来了。他来了之后，看了一眼众人，径直来到一个角落里，独自一人坐下，也不见上前找谁去说话。冥河在这洪荒之中，算是最为特殊的一个人，他谁也不去结交，一直冷眼旁观。如今也不见有哪些朋友。
<br/>　　紧接着，太阴之主，月神望舒带着他的徒弟嫦羲来了。她看了一眼高台之上，却不见元始三清的声影。
<br/>　　“哼！”
<br/>　　冷哼一声，她们来到左边角落里坐下。
<br/>　　不远处的妖帝帝俊感受到两人身上浓浓的太阴之力，与自己身上的太阳之力形成对比，若有所思。
<br/>　　之后，竟然是巫族的后土与玄冥祖巫到了。看到两个巫族的女子的到来，众人还是有些惊讶的，他们实在是想不到元始为何会邀请一个没有元神的祖巫来听道呢。还是说，祖巫与后土之间有什么关系。
<br/>　　想到此处，紧接着，他们又想到祖巫与三清同属于盘古后裔这一层关系，顿时全身一震。于是，他们看向两祖巫的眼神变得不一样了，同时看向妖族的眼神也变得不同了。在他们看来，若是真的祖巫与三清之间存在某种关系，那么，妖族可就要倒霉了，毕竟三清代表的可是日后的三位圣人啊。。
<br/>　　大殿之中，帝俊等妖族见到后土与玄冥的到来，也是有些意外。但紧接着，他的脸色就变得有些难看了，因为他也想到了这一层关系。不过，他也没有多说什么，毕竟，此处是元始的地盘，不看僧面还要看些佛面的。
<br/>　　在祖巫之后，是东王公与西王母。这两位可是最近在洪荒之间闹得很欢的人物，如今也在元始的邀请之列。
<br/>　　看见两人，妖族与巫族的眼神顿时变得十分不善了。毕竟，这两人在洪荒的众多行为，已经侵犯了他们的利益。不过，他们也没有多说什么。这里显然不是解决恩怨的地方。
<br/>　　“善哉！善哉！看来贫道二人是来迟了。”
<br/>　　最后，西方的准提道人与接引道人也来了。
<br/>　　“咚——”
<br/>　　“哈哈哈！欢迎各位道友前来昆仑交流论道。山门简陋，切莫见怪！”
<br/>　　最后，直到所有人到齐了。元始才带着老子与通天来到大殿之上，对着众修士招呼道。
<br/>　　这次出来讲道的却是元始本尊，至于天罚圣主原始，则是去闭关炼化两道天道本源去了，只有完全炼化，他才算是真正的执掌它们。
<br/>　　这就是以力证道与其他证道的不同。用其他方式证道成圣，只能魂寄天道，虽然也可以通过天道去领悟那些天道法则，但是却不可能像天罚圣主原始那样，破开天道空间，再将意识烙印在本源之上，最后通过炼化本源，彻底掌控天道本源。
<br/>　　如今原始才刚将因果与天罚本源烙印，还没有完全炼化，所以，他需要闭关，花费大量时间去炼化它们，直至完全掌控。
<br/>　　元始本尊于原始记忆相同，对于大道领悟也相同，两人谁来讲道，都不会有影响。
<br/>　　“见过天罚圣主！”
<br/>　　众修士站起来对着元始行礼。同时，它们心中也很好奇，好奇这元始的实力到底到了哪一步了，为何他们在元始的身体之上，感受不到丝毫的法力波动。
<br/>　　“不必多礼，来人，上茶！”
<br/>　　元始大手一挥，便叫人端上此次茶会的悟道茶。
<br/>　　接着，童子们端着元始用悟道茶树枝做的茶杯，各自摆放到众大神面前。台桌之上，还有一些顶级灵果。之后，童子们又为众修士满上茶水。
<br/>　　只见那茶杯之上，云雾缭绕，一层一层，圈圈缭绕，共有九重，就仿佛九重天一般。之后，在那云雾之中，会传出一阵阵真龙吼叫，凤凰鸣啼，神钟撞鸣，仙音飘摇等各种大道声音。
<br/>　　后来，就连各种神兽与神物的虚影都会在茶杯之上显现出来。小鼎、神凤、仙人、神钟、山河、云朵、八卦、太阴、太阳、星辰等各种异象，栩栩如生。
<br/>　　同时，大殿之中飘散出那令人心旷神怡、气定神闲、烦乱尽消的茶香，满殿的芬芳之气，扑鼻而来。
<br/>　　轻轻吸一口，顿时脑中各种烦恼尽皆消散，脑海变得异常清明，元神变得灵动不已，思维散开，紧接着，平时各种不懂之处，一下子想明白了。各种领悟，纷纷涌来。
<br/>　　“这……这时什么茶？竟会有如此神效？！”
<br/>　　……
