　　第276章
<br/>　　琴川一带的确没有匪，因为匪就是官，官就是匪。
<br/>　　莫鸣米被招安后，成为琴川关的守将，琴川一地的土匪强盗也被他招安了，莫鸣米一口气封了十八家校尉。这些人有陆路的，有水路的，每个人各管一片。
<br/>　　凡是做生意的，从他们路上过，都要缴纳一定的保护费，至于私盐、茶叶、铁器之类的违禁品，直接是被他们垄断了。
<br/>　　这云中龙就是十八家校尉之一，今日高飞豹过的那段路恰好就是他管着，所以他才会抢高飞豹的货。也亏得高飞豹先下手为强，不然程大雷恐怕就见不着他了。
<br/>　　“一口气封了十八家……”程大雷撇撇嘴：“瞧瞧这没文化的劲儿。”
<br/>　　“大当家，要不派出兄弟，把他们灭了吧。”高飞豹心里还是咽不下这口气，往常都是他抢人，今天却是人抢他。
<br/>　　“这个……先不着急，反正今天咱们也没吃什么亏，需要的盐巴从别的地方运过来就是，咱们还是先安家。”
<br/>　　据说这云龙山手下有三四百号弟兄，纵然程大雷人数少，真打起来也不惧对方。可实在没有拼命的必要，程当家现在怎么说也是名声在外，怎么能动不动就和人拼命呢。
<br/>　　现在还是安家重要，大蛇把原先的山寨建筑毁了大半，建筑废墟都要清理出去。反正已经残破不全，索性拆掉重建。岛上原先的码头要修葺一番，程大雷还打算由岛到岸边修一座桥，这样马匹来往就方便了。
<br/>　　过了两天，走小路把盐巴运上岛，那条大蛇的尸体被分割腌渍起来。程大雷也派出弟兄，打探云中龙的情况，大家是初来乍到，摸不准情况程大雷不会莽撞出手。
<br/>　　“大当家，咱们什么时候去打云中龙？”高飞豹。
<br/>　　“不着急，再等等。”
<br/>　　高飞豹垂头叹气而去。
<br/>　　“大当家，今天要不要动手，我已经把刀磨好了？”
<br/>　　“动什么手，砍树去。”
<br/>　　高飞豹再次垂头叹气。
<br/>　　……
<br/>　　如此好几次，高飞豹经常垂头叹气，程大雷心想这孩子不会落下什么病吧。
<br/>　　高飞豹没有病，而且还很精神，所以说复仇是力量之源。如今的他，是每天生龙活虎的在岛上瞎转悠。
<br/>　　高飞豹如今也是个人物，在南方也是一山的二当家，向来只有他欺负人，没有人欺负他的份儿。这次买盐，半路被人劫了，他心中如何咽得下这口气。
<br/>　　大家该怎么看我，以后还能不能愉快的一起玩耍。
<br/>　　高飞豹觉得自己丢了人，程大雷心里的打算他也不知道。看程大雷忙里忙外的意思，是不打算对云龙中动手了。
<br/>　　他不动手，那我就自己动手。
<br/>　　所以说，江山易改本性难移，高飞豹已经不是一年前那个简单的愣头青，现在他是一个复杂的愣头青。
<br/>　　当然，他也没愣到单枪匹马去闯人家大本营，他决定找一个帮手。
<br/>　　岛上公认的战斗力第一，肯定是赵子龙，所以高飞豹第一个找上的就是他。
<br/>　　“子龙贤弟，子龙贤弟……”高飞豹提着酒壶：“子龙兄弟干嘛呢？”
<br/>　　“练枪。”
<br/>　　赵子龙盘膝坐在岸边的石头上，面对着滔滔河水，长枪横在膝前。
<br/>　　“这样有用么？”
<br/>　　“有用。”赵子龙惜字如金，又补充一句：“对我有用。”
<br/>　　意思是我天赋不够，对我来说没用了。知道你强，可看不起我的意思也不用这么明显吧。
<br/>　　“你找我有事？”
<br/>　　“哈，我能有什么事，不过就是找你喝喝酒，知道你是咱们这里最有本事的人，老哥心里这点委屈……”
<br/>　　“哦，没事我先走了。”
<br/>　　“呃……”
<br/>　　高飞豹看着赵子龙离去的背影，再次垂头叹气。
<br/>　　除了赵子龙外，就是秦蛮了，不过秦蛮是个稳重人，肯定不会陪着自己冒险。至于关鱼……不知怎么回事，在他面前，高飞豹总觉得自己没什么存在感。
<br/>　　当然，关鱼不是单纯的看不起自己，他是看不起所有人。
<br/>　　最后，高飞豹找到了张肥。
<br/>　　张肥正在分鱼，一柄菜刀运转如飞，鱼肉被他分成极薄的小片，如纷飞的花朵一般。
<br/>　　“好！”
<br/>　　张肥停下手中刀，看到高飞豹提着酒壶过来。
<br/>　　“高二爷看得出我这刀法好？”
<br/>　　“意于身合，身于刀合，才有如此犀利凌厉的刀法。在整个山寨，说到快自然是大当家的快剑，但说到准，张三爷不愧是山寨第一人。”
<br/>　　“山寨第一人？”张肥摸着络腮胡子：“谁说的？”
<br/>　　“大家都这么说，张三爷不知道么。”
<br/>　　“呃，哇哈哈。”张肥大笑：“我这个人比较低调，不喜欢听这些，你自己知道就可以了。”
<br/>　　“我正好带着酒，来来，咱们边喝边聊。”
<br/>　　“好，咱们仔细聊聊山寨第一人的事情。”
<br/>　　二人在岸边分鱼而食，先用火将石板烤热，再将鱼肉搁在石板上，一边喝酒一边分肉。
<br/>　　不多时，二人都已有些酒意。过程中，张肥看到高飞豹时不时垂头叹气。
<br/>　　“高二爷，我以前没和你聊过，今天才发现你的眼光很独到，山寨第一人尤其看得准。你要是有什么心事，可别瞒我？”
<br/>　　“哪里有什么心事，还不是上次……哎，喝酒喝酒。”
<br/>　　“如果有用得着张某人的，可千万不要客气。”
<br/>　　“咱们兄弟，我还能和你客气。”高飞豹扼腕道：“还不是在云中龙那里受了气，可大当家不想动手，凭咱们兄弟也做不了什么事……”
<br/>　　“不就是那云中龙么，还反了他了，放心包在我老张身上。”
<br/>　　“大当家如果知道……”
<br/>　　“嗳，咱们偷偷去，偷偷回，他也不会知道。”
<br/>　　“只有咱们两个，是不是需要小心些，要不再找几个弟兄。”高飞豹显得很谨慎。
<br/>　　“人多目标大，咱们两个就将事情办了。”张肥拍着胸脯道：“凭俺一根蛇矛，便是千军万马也不惧他。”
<br/>　　“那么……一定要去？”
<br/>　　“一定要去。”
<br/>　　“必须要去？”
<br/>　　“必须要去！”
<br/>　　二人雄赳赳气昂昂，趁着夜黑无人问，只给守船的弟兄说出去透透气，运了两匹马上岸，高飞豹扛着他的大刀，张肥扛着他的蛇矛，二人快马加鞭，一溜烟不见了踪影。
<br/>(本章完)
