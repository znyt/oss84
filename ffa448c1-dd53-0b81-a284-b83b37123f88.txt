　　楚风眠既然是收下了灵剑，也就算是同意将恩怨一笔勾销的意思。
<br/>　　对陈家来说。
<br/>　　虽然耗费了这样一把灵剑，可如果可以跟楚风眠化解恩怨，甚至是可以跟楚风眠交好，那便是值得的。
<br/>　　毕竟如今楚风眠的名声在外，已经是响彻无比，无数议论之中，楚风眠已经是最有希望踏入天人境的天才了。
<br/>　　这名声响亮，甚至超出了当年的少帝，这陈家家主送出灵剑，也有着想要攀附楚风眠的心思。
<br/>　　这陈家家主的心思楚风眠也是心知肚明，不过他也没有点破。
<br/>　　毕竟如今楚风眠在这阎罗星域之中，也没有什么认识的人，有着陈家相助，楚风眠做事也会方便许多。
<br/>　　“不知道楚公子这一次来陈家商会，是想要买些什么东西？”
<br/>　　陈进看着楚风眠缓缓开口道。
<br/>　　他现在也看得出来，楚风眠这一次来陈家商会，其实并未是为了报仇而来。
<br/>　　不然以楚风眠的性格，绝不会表现的那么平静。
<br/>　　“须弥空间，我需要足以容纳千丈大小生灵的须弥空间。”
<br/>　　楚风眠直接开口道。
<br/>　　他这一次来的目的，就是想要看看这陈家商会里，有没有千丈大小的须弥空间，来盛放太古蜈蚣所用。
<br/>　　“须弥空间，千丈大小？”
<br/>　　陈进听到楚风眠的话，眉头紧锁起来，须弥空间，虽然算是至宝，在如今的时代之中，都已经是没有炼器师可以炼制了。
<br/>　　可毕竟从远古时代之中，还是有着许许多多的须弥空间流传下来的，在陈家商会之中，这样的须弥空间，少数几千件还是有的。
<br/>　　可楚风眠后半句话，却是难住了陈进，千丈大小。
<br/>　　寻常的须弥空间，其中的空间也就足以容纳几个人大小，就像是楚风眠曾经得到的那黑盒子状的须弥空间类似。
<br/>　　十丈左右的须弥空间，都算是极为巨大的了，足以蒙养一些妖兽，至于百丈大小的，更是寥寥无几，举世罕见了。
<br/>　　而千丈大小的须弥空间，对陈进来说，也只是听闻过，可现在陈家商会之中，根本没有。
<br/>　　陈进本来是想要借此机会，拉拢楚风眠的，可听到楚风眠要买的东西，他也一下子犯了愁。
<br/>　　如果有的话，陈进也愿意现在卖给楚风眠，来跟楚风眠交好，可千丈大小的须弥空间，现在的陈家商会之中根本没有。
<br/>　　“没有？”
<br/>　　楚风眠注意到了这陈进脸色的变化，不由的开口道。
<br/>　　“千丈大小的须弥空间，我们陈家商会，的确没有，这个大小的须弥空间，不止是我们陈家商会，只怕是阎罗星域之中都没有。”
<br/>　　陈进坦然的回答道。
<br/>　　千丈大小的须弥空间，简直是要比起半仙器还要珍贵，价值直逼极道仙兵。
<br/>　　这样的至宝，纵然是整个阎罗星域之中，都未必能够拥有。
<br/>　　听到陈家的话，楚风眠沉默了下来，脑海之中思考这。
<br/>　　如果没有千丈大小的须弥空间，便是无法带着太古蜈蚣离开，这太古蜈蚣，没有灵智，所以想要让他缩小身躯，根本不可能做到。
<br/>　　可陈家商会没有，按照陈进的意思，只怕是其他商会之中，也不会有这千丈大小的须弥空间。
<br/>　　在这件事上，陈进没有欺骗楚风眠的理由。
<br/>　　楚风眠想了一下，再度开口道。
<br/>　　“那陈家商会之中，最大的须弥空间，有多大的？”
<br/>　　“七百丈大小。”
<br/>　　陈进回答道。
<br/>　　“我买下了。”
<br/>　　楚风眠直接开口道。
<br/>　　“什么价格？我以神品灵石支付。”
<br/>　　神品灵石，可是跟圣品灵石不同，乃是绝对的硬通货，毕竟对许许多多天人境的武者，已经是成仙的人物，神品灵石都有着极大的用处。
<br/>　　这陈家商会当然也不会拒绝，而且在如今这个时代之中，高品质的灵石，已经是无比罕见了。
<br/>　　楚风眠能够拿出神品灵石来交易，这陈家商会来说，也可以大赚一笔。
<br/>　　“神品灵石？”
<br/>　　那陈进都没有想到楚风眠一出手，就是直接以神品灵石交易，他的目光闪烁着，似乎是在思考这其中的价格。
<br/>　　“五百，五百神品灵石。”
<br/>　　陈进想了一下，开口道。
<br/>　　这七百丈的须弥空间，也是陈家商会之中压箱底的宝物了，眼下陈进虽然有着想要结交楚风眠的心思，可不可能在跟楚风眠做赔钱买卖了。
<br/>　　五百神品灵石，这个价格，已经算是偏低了。
<br/>　　楚风眠也不犹豫，大手一挥，一堆灵石便是飞了出来，一枚枚拇指大小的无色灵石组成的一座小山峰，直接飞了出来，正是五百神品灵石。
<br/>　　楚风眠随着实力提升，他现在凝聚神品灵石，也不再像是之前那般的艰难了。
<br/>　　在前往阎罗星域的路上，楚风眠也是在不断的凝炼神品灵石，现在楚风眠的空戒之中，足有数万枚之多。
<br/>　　拿出五百枚来，对楚风眠来说，不值一提，更何况就算是将这数万枚神品灵石全部消耗完，楚风眠在花费一些时间，依然可以凝聚出来。
<br/>　　“不错，真的是神品灵石，而且灵力无比精粹，根本没有流逝，这怎么可能？”
<br/>　　陈进一把接过五百枚神品灵石，感受着其中的灵力，如今都是吃了一惊。
<br/>　　阎罗星域有着星域传送阵，比起大罗天星域贸易要繁华的多，所以陈进倒是见到过神品灵石。
<br/>　　可如今这个时代的神品灵石，绝大部分都是从古老的时代之中，流传下来的，经历岁月，其中都会或多好少的消耗灵力。
<br/>　　而现在楚风眠拿出的这五百枚神品灵石，却是每一枚灵力都没有丝毫消耗，这种品质，在如今流通的神品灵石之中，都堪称极品了。
<br/>　　“我这就去将宝物给楚公子取来。”
<br/>　　收取了神品灵石，陈进也是当机立断，身形一闪突然消失，过了不到半分钟的时间，陈进的身形再度回到了第九层之中。
<br/>　　不过这一次，从他的手心之中，倒是多了一个布袋。
<br/>　　楚风眠一眼看过去，隐隐预约之间，都可以从这布袋之上，感受到一股股玄妙的空间之力。
