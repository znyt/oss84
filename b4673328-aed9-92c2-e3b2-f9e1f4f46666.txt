　　紫霄宫中，道祖鸿钧讲道三千年，众修士的头顶，各种领悟，通过异象显现出来，其中，元始的异象最为惊人，鸿钧看后深感意外。
<br/>　　之后，鸿钧又看向老子等人，具皆点了点头，至于大殿中的其余人，就只是微微摇了摇头，显然，他知道其余人对道的领悟并不深。
<br/>　　元始身旁，是老子，老子的头顶，三花显现，庆云翻滚，天地玄黄玲珑宝塔屹立其中，十一品净世白莲，青莲所化的拐杖，分别占一朵金花，阴阳二气呈现，交融流转，不时汇聚成太极之相。
<br/>　　通天的头顶也是三朵金莲，正是三花聚顶，五气朝元之相，上有一团青气凝聚成剑形，幻化无常，庆云之上，青萍剑、净世白莲莲台翻滚不休。
<br/>　　女娲则是一团白雾之中凝聚成一条巨蛇不断游动，造化玄光显现，至阴之道流转不休，头顶庆云之上，并没有什么灵宝。
<br/>　　接引头顶则是一片金光，内有金身法相显现，有普度众生的慈悲之感，有因果之线，寂灭之相，显然，接引也领悟了因果法则与寂灭法则，只是不知道他的因果与寂灭有了几成。
<br/>　　准提的头顶，一棵巨大的金色菩提树显现，慧光洒下，目露慈悲，又有各种幻像丛生，大梦千秋之象，显然，不止元始一人会梦幻法则，准提也领悟的梦幻法则。
<br/>　　其余众人也各自有属于自己的庆云异象，各自的领悟也在庆云之上显现出来，大殿之内异象纷呈。
<br/>　　资质不同，个人领悟也就不同。除了蒲团上的六人外，帝俊、太一、冥河、镇元子、鲲鹏、红云……等一些个佼佼者也是明悟诸多
<br/>　　众人被大道之音所吸引，不知时间的流逝，直到鸿钧停止讲道良久，众人才清醒过来。
<br/>　　“三千年已至，此次讲道就此结束，万年之后，再次开讲。”
<br/>　　鸿钧望着紫霄宫众人，脸色平静，淡淡道，。
<br/>　　“我等谢过老师，弟子告退。”众人纷纷开口，望着鸿钧道谢道。
<br/>　　此时后土忽然站了起来出声询问，向鸿钧提问道：
<br/>　　“启禀道祖，我巫族没有元神，如何成道？”
<br/>　　鸿钧看了她一眼，淡淡应了一句：“日后自有缘法。”
<br/>　　接着，不等众人反应，就直接消失不见，两个童子童女也随之退入后殿，而关闭了三千年的大门，此刻也被重新打开。
<br/>　　后土无奈，俏脸有些阴沉，而后径直出了紫霄宫去了。
<br/>　　众修士也是开始陆陆续续的离开紫霄宫，返回那洪荒之中，静静体悟此次听道所得的内容。
<br/>　　“道兄，不知可否到我巫族一聚？”出了紫霄宫，这时，后土来到元始三人身旁，对着他们邀请道。
<br/>　　“此次听道祖讲道，吾等大有所得，还需要回去静静体悟，就不去叨唠了，下次有机会，我等再去不迟。”元始婉拒道。
<br/>　　“哎，我巫族的道又在何方啊？”
<br/>　　此次见到这么多大神通者听道，后土对日后巫族的道路担忧不已，脸色不由地有些阴沉。
<br/>　　元始见状，不由安慰道：
<br/>　　“道友不必担心，万物有阴就有阳，你巫族虽说没有元神，但肉体天生强大无比，不修元神，肉身大道又何尝不是一种大道呢，想当初盘古神尊，肉身强大，单靠巨大无比的力量，就可大战群魔。可见，肉身修到极致，力量同样强大无比，更何况，力量法则可是排名第一的大道法则啊。”
<br/>　　“对啊，力量法则同样是强大的法则，我巫族乃盘古父神所化，当不输于任何人。”后土眼睛变得明亮，如同找到了方向，大喜道。
<br/>　　“大谢道友提醒，之前是我太过执着了。”
<br/>　　接着，后土向着元始行了一礼，道谢道。
<br/>　　“道友不必多礼，我等本出同源，这是应该的。更何况，就算我不说，凭借道友的聪明才智这些早晚能够想通的。”
<br/>　　后土笑了笑，仿佛听到有人夸她，她很是高兴一般。
<br/>　　“后土道友，我这里有句话不知当讲不当讲？”
<br/>　　“叫后土多生分啊，以后叫我后土就行。但讲无妨！”
<br/>　　“好吧，后土妹子，巫族虽然为盘古精血所化，有盘古余荫庇护，但余荫虽利，却不能持久。若是有一天没有了这些，尔等又会如何。你还是仔细想想。”
<br/>　　后土若有所思，似有所悟，却又有些不太明白，就对着元始问道：
<br/>　　“你所说何意？”
<br/>　　“不明白？你早晚会明白的，此时多说无益。最后，再奉劝你一句话，小心妖族。”元始轻声对着后土说道，之后，又对着老子与通天道：
<br/>　　“我们走。”
<br/>　　接着，元始三人不管后土两女，直接离开紫霄宫，向着洪荒大地而去。
<br/>　　“这帝俊与太一两只乌鸦跑得倒是挺快的。”
<br/>　　元始看了看四周，发现早就不见帝俊与太一的身影，他还想着讲道结束后找他们算账呢，想不到他们倒是挺识趣的，早早地就离开了。
<br/>　　“大兄，我早就看那两只扁毛雀不爽了，要是哪天碰上，非要将他们打残不可。”这时，一旁的通天开口说道。
<br/>　　“无妨，他们跑不了，洪荒就这么大，我不信他们不会现身，这样的机会以后多的是。”
<br/>　　元始摆了摆手，无所谓道。以他如今的实力，他还不将帝俊与太一两人放在眼里，你可以说他这是高傲，也可以说他这是自大，但元始的确有这个自信，若是连这点自信都没有，那他这么多年大道就白修了。
<br/>　　“大兄，我们如今去哪里？”老子一向是一副淡漠无为的样子，平时话少得可怜，此时却对着元始问道。
<br/>　　“我等先去不周山一趟。”
<br/>　　“干什么？”通天道。
<br/>　　“唔，本座当年在那里种了两株先天灵根，此时想来却是快要熟了。”
<br/>　　老子：“……”
<br/>　　通天：“……”
<br/>　　种了两株先天灵根？要不要这么吊啊！你何时在那里种了先天灵根了，我们怎么不知道？还有，要种为什么要种在不周山，而不是带回去种在家中，在不周山，你不怕别人将它们偷了？
<br/>　　两人真的很是怀疑，那不是你种的，而是你发现了带不走，才会将他们放在那里，如若不然，以你雁过拔毛的习性，他们还能在那里静静待着？
<br/>　　不过，两人不敢将这话说出来，他们只是在心中吐槽一句也就罢了，若真是说了出来，那还不被元始以切磋的理由狠狠地揍一顿啊。
<br/>　　其实，他们猜到还是不错的，这两株先天灵根的确是当初元始不能带走的，那灵根，只适合在不周山上种植。
<br/>　　“走！”
<br/>　　元始带着通天与老子，目的毫不掩饰，径直往不周山飞去。
<br/>(本章完)
