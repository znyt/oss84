　　站在不远处，见九龙雷罡火如此轻易就被慕清霄收取，雷震心中无奈的同时，也感觉到一阵惊骇。
<br/>　　九龙雷罡火中有他种下的灵魂印记，此刻他居然能感觉到，这枚异火榜排名第十二的异火居然传递出一股惊惧的感觉。
<br/>　　让九龙雷罡火感到惊惧，难道是异火榜排名前几的异火？
<br/>　　慕清霄望着手中的异火，道：“不知唐谷主可否将灵魂印记抹去。”
<br/>　　唐震轻轻点头，双手结印，将自己种下的灵魂印记抹去，下一秒脸色就变得苍白了一些。
<br/>　　待灵魂印记抹去，九龙雷罡炎已是无主之物，静静的躺在慕清霄的手心，丝毫都不敢动弹一下。
<br/>　　唐震叹息一声道：“九龙雷罡火已经赠予前辈，还请前辈履行诺言，为在下炼制一枚丹药。”
<br/>　　“我承诺的事情，绝不会食言。”
<br/>　　说罢，慕清霄手中窜出一缕黑色火焰，火焰将九龙雷罡炎包裹住，随后消散不见。
<br/>　　“等你何时将需要炼制丹药的灵药准备好，何时就开始炼丹。”
<br/>　　不经意间看到慕清霄手中的黑色火焰，唐震震惊无比，黑色火焰并且将九龙雷罡炎吞噬，难道是异火榜排名第二的虚无吞炎？
<br/>　　虽然心中有些确信，好奇的同时，唐震并没有开口询问。
<br/>　　闻言，唐震松了口气，道：“灵药已经准备许多时日，不如就请前辈下午就替在下炼制丹药吧。”
<br/>　　……
<br/>　　……
<br/>　　庞大的山谷，绿荫葱郁，在山谷的中央位置，有一座用赤红色火山岩石建造出的宽阔广场。
<br/>　　广场中央有一座高达十丈左右的石台，在广场的周围站着寥寥数人，火云老祖，唐震与其女唐火儿站在石台下。
<br/>　　火云老祖与唐震眼中都充满了期待，他们虽然是斗圣斗尊，但也是第一次见九品炼药师出手炼丹。
<br/>　　至于唐火儿则好奇的望着石台上的青年，丹凤眼中满是好奇。
<br/>　　小医仙美眸望着慕清霄，其中满含温柔，肩头上的彩鳞亦是如此。
<br/>　　慕清霄盘坐在石台上，瞥了眼手中的卷轴，没想到如此好的机会，唐震居然只需要他炼制七品丹药，真是杀鸡用宰牛刀。
<br/>　　当然，慕清霄是觉得无所谓，只要不超过九品，他都可以炼制。
<br/>　　唐震需要炼制的丹药名为‘火菩提’的七品高级丹药，对于他来说，简直小菜一碟。
<br/>　　由于慕清霄的要求，整个广场已经被空间结界封锁，焚炎谷中的弟子并不知道其中的情形。
<br/>　　将卷轴放下，慕清霄面色不变，双手结印间，只见一尊药鼎自储存空间中飞射而出，在空中旋转几圈，携着破风声轻轻的落在石台上。
<br/>　　药鼎并不大，通体青色，丹鼎圆壁上蟠龙刻凤，透着古朴沧桑的气息，刚出现就有一片清香飘荡而出。
<br/>　　未炼制丹药，药鼎就自生其香，只是仅仅嗅上一口，便令人虎躯一震，心旷神怡。
<br/>　　火云老祖眼中闪过一抹震惊，此鼎品级恐怕极高，但他又未曾在天鼎榜上见过，心中顿时充满了好奇。
<br/>　　慕清霄望着身前的药鼎，他也是第一次用神农鼎炼丹，要知道这可是真正的神器。
<br/>　　相比神农鼎，斗破世界天鼎榜上的丹鼎根本不值一提。
<br/>　　将神农鼎摆放好，慕清霄取出唐震事先给他的纳戒，随后取出其中的数百种灵药，紧致有序的摆放在石台上。
<br/>　　灵药出现时，一股浓郁的药香味就弥漫而出，这些都皆是炼制火菩提的材料。
<br/>　　七品丹药已经算是高阶丹药，炼制起来相当麻烦，但对于身为九品炼丹师的慕清霄来说，却算不得复杂。
<br/>　　掌心中窜起一缕黑色火焰，火焰出现时，瞬间将虚空焚烧出数丈的虚无空间。
<br/>　　不管是火云老祖，或者是唐震和唐火儿，眼中皆是震惊无比，火焰出现就将空间给烧穿，威力简直匪夷所思。
<br/>　　更令人奇怪的是，他们居然察觉不到火焰有一丝的温度。
<br/>　　屈指一弹，九霄琉璃焱就射入神农鼎中，旋即一幕奇异的画面浮现。
<br/>　　只见神农鼎圆壁上的龙凤上居然浮现一缕缕青色光芒，仿佛画龙点睛，栩栩如生，龙凤齐鸣……
<br/>　　慕清霄瞥了一眼周身的绿色荧光，心中有些震惊。
<br/>　　周围的绿色荧光居然有凝神的作用，炼丹师炼丹时最注重是灵魂力量的集中。
<br/>　　药鼎中升起火焰，慕清霄盘坐于鼎前纹丝不动，石台上的一株株灵药则在灵魂的控制下源源不断的投入药鼎当中。
<br/>　　药材进入药鼎后，顷刻间被黑色火焰吞噬，随后巧妙的提炼其中的精华。
<br/>　　火云老祖与唐震等人面面相觑，一次性提炼数百种灵药的，他们真是头一低见，九品炼丹师果然不同凡响。
<br/>　　而且，数百种灵药是一瞬间就被焚化，并且其中的精华能量没有一点遗失。
<br/>　　等将各种药材精华提炼出来，慕清霄释放出一丝灵魂力量，将能量全部包裹，随后数百种力量在药鼎中以肉眼可见的速度融合在一起。
<br/>　　能量融合速度极快，没有一丝的停滞，并且其中没有出现任何差池，可见慕清霄在炼丹上的技术和控火技术何其高明。
<br/>　　七品丹药的炼制是一件极其繁琐的事情，唐震在控火上有极高的天赋，但奈何条件不足无法成为炼丹师。
<br/>　　但是，以前靠着九龙雷罡火依旧能够炼制出六品丹药，但七品丹药却是无法炼制而出。
<br/>　　像慕清霄这种，炼丹时看都不看一眼，也不动手，直接用灵魂力量控制一切，这是对自己灵魂强大自信到了何种程度？
<br/>　　石台上，神农鼎中黑色火焰闪烁，慕清霄周围空间都扭曲起来。
<br/>　　第一次用神器炼制丹药，慕清霄感觉非常的好，特别是神农鼎中释放出的绿色荧光，凝神后炼丹师根本没有一丝的疲惫。
<br/>　　甚至，对于控火，对于灵魂的控制极大的提高，对于炼丹师来说，一尊好的药鼎果然是关键。
<br/>　　在慕清霄灵魂力量的控制下，药鼎中色彩斑斓的精华能量已经渐渐融合成粘稠的液体，并且不断自其中渗透出一股奇异的药香味。
