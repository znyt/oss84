　　“老师！”
<br/>　　玉虚宫门之外的风晓月见到元始走出宫门，过来低头行礼道。
<br/>　　数十万年过去了，当初十一二岁的小女孩如今已经长到十七八岁的少女了，一身白色衣裙如莲花般雪白，一头黑发直达腰间，头接白色丝带，脸色红润，亭亭玉立，宛若人境仙子。
<br/>　　“嗯。”
<br/>　　元始淡淡应了一声，之后对着她道：
<br/>　　“不错，已经金仙圆满了，不久便能突破太乙金仙。看来你平时还是听为师的话，刻苦修炼。”
<br/>　　“多谢老师夸奖。”
<br/>　　风晓月对着老师感谢道，小脸之上，洋溢着一丝迷人的笑容。
<br/>　　“为师已出关，去将你三位师兄师姐叫过来。”
<br/>　　元始挥挥手，淡淡道。
<br/>　　“是，老师！”风晓月低声退下，却是去叫元始其余三位记名弟子去了。
<br/>　　不一会儿，风晓月就领着三人来到元始面前，对着元始躬身行礼，道：
<br/>　　“老师好，望老师早证大道！”
<br/>　　元始看着他的四名记名弟子，距如今收徒已经数十万年了，四人从当初的小孩变为如今的少年少女，男的英俊潇洒，女的貌若仙女。而且一身修为也从刚开始的真仙境到了如今的金仙乃至太乙金仙。玄青与玉玲珑甚至已经突破金仙，成就太乙了。尽管这其中有元始的资源帮助，但以如此短的时间到达如今这个地步，也可见他们修炼的确刻苦了。
<br/>　　元始也感到有些欣慰，他们的确不错。
<br/>　　“行了，如今大劫已过，天地之间杀劫渐少，高手不出，尔等是时候出去历练一番了。”
<br/>　　“是，老师！”
<br/>　　四人恭敬道，小脸之上露出微笑，他们都感到很高兴。
<br/>　　之前老师不发话，他们都不敢出去永恒岛外，一直待在岛上，待了数十万年，他们也有些烦了，一直想象着何时能够出去洪荒大地之中走走，如今如愿以偿，他们不高兴才怪。
<br/>　　“尔等也不要高兴太早，洪荒之中可比这永恒岛危险多了，就算如今高手大多消失了，但也难保不会出现几个意料之外的高手，到时候，是生是死，就看你们自己的了，为师可不会多管。”
<br/>　　“此次出去游历，本座只有一个要求，你们每人至少突破一个境界，白衣与风晓月至少突破太乙金仙，玄青与玲珑至少突破太乙中期，不然尔等就别回永恒岛了。”
<br/>　　“尔等之中，玄青作为大弟子为人沉稳，玲珑作为师姐为人细腻，此次历练尔等不要分开，就听玄青与玲珑的。”
<br/>　　“是，谨记老师教诲！”
<br/>　　四人收起心思，变得谨慎起来，对着元始道。
<br/>　　“好了，尔等去吧！”
<br/>　　元始挥了挥手，将四人送出岛外。
<br/>　　玄青四人看了看四周，发现自己四人已经出了仙岛，到了大海之中，看了看四周，一片陌生，不知从何走起。
<br/>　　“师兄，我等如今怎么办？”
<br/>　　一身墨绿色衣裙的玉玲珑对着一身黑衣的玄青说道。
<br/>　　“老师说到做到，说不会管我等就不会多管我等，是故，如今我等只能靠自己。看眼前这种情况，我等是身处东海之上。现在我们就朝着中部而去，等出了东海，再做计较。”
<br/>　　玄青思索了一会儿，对着三人说道。
<br/>　　“好，我等听师兄的。”
<br/>　　说完，四人就朝着中部飞去。
<br/>　　四人走后，元始一步踏出，就消失于虚空之中，等到再次出现，却已经出现在太清岛上。
<br/>　　自从领悟了空间法则之后，元始不仅法力大增，每一个细胞都变成一个巨大空间，之后再以各种法则强化空间晶壁，使之加固，可以说元始的《玄黄不灭》直接跨过五六重，成就第七重虚空成虚境，他如今已经算得上是一般的虚空体了。
<br/>　　而且，他行走的速度变得不是一般的快，压缩空间，跨越空间可比什么飞行之术快多了。
<br/>　　“嗯？”
<br/>　　太清岛上，在元始出现在太清岛上那一刻，太清老子微微皱眉，不过，很快，他就知道来人是谁了，赶紧出来迎接。
<br/>　　“大兄！”
<br/>　　老子出了宫门，对着元始道了一声：
<br/>　　“大兄来此，所为何事？”
<br/>　　“大劫已过，是该出去洪荒转转了。”元始看了一眼老子，淡淡道。
<br/>　　“大兄所言甚是！”
<br/>　　之后，两人再次来到五行岛，约上通天，三人向着洪荒而去，开始了三清游洪荒的经历。
<br/>　　洪荒之中，经过三族大战以及道魔大战之后，变得难得的平静。各族相安无事，没有什么大战爆发，当然，个别小战还是不可避免的，要在这洪荒之中生存，就必须经过战斗的洗礼，才能活在洪荒。
<br/>　　三族之中，龙族自龙祖死后，就潜在四海之中静心修养，不肯出来。不过，龙族还剩一个虚空烛龙以及几个长老级别的人物，但龙族目前掌权的是龙族四太子敖玉。
<br/>　　敖玉在此次三族大战之中并没有出去，而是躲在东海之中修炼，他的父皇与母后全部战死，龙祖的子嗣之中，就属于他修为最强，故而他理所应当的上位了。
<br/>　　凤族之中，凤祖当初以身补洪荒，只剩下残破的真灵飞向南方不死火山，谁也不知道凤凰族还剩下什么，毕竟凤凰可是有涅槃之力的，到底死没有死，谁也不知道。
<br/>　　至于麒麟一族，就只剩下了零零散散的几个族人分布在洪荒各地，下场凄惨。
<br/>　　其余百族，也只剩下了几个低调的种族，三族大战之中，他们没有资格参加，现如今大战结束了，他们就出来闹腾了。
<br/>　　山中无老虎，猴子称大王！
<br/>　　如今三族落败，剩下的几个小族却在这洪荒之中占山称王，打闹不休。不过，他们也闹腾不了几天，就悄无声息了。
<br/>　　洪荒重新焕发生机，大劫过后一元会之中，天道一成清明，不少有识之士，都抓紧时间修炼，道行大有进步！
<br/>　　之后万年之内，大罗金仙境界的高手，如雨后春笋般冒了出来。
<br/>　　不周山女娲、伏羲兄妹，西方接引与准提，万寿山有镇元子，火云洞有红云，太阳星有帝俊、太一，北冥有鲲鹏，血海冥河等，这些都是大罗金仙境界的高手。
<br/>　　甚至巫族也出来征战苍穹了，十二祖巫威名传遍洪荒。
<br/>　　就连元始三人，经过万年时间的洪荒游历，也名传洪荒了。
<br/>　　而且元始也听到了自己四位徒弟的名号，不动玄青、倾城白衣、天音玲珑与月舞仙子。
<br/>　　一时之间，群雄争锋，洪荒强者无数！
<br/>　　这万年时间，元始三清从东海出发，去过东方群山，也去过西方大战后的破灭之地，去过南方火山森林，也去过北方冥海之眼，还到过不周神山。游过大江，寻过仙山，访过荒古遗迹，到过寂灭之墟。
<br/>　　有过游历，有过战斗，有过感悟，有过收获！
<br/>　　灵根寻了一大堆，都是顶级灵根，不是顶级的他们还不要。先天灵器也遇到了好几件，有的是与人争抢过来的，而有的是在绝地之中捡到的。
<br/>　　万年之后，三人再次回到昆仑神山。
<br/>(本章完)
