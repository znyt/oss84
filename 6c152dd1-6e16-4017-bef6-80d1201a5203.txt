　　“轰隆隆！”
<br/>　　突然，三方世界在一瞬之间，猛然轰击在一起，玄天携带全身的力量，精气神汇聚，发出神威，进行硬撼。
<br/>　　神光璀璨，爆炸声不断，玄天在咳血，但他依旧没有放弃，他在尝试，三口巨大的洞天世界，在此刻不断碰撞着，化成一片。
<br/>　　巨大的动静，惊天动地，火光冲天，光芒四射，神威莫测，这一刻的玄天，宛若一尊天神。
<br/>　　洪荒之中的大神通者们，纷纷被这巨大的动静给惊醒了。他们纷纷向着人族首阳山看来，看到的却是人族武祖玄天的疯狂。
<br/>　　“天啊！武祖这是在自损根基吗？”
<br/>　　“这个家伙疯了吗！怎会如此疯狂？”人们目瞪口呆，这也太吓人了，这是要将自己废了吗。
<br/>　　“玄天，他这是……在冲击混元境界！”有人才出玄天的目的，目光顿时变得凝重起来了。
<br/>　　“又是一个想要突破混元的强者，人族玄天，能否如之前的万劫魔尊那般成功突破？”
<br/>　　一个个大神将眼睛瞪得老大，看着玄天的动作，毕竟，玄天此刻所经历的，便是他们以后所要经历的。
<br/>　　玄天不言不语，盘坐在那里，仿佛与外界隔绝。沉浸在自己的世界之中，竭尽所能，想要融合天地人三方洞天世界。
<br/>　　“轰！”
<br/>　　经过不断的融合，不断的咳血，终于，事情出现了变机，三方洞天巨震，化成一片，交融在一起，化成一方天地！
<br/>　　众人震撼，那是怎样一种奇景？
<br/>　　恍惚间，人们似有见到一道道虚影透出，灵山巍峨，秀峰青翠，神瀑一道道飞下，彩霞满天飞洒，瑞兽与珍禽出没，神龙腾云，九凤飞舞，朱雀啸天，白虎飞跃，玄武覆海，麒麟镇世……
<br/>　　这……简直就像是一片仙域啊！
<br/>　　这是虚影，洞天世界内演化出来的虚影，表明洞天内的确有这样是事物，如今这是透过仙光投射了出来。
<br/>　　玄天双目大睁，电芒飞射，神采飞扬，身上神威莫测，气势凌人，就仿佛一尊无双魔王降世，镇压诸天。
<br/>　　天、地、人三大洞天合一，化成了一方宏大的天地，广袤无边，仿佛可以容纳下日月星辰，诸天万界，尽在其中。
<br/>　　“好强悍啊！”众神在赞叹，在感慨。
<br/>　　“武道，不愧是霸道之道，一不小心，便会将自己活祭了。不仅对别人狠，对自己更狠！”有人感慨道。
<br/>　　他的话音刚落，那片仙域世界便剧烈震动，三方洞天并没有真正交融在一起，彼此之间碰撞，竟然要破开此地。
<br/>　　连带着玄天的肉身也在摇动，受到巨大冲击，他嘴角不断咳血，这比刚才还要严重。
<br/>　　三大洞天世界一起抖动，若真是撞在一切爆炸，那么他必然形神俱灭，纵然他手段再如何高超，也要陨落。
<br/>　　“给我合！”
<br/>　　玄天一声冷喝，脸色冷峻，尽管他受到巨大重创，但他依旧没有放弃丝毫，心中的信念也没有动摇分毫，反而更加坚定了。
<br/>　　玄天静心凝神，牵引无尽精气冲来，补充自己，让三方洞天重新排列，释放出不朽的波动，垂下无尽精气长河，浇灌在身上。
<br/>　　“轰！”
<br/>　　不断熔炼，反复交融，最终三方洞天终于归一了，成为一方浩大的仙域世界。
<br/>　　玄天没有就此罢手，他继续凝神静气，将体内的三百六十五颗星辰金丹引入到仙域之中，他曾经参悟过妖族的镇族大阵三百六十五周天星辰大阵，此刻，天上的星辰，正是以此而排列，有着玄奇的轨迹，神妙不可言。
<br/>　　金丹化为星辰，他体内的三百六十五道金丹空间，随着世界的形成，也纷纷演化出一方方小世界。
<br/>　　天上星辰聚集，清气一震，下降的速度慢了许多，玄天手捏印诀，阴阳五行大道运转，天地变厚，大地凝实，五行相生，阴阳相合，演化天地。
<br/>　　随后，玄天又将一些天才地宝抛进世界之中，滋养着世界。
<br/>　　灵气如雨，大地土黄，海水碧绿，山峰有灵，古树苍郁，花草遍地，瀑布飞落，充满无尽生命之力，一片生机勃勃，生气盎然的景象。
<br/>　　自此，一方充满无限可能的世界渐渐成形了，玄天的武道境界，也正式突破了世界境，也就是混元大罗金仙境界，不过，武道上称呼此境为“极道帝境”。
<br/>　　帝者，老子第一，老天第二！
<br/>　　武道世界境，便是要不断完善体内的世界，蜕变一次，便是突破一重天，直至蜕变十二次，最后才可形成三千大道俱全，天地人三才平衡的完美世界。而想要成为真正的不朽强者，便是要将世界蜕变成真正的永恒真界。
<br/>　　何为永恒真界？时间不能留下痕，岁月不能灭其身，天地唯一，永生不朽，亘古如此！
<br/>　　当然，距离那个目标，玄天还差得远，他如今也只是开辟一方中千世界而已，距离大千世界，都还差得远，更不用说完美世界了，永恒真界什么的，那就更远了。
<br/>　　世界开辟，天道境之前，玄天的混元之路，却是被铺平了，今后只要不断积累，不断完善世界规则，便没有什么瓶颈的阻挡，他可以轻松突破混元三重天，六重天，九重天等境界的壁垒。

<br/>　　轰隆隆！
<br/>　　忽然，天地剧震，天上的星辰不断坠落，日月开始不稳，虚空开始抖动，大地龟裂，一片黑烟升腾，清浊二气似有相合的趋势。
<br/>　　“看来还是需要天柱的支撑啊！”
<br/>　　玄天叹息一声，顿时明白了什么，他眼珠一转，却是有了主意。

<br/>　　轰！
<br/>　　世界一震，一股无形的波动荡漾开来。世界仿佛变得不一样了，变得更加凝实了，天上星光璀璨，闪耀星辉，日升月落，井然有序，大地聚合，山脉沉重，四季轮回，一切的一切，都恢复了原样，再无后顾之忧了。
