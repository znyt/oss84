　　在场众人，包括苏钦章在内，都是目瞪口呆，不敢相自己所看到的。
<br/>　　但是道风还没完，杀了悦然之后，转过身来，全力对付神道清。
<br/>　　神道清入门比悦然早十几年，法力更强了不少，与道风僵持了一会，不过最后也被完全压制，最终被五朝元气缠住周身，肉身被绞成了粉碎……
<br/>　　不过他也算是早有准备，借助着肉身的自/爆，一缕魂魄飞向远处。
<br/>　　“道风，你好狠啊！！”
<br/>　　空气中传来神道清凄惨的声音。
<br/>　　“今日之事，我法术公会，与你不死不休！”
<br/>　　道风也没有追他，很平静地目送他离去，这才低头看着地上悦然的尸体，唤道：“小苏。”
<br/>　　苏钦章回过神来，急忙走过来，说道：“大师兄！”
<br/>　　“找人把这尸体先埋了。”
<br/>　　苏钦章呆呆地看着尸体，有点心惊肉跳，喃喃道：“埋在哪里呢？”
<br/>　　“先埋了，会有人找你来领尸体的。”
<br/>　　苏钦章急忙召来两个弟子，吩咐他们不要乱说，然后将尸体往后山抬去。
<br/>　　“不要埋在后山，玷污了我茅山的山水。”
<br/>　　苏钦章一愣，只好又让两人把尸体抬到前山，随便找个没人的地方掩埋。等尸体抬走之后，苏钦章这才站在道风身边，傻傻地看着他。
<br/>　　对面那些各派宗师，大部分也是表情呆滞，傻傻看着道风。
<br/>　　谁也没想到，道风出手会这么不讲情面，说杀就杀了。
<br/>　　进入现代社会之后，有法律的约束，法师再怎么牛逼也属于人类，杀人是犯法的，因此法师之前，就算再有仇也不会真的杀人，实在不行就作法，不给自己留下证据，以免后患。
<br/>　　而道风，就这么众目睽睽地杀人……虽然他已经不是人类，人间的法律不能把他怎么样，但终归有点触目惊心，最最重要的是，他杀的不是一般人，而是星月奴的弟子……法术公会的几位明日之星之一，就这么让他给杀了，还斩杀了神道清的肉身……
<br/>　　大伙一时间都不知道说什么好，怔怔地看着道风，直到道风先开口：“玉玑子。”
<br/>　　玉玑子浑身一颤，硬着头皮从人群中走出去，看向道风，想要做出凶狠的表情，但是眼神中只有恐惧。
<br/>　　是的，在悬空寺一战的时候，自己在道义上至少站得住脚，但是现在……玉玑子突然想到什么，说道：“道风，我听说你去昭狱劫走了白起的元神，阴司已将你列入通缉犯人，你……”说到这，玉玑子实在不知道该说什么，劝人去自首？还是责问他为什么要这么做？
<br/>　　玉玑子正在踌躇，道风反问了一句：“关你什么事？”
<br/>　　玉玑子一下就懵了，是啊，关自己什么事，自己只是个人间法师，又不是地府阴神，最多从道义上谴责他一下，并不能真的把他怎么样？
<br/>　　道风往他面前走了几步，玉玑子吓的立刻往回缩。
<br/>　　道风站住，面冲他说道：“今日之事，我知道是你引他们来的。你是不是觉得我师父死了，少阳不在人间，茅山就可以任你欺负了？”
<br/>　　被当着这么多宗师的面责问，玉玑子自己也是一派掌教，这面子实在过不去，脸色一阵红一阵白，最后一咬牙说道：“道风，你铲了我山门，杀了我师弟和最好的师侄，这笔帐，我就是拼了老命也要跟你算！”
<br/>　　要是叶少阳在这，肯定要跟他辩解一番，证明当初杀玉辰子和凌宇轩是理所当然，不过道风没有辩解，他不屑，你说他杀人也好，说他入魔也罢，他从来就没有辩解过。
<br/>　　面对玉玑子的责问，道风点点头，道：“很好，来吧。”
<br/>　　来……
<br/>　　玉玑子一下就懵了。
<br/>　　“是啊，你说要报仇，今日我就在这里，你不来找我报仇吗？”
<br/>　　玉玑子愕然。
<br/>　　人群中爆发出一声憋不住的哄笑，也不知道是谁发出的，急忙就止住了。
<br/>　　玉玑子一张脸红成了猪肝色，犹豫半天，说道：“道风，你这算盘打的不错，你明知道我不是你对手，让我跟你决斗，不是找死么？”
<br/>　　道风一脸面瘫的脸上也被他逗的露出了微笑。“玉玑子，你自己说要报仇，我现在给你机会，你又说不是我对手……那你说这怎么办？”
<br/>　　这又是一句让人没法接的话……玉玑子只恨地上没有个地缝，让自己能够钻下去，道风怎么说也是自己的晚辈，如今被他用这么傲慢的姿态诘问，自己实在不知道应该怎么回应……
<br/>　　道风这时候却再度说道：“我只问你一句，想死，还是想活？”
<br/>　　“什么！”玉玑子一怔，“道风，你不要逼人太甚！”
<br/>　　“想死，还是想活？”道风眼中逐渐露出杀气，同时手中的打神鞭开始有青光渗出……
<br/>　　这打神鞭本来就是昆仑派的东西，玉玑子自然知道，这代表着打神鞭的灵力被激发，随时可以动手。
<br/>　　他根本不是闹着玩的。
<br/>　　就在刚刚，他才对法术公会的人动手，一死一伤……连他们都敢杀，玉玑子毫不怀疑他会对自己动手。
<br/>　　“诸位，难道你们就怎么眼睁睁看着道风对我动手？”实在没办法，脸皮也不要了，玉玑子开始对身后众人求援。
<br/>　　因为张无生站在最前面，玉玑子一回头，第一个看见的就是他。
<br/>　　看着玉玑子求助的神情，张无生的眼神充满了鄙夷。“若我知道，法术公会是你引来的，我绝不与你同行，现在人要杀你，你倒是想起找我们帮忙了。呵呵。”
<br/>　　他这一声呵呵，玉玑子心中就没底了，急忙说道：“不是啊，我也是为了法术界，神道清说，愿意将轩辕剑拿出来作为彩头，奖励给龙华会上选出来的信任首席弟子……这件事，我还没有说呢。”
<br/>　　“轩辕剑……”不止一个人叫出声来。
<br/>　　道风本来是不想听他磨叽的，听见“轩辕剑”三个字，也是止住了想要继续逼他的念头，陷入思考中。
<br/>　　轩辕剑，当然不是一个游戏的名字——道风当初在茅山下的网吧里也玩过这款游戏。
