        季思妍依偎在傅昼景的怀中，双手紧紧地箍住他的腰畔，仰起来的面庞，略施粉黛，在大厅的灯光下泛着一缕白光。
<br/>
<br/>        比起五年前那素颜清纯的样子，现在的她要成熟妩媚了许多。
<br/>
<br/>        “我已经对真相不感兴趣了！”傅昼景再次推开季思妍，收手干脆利落。
<br/>
<br/>        他的声音仍旧非常冷淡，似乎对她完全不念旧情。
<br/>
<br/>        季思妍身心一怔，瞪着眼睛，有些难以置信地看着傅昼景。
<br/>
<br/>        她的双手紧握成拳，却分别无助地捶在雪白的大腿两边，而那做了美甲的指甲也深深地嵌入了手心。
<br/>
<br/>        这手心里被指甲刺痛的感觉，远远比不上他此刻对她的冷淡，让她心痛得难以呼吸。
<br/>
<br/>        傅昼景不再跟季思妍二话，甚至连看都不愿再多看她一眼，而是再次绕开她，径直往前走去，只身离开了大厅的正门。
<br/>
<br/>        今日，傅氏集团未来继承人傅夜沉第一天例行检查，旗下子公司各部门的负责人均得去总部，也就是傅氏集团大厦报到就位。
<br/>
<br/>        傅夜沉虽然还未正式上任掌管傅氏集团，但是大家都像供着太子爷一样，对他阿谀奉承的。
<br/>
<br/>        傅氏集团大厦里，他所经之处，不断地响起一句毕恭毕敬的问候：“傅总好”。
<br/>
<br/>        甚至在大家的私底下，已经改称傅昼景为“小傅总”了。
<br/>
<br/>        在大家心里，谁的分量重，谁的分量轻，掂量得一清二楚。
<br/>
<br/>        而傅昼景和季思妍在地利置业大厦正厅里的小插曲，被正在这傅氏集团大厦总部里等着开会的姚明娜得知了，这一股子的酸味就像她是一坛被打翻的陈年老醋。
<br/>
<br/>        她姚明娜是傅昼景的什么人，大家都知道。
<br/>
<br/>        这姚明娜不仅和傅昼景是青梅竹马，而且和傅氏集团未来的继承人傅夜沉从小就认识。
<br/>
<br/>        公司里总有那么一个专属于女人的八卦群，傅昼景和季思妍在一楼大门正厅里纠缠那么几分钟的时间，就被路过的好几个女职员看在眼里，并拍了短视频发到了群中。
<br/>
<br/>        姚明娜记得，视频里的这个女人不是别人，正是五年前抛弃了昼景的初恋情人——季思妍。
<br/>
<br/>        她真的没想到，这个季思妍还有脸回来找昼景！
<br/>
<br/>        就在姚明娜看完手机短视频后刚收好手机，那边电梯门一开，“叮”地一声，吸引了她的注意力，抬眸时，就看到傅昼景和公司里的一群高管一同从电梯里走了出来。
<br/>
<br/>        姚明娜完全不顾旁人的眼光，径直走到傅昼景的面前，抬头挺胸，醋意十足地质问：“昼景，你跟那个季思妍是怎么一回事？”
<br/>
<br/>        她真就想不明白了，之前那个什么叫筱颖的女孩子跟他纠缠不清的时候，是她误会了，她无话可讲，那现在呢？
<br/>
<br/>        这个季思妍大庭广众之下，在公司里跟他拉拉扯扯，是几个意思？
<br/>
<br/>        傅昼景剑眉微蹙，直接无视掉姚明娜，和身旁的几个高管人员一起往会议室里走去。
<br/>
<br/>        姚明娜是因为他们家和他们傅家是世交的缘故，爷爷傅炎廷才同意让她进他们公司里工作的。
<br/>
<br/>        现在，她爬到财务部副部长的职位，也是爷爷的意思，但是她却总是在自作多情，误以为是他傅昼景在暗地里提携了她，她还一厢情愿地缠着他不放。
<br/>
<br/>        从小时候起，一直到现在，她缠了他整整十五年。
<br/>
<br/>        一年前，她还差点闹到他爷爷那儿去，嚷着要他爷爷给冷姚两家联姻，让他傅昼景娶了她姚明娜。
<br/>
<br/>        好在，爷爷傅炎廷不是那种非得讲究门当户对的老古董，他一句对她姚明娜没感情，爷爷便就此作罢了。
<br/>
<br/>        但是，姚明娜一直对他不死心。
<br/>
<br/>        傅昼景打心底地不喜欢姚明娜这种爱闹腾的女人，他喜欢安静点、乖巧点的，比如，筱颖这种类型的女孩。
<br/>
<br/>        姚明娜继续不依不饶地跟在傅昼景身后，就连开会的时候，她都选择跟傅昼景坐在一块。
<br/>
<br/>        傅昼景坐在会议室里，思绪开了一会儿小差，直到傅夜沉在爷爷傅炎廷的“老忠臣”郑忠威的陪同下一起走进来时，他才跟着大家一起纷纷自觉地从椅子上站了起来。
<br/>
<br/>        在场，第一次见到傅夜沉的女人们，一个个从惊艳到咋舌，不过数秒。
<br/>
<br/>        傅夜沉身着一件高定版银灰色的西装，剪裁合体，衬得他双腿修长。
<br/>
<br/>        小麦的肤色，让他看上去既有成熟内敛的风度，又有俊逸脱俗，犹如阳光般硬朗的气质，身上还微微散发着一股独特的男人魅力。
<br/>
<br/>        就连姚明娜也看出了神。
<br/>
<br/>        夜沉哥好帅！
<br/>
<br/>        这是她发自内心的感慨。
<br/>
<br/>        姚明娜依稀记得，自己大概有七八年没见过傅夜沉了吧！
<br/>
<br/>        没想到，傅夜沉的变化竟然如此之大。
<br/>
<br/>        话说回来，他们傅家兄弟两，天生就是大帅哥的潜质。
<br/>
<br/>        不过，光长相来看，傅夜沉确实要比傅昼景更帅一点。但是，傅昼景却比傅夜沉更受女孩子欢迎。
<br/>
<br/>        因为，傅昼景似乎更懂女孩们的心思，性格温和，不像傅夜沉那般冷酷到总是拒女孩子于千里之外。
<br/>
<br/>        令姚明娜印象最深刻的事，也是这兄弟两之间的区别最有对比性的一件事情就是——
<br/>
<br/>        他们十三岁那年，她的好朋友、当年初中部的校花李忆莎，自信满满地拿着粉色情书和爱心巧克力，在情人节那天，当众在校道上拦住了他们兄弟两的去路。
<br/>
<br/>        莎莎红着脸，鼓足了勇气，娇羞动人地向高了她们一个年级的傅夜沉表白。
<br/>
<br/>        结果，傅夜沉无视掉莎莎的存在，与她擦肩而过。是傅昼景如沐春风般地微笑着，接过莎莎手中的粉色情书和爱心巧克力，替莎莎化解了被拒绝的尴尬。
<br/>
<br/>        外边传闻莎莎是向傅昼景表白，而实际上，莎莎是向傅夜沉表白被拒了。
<br/>
<br/>        知道事情真相的人，除了他们三，就只有她姚明娜。
<br/>
<br/>        因为，当时她也在场，是她陪着莎莎去给傅夜沉送情人节礼物的。
<br/>
<br/>        所以，女孩子们一般对傅夜沉是一种“可望而不可即”的态度，相比之下，自然而然地就更加亲近傅昼景了。
