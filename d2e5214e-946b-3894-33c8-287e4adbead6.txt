　　运用罡煞合一秘法后，剑气符更加复杂，颜色也由原本的灰色变成白色。
<br/>　　虽然变得更难凝聚，却意外的容易操控。
<br/>　　坐在小竹屋内，苏重拿着断剑充当媒介吸引凶戾之气。
<br/>　　等一丝冰冷凶悍力量冲入体内，他立刻调动聚元内力，运转罡煞合一秘法扑在戾气上。
<br/>　　就好似凉水浇在烧红的铁板上，耳边传来呲的一声响，聚元内力瞬间被蒸干消失。
<br/>　　凶戾之气却没有丝毫变化。苏重对此习以为常，体内海量聚元内力前仆后继，不断涌入凶戾之气。
<br/>　　等他耗费大半聚元内力后，凶戾之气才稍微有了点儿变化。
<br/>　　灰色逐渐变浅，有了点儿发白的意思。如脱缰野马般的扭动挣扎也慢慢平息。整条凶戾之气充满一种活泼灵动感觉，好似有了意识。
<br/>　　苏重长出一口气，睁开眼稍微休息了一会儿。
<br/>　　“还真是凶险。”苏重感慨。
<br/>　　尽管有罡煞合一炼化凶戾之气，但在炼化之前，它仍然危险十足。
<br/>　　从吸入体内，到初步掌控折断时间。凶戾之气暴躁异常，一旦失去控制，立刻就会反噬苏重。
<br/>　　这半年修炼，他有好几次都是因此而受伤。
<br/>　　只有等炼化戾气，他才能再利用聚元内功修复身体损伤。
<br/>　　稍微恢复了一会儿精神，苏重先把刚才消耗的聚元内力练回来。然后才继续炼化凶戾之气。忙活了一整夜，直到第二天天光微量，苏重才把凶戾之气彻底驯服，凝聚出一枚庚金剑气符。
<br/>　　“凝聚一丝剑气符，需要近乎其百倍的聚元内力才能完成罡煞合一，这还不包括滋养身体所需。一旦炼化出现失误，又要大量消耗聚元内力。”苏重月琢磨越觉得不是个办法。
<br/>　　庚金剑气符修炼基本进入正轨，但聚元内力却成了消耗大户。尽管聚元功量大修炼快，但总体而言，修炼效率有些低。
<br/>　　“要么想办法提高聚元内力品质，要么加快聚元内力汇聚速度！”苏重有了想法。
<br/>　　聚元功还得继续修改。
<br/>　　走出竹屋，苏重深吸一口气。大竹峰后山空气清新，忙活一夜的疲惫精神为之一畅。
<br/>　　突然听到一阵当当当敲击声。
<br/>　　苏重对这声音非常熟悉：“这是谁还在砍伐竹子？”
<br/>　　有了聚元功滋养身体，田不易已经不太在意底子们是否砍伐黑竹。
<br/>　　自己那几个师兄就没一个勤快的，有聚元功作弊修炼，谁还劳心劳力的去砍竹子。
<br/>　　循着声音走去，七拐八拐，苏重很快找到声音来源。
<br/>　　“小凡，你怎么还在砍伐黑竹？”苏重诧异。
<br/>　　他这段时间一直忙活着自己修炼，对这位同乡兼世界主角，一直没怎么关注。
<br/>　　“你怎么不去练聚元功？”苏重好奇。
<br/>　　聚元功经过苏重改造，传授给大竹峰师兄弟们的功夫已经大不相同，取消了快速凝集聚元内力的功效，但增强了滋养身体的性质。
<br/>　　只要按部就班修炼，身体素质很快就会提升。
<br/>　　而且聚元功不只是盘膝静坐，还有相应的导引拳法，同样能锻炼体魄，怎么都比砍伐黑竹效率高。
<br/>　　张小凡挠了挠头嘿嘿一笑：“我……我修炼聚元功没什么效果。”
<br/>　　他不敢对其他师兄们说，怕师兄师姐们笑话他笨。但苏重不一样。
<br/>　　苏重这具身体同样出自草庙村，和张小凡是自小的玩伴关系很好。张小凡没想隐瞒直接开口道出原因。
<br/>　　“聚元功没有效果？”这怎么可能呢。
<br/>　　即便是世俗烦人修炼，也能用它练出一身同皮铁骨。更何况此地是青云大竹峰，天地灵气浓郁草木精气丰厚。
<br/>　　苏重上下打量张小凡：“走，去我那里，你当我面练一边我看看怎么回事。”
<br/>　　苏重有些好奇，难道是大梵般若的原因。张小凡被梵音寺四大神僧普智传授大梵般若，旁人不知道他可是非常清楚的。
<br/>　　难道自己的聚元功和大梵般若也有冲突？
<br/>　　回到竹屋，苏重把张小凡按在蒲团上，让他修炼。他却站在背后，一手搭在他肩膀上感知张小凡修炼情况。
<br/>　　很快，熟悉的聚元功波动从张小凡身上传来。草木精气天地灵气一股脑钻进张小凡体内。
<br/>　　很快就修炼出一大堆聚元内力。
<br/>　　相比于苏重的聚元内力，新品种聚元功品质要稍微好一些。因为他不只是炼化草木精气，还有天地灵气在其中。
<br/>　　苏重自己留不住天地灵气，不代表其他人不行。他当初修改聚元功，就已经把这种情况考虑进去。
<br/>　　“一切正常啊？怎么会没有效果。”
<br/>　　只要修炼出聚元内力，让其在体内慢慢运转，就能逐渐强化身体。等到聚元自己消耗干净，就再次汇聚一丹田内力继续改造身体。
<br/>　　开始的时候，聚元内力消耗的会很快，然后回越来越慢，甚至需要动用秘法，主动把聚元内力的打散融入体内。
<br/>　　到最后身体再也无法吸收聚元内力的时候，聚元功也就算炼成了。
<br/>　　多的聚元内力也不用理会，太极玄清道修炼出来的法力更加精纯，很快就能把多余的聚元内力融合吸收。
<br/>　　张小凡修炼一切正常，但他的身体素质却增长缓慢。
<br/>　　除非张小凡体内的聚元内力数量不够，果然是因为大梵般若吗？
<br/>　　张小凡吸收的聚元内力，全都被打饭班若给排出体外了。
<br/>　　“二蛋，我是不是太笨了。”张小凡一脸沮丧。
<br/>　　苏重脸皮一抽，自己这个二蛋的外号算是甩不掉了。
<br/>　　“不，你一点儿都不笨。”苏重可是知道，眼前这是位面猪脚，是个内秀的家伙，是个一人修炼五卷天书的人物。
<br/>　　“你先停止修炼太极玄清道，专心修炼聚元功。大约也就半年时间，就能把聚元功修炼圆满，到时候你在继续修炼太极玄清道。主要原因，还是修炼功法和聚元功起了冲突。”苏重意味深长道。
<br/>　　张小凡眼神躲闪。起冲突的恐怕不是玄清道，而是大梵般若！
<br/>　　其他师兄师姐们修炼聚元功，可没出现过他的情况。甚至和太极玄清道相辅相成，一个个进步飞速。
<br/>　　看着苏重玩味眼神，张小凡有些紧张。
