　　一脚踏入到传送法阵之中，楚风眠身边的空间场景，便是彻底变幻这。
<br/>　　楚风眠缓缓的睁开双眼，突然便是发现，他来到了一处独特的空间之中。
<br/>　　在他的面前，是一座巨大无比的山谷，一眼望过去，都看不见这山谷的尽头中，到底是存在这什么。
<br/>　　一眼望向其中所足以看见的，便是无穷无尽的黑暗。
<br/>　　“呼！”
<br/>　　一股狂风，突然便是从这山谷之中，猛然的呼啸而出，这狂风的威力，完全不亚于一道风刃，直接向着楚风眠，轰击过来。
<br/>　　楚风眠感觉到这风刃的袭来，猛然大手一挥，但是就在楚风眠调动灵力的一刻，他突然感觉到，这周围空间的法则，甚至是比起外面，比起归墟天中还要更加凝实许多。
<br/>　　楚风眠调调动灵力，便是感觉到了一种极大的压力，不得不动用比起平时要十倍，百倍的力量，才可以调动的起来。
<br/>　　这令楚风眠都是吓了一跳，他居然是来到了这样的一处空间之中。
<br/>　　在这里楚风眠感觉到的法则，比起楚风眠从外面碰到过的所有法则，都要强大的多。
<br/>　　“居然还有如此一处空间，在归墟天中，曾经云羽天宫的势力，位于九域之巅，很有可能这里就是云羽天宫的所在！”
<br/>　　楚风眠一下子便是猜测出来，这到底是一处什么地方。
<br/>　　这周围的法则，对他的压制实在是太过巨大，以至于楚风眠感觉到，在这空间之中，楚风眠甚至是就连飞行都无法做到。
<br/>　　遁光都无法催动，更何况是飞行。
<br/>　　眼前，除了通过这一道山谷，不然的话，根本没有其他道路。
<br/>　　“看来这就应该是云羽天宫的考验了。”
<br/>　　楚风眠心中暗自想到。
<br/>　　这山谷中的狂风，比起刚刚袭来的那一道，还要恐怖的多，一旦是踏入到了这山谷之中，便是会受到无穷无尽的狂风雷霆袭击。
<br/>　　这些狂风，并不可怕，但是在如今这法则的压制之下，每调动一次灵力所需要的，便是十倍，百倍。
<br/>　　想要从容对付这些狂风，就算是楚风眠，都不一定做得到。
<br/>　　“这里是什么地方？”
<br/>　　在楚风眠的身边，一道人影突然出现，正是那剑苍穹，他也是紧跟其后，从天羽城中的传送阵法中，直接追了上来。
<br/>　　紧接着，盘纯，小魔主，以及那些天剑宗，圣王宗，九魔城的弟子。
<br/>　　一道道身影，纷纷出现在了这周围。
<br/>　　“小子，这次我看你往哪里逃！”
<br/>　　剑苍穹刚刚过来，便是一双眼睛猛然看向楚风眠，其中双眼中尽是杀意，突然出手，一剑之下向着楚风眠，轰击过来。
<br/>　　“走！”
<br/>　　楚风眠可没有兴趣，现在跟剑苍穹他们在纠缠下去。
<br/>　　眼前这云羽天宫的宝藏，就在面前，当然是这云羽天宫的宝藏，要更加重要。
<br/>　　要是现在楚风眠跟剑苍穹他们纠缠，就算是杀了他们，但是若是让其他人得到了这云羽天宫的宝藏，那楚风眠可就没地方后悔去了。
<br/>　　楚风眠身形一动，直接脚下一跳，催动魅影身法，顿时一跃之间，冲入到了那山谷之中去。
<br/>　　“他想逃，追上去！”
<br/>　　剑苍穹看到楚风眠踏入到山谷之中去，也是想要催动遁光，直接追了上去。
<br/>　　可是他也突然之间发现，这周围的法则，实在是太过凝实，令他根本无法催动遁光，也是脚下一动，催动身法，追了上去。
<br/>　　那些天剑宗弟子紧跟其后，全部冲入到了山谷之中。
<br/>　　“我们也走！”
<br/>　　盘纯，小魔主，打量了一样这周围空间，也是都明白过来。
<br/>　　在这处空间之中，也就只有这一座山谷，这一条道路。
<br/>　　无数武者，都是纷纷催动身法，直接进入到了山谷之中。
<br/>　　进入山谷，楚风眠便是感觉到，周围的无数狂风，突然之间便是苏醒起来。
<br/>　　好像是刚刚在这山谷之中的狂风，都是在沉睡一般。
<br/>　　如今随着武者，踏入到了这其中，狂风的威力，突然骤然暴涨十倍。
<br/>　　一股股狂风，席卷着无数风刃，全部冲着楚风眠斩杀过来。
<br/>　　“小小狂风，休想伤我！”
<br/>　　楚风眠身躯周围，一股灵力顿时化为无数金色龙鳞，他的远古龙皇身，被彻底激发出来。
<br/>　　包裹这他的身躯，抵挡这周围的无数风刃，楚风眠身形连连闪动，正是从这山谷之中，一阵穿行。
<br/>　　这里还是山谷的外围，狂风的力量还没有达到极致，楚风眠以远古龙皇身，如今还足以抵挡这些风刃。
<br/>　　而在楚风眠的背后，仅仅跟随的便是剑苍穹，以及其他那些天剑宗弟子。
<br/>　　他们紧紧跟随在楚风眠的身后，追逐这楚风眠。
<br/>　　他们四人，如今仅仅依靠在一起，组成剑阵，来对付这其中的无数狂风。
<br/>　　一道道狂风向着他们袭击，却都是被他们的剑气所剿灭。
<br/>　　“此人一个人要抵挡这狂风，肯定会消耗大量的灵力，我们就跟随他，等到他灵力不足的一刻，冲上去杀了他！”
<br/>　　剑苍穹看着楚风眠，眼神中杀意涌现，但是他并不着急。
<br/>　　这山谷，如今一眼望过去，还是深不见底，显然是还远远没有达到尽头。
<br/>　　他们四人如今联合起来，布置剑阵抵挡狂风，跟楚风眠一个人相比，要轻松许多。
<br/>　　如今他们便是等到楚风眠灵力耗尽的一刻，就是楚风眠的死期。
<br/>　　“愿意跟，就跟着吧！”
<br/>　　楚风眠瞟了一眼身后剑苍穹跟那些天剑宗弟子，冷笑一声。
<br/>　　这剑苍穹心中的打算，楚风眠心知肚明，想要等到他灵力消耗殆尽？
<br/>　　楚风眠拥有神脉，灵力无穷无尽，消耗的同时，也在源源不断的补充，根本不可能灵力耗尽。
<br/>　　不过眼前楚风眠首先要得到的，是云羽天宫的宝藏，至于这剑苍穹等人，在得到了云羽天宫宝藏之后，在杀也不迟。
<br/>　　楚风眠的身形，连连闪动，很快便是进入到了这山谷之中的中心。
<br/>　　周围的狂风，逐渐的便是小了不少。
<br/>　　“度过了？”
<br/>　　楚风眠打量了周围一眼，他感觉到这狂风，的确是已经快要消失。
