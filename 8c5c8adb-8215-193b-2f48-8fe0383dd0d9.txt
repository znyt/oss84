　　第364章
<br/>　　樊梨花被拖到一个干涸的河道中，这里大概是怪人的巢穴，空气中散发着臭味，地上散落着各种兽骨。
<br/>　　樊梨花险些没有吐出来，因为恐惧令她暂时遗忘身体的不适。事实上，撑到现在精神还没有崩溃，樊梨花真的有些忍不住佩服自己。
<br/>　　一只狗不会令人害怕，可当一只狗开始直立行走，甚至口吐人言时，便不由的人不怕了。同样，一个人不会令人害怕，但当一个人像一只狗这样生活时，很少有人会不觉得恐惧。
<br/>　　怪人一路上没有说什么话，抵达这巢穴后更加沉默。他只是不动声色的生起一堆篝火，折断树枝将火点旺。
<br/>　　偶尔目光扫过樊梨花，倒不像是窥伺自己美色，像是准备拿自己裹腹。
<br/>　　她深吸一口气，制止住砰砰乱跳的心脏。事实已告诉樊梨花，动手自己根本打不过对方，何况自己现在还受了伤。樊梨花决定用言语打动对方的心，忽悠对方放过自己。
<br/>　　“你为什么要杀人？”樊梨花。
<br/>　　“杀人……很好玩。”
<br/>　　这回答并不令樊梨花意外，但还是吓着了她。当然，樊梨花也杀过很多人，甚至有些本不该死，樊梨花也并不想给自己按个女侠的名头。但从始至终，樊梨花都不觉得杀人是件很好玩的事情。
<br/>　　“杀人是不对的。”樊梨花如此道。
<br/>　　“因为不对，才好玩呐。”怪人回过头，露出一个狡黠的笑容。
<br/>　　这笑容居高临下，有几分嘲讽，似乎暴露某种智商上的优越感。
<br/>　　樊梨花恍然意识到自己搞错了，这人不是某根神经搭错线的杀人狂，他是一个很冷静的杀人狂。
<br/>　　可为何他会冒失的在野外生火，如此岂不是成了黑夜中的靶子。
<br/>　　也或者，他是故意这么做的。
<br/>　　火焰噼啪声响，衬得怪人的身影无比高大。正在此时，一个人影出现在河道上，手摁在剑柄上，眉绽杀机。
<br/>　　怪人几乎是第一时间意识到对方存在，猛地抬头，视线看了过去。
<br/>　　“程大雷？”
<br/>　　程大雷点点头：“你在等我？”
<br/>　　程大雷阅读对方的信息。
<br/>　　姓名：崔弄海（名动一方的绝世恶人）
<br/>　　年龄：41
<br/>　　技能：无
<br/>　　隐藏属性：嗜血者
<br/>　　一开始果然没有估计错，对方是一个可以和自己匹敌的存在，同样的绝世级别高手。
<br/>　　怪人裂开嘴，露出满口惨白的牙齿，如某种凶残的野兽。
<br/>　　“我要在这里杀掉你。”
<br/>　　他将刀扎进樊梨花的掌心，樊梨花爆发出一声惨叫，这几乎是将樊梨花钉在地上。
<br/>　　速度很快，程大雷几乎没有反应过来，然后崔弄海又以同样快的速度扑向程大雷。
<br/>　　毫无章法，只是速度很快，力量很大。程大雷甚至都没有出剑的机会，便被他逼的倒退，胸口重重挨了一下，五指抓破衣服。
<br/>　　程大雷轻舒了口气，此人没有任何技能，竟然能升到绝世，果然……
<br/>　　果然是可以做自己对手的人。
<br/>　　匹夫剑出鞘。
<br/>　　阿飞快剑第一式：直来直往。
<br/>　　疼痛撕扯着樊梨花的神经，可她也是修罗场上厮杀出来的人，疼痛并没有令她神经崩溃。
<br/>　　当目睹到程大雷的快剑时，令她有种惊心动魄的感觉。如今程大雷已经很少有出手的机会，在樊梨花面前又嘻嘻哈哈的，这令樊梨花对程大雷的实力有种误解，觉得也就和自己差不多。
<br/>　　事实上差了很多。
<br/>　　连环快剑，一剑快过一剑，最后剑与人都消失不见，只剩下简单干脆淋漓的杀机。
<br/>　　可程大雷的尴尬，是樊梨花不能理解的。
<br/>　　迈入绝世等级后，程大雷的速度和力量都已大幅度提升。阿飞快剑使出来更加得心应手，面对崔弄海，他已搬出全部实力。而对阿飞快剑来说，若是第一剑刺不中，往后多少剑也同样无用。
<br/>　　一剑快过一剑，可崔弄海躲得更急，身体不停转移着位置，距离程大雷同样是一剑的距离，有时候只偏差毫厘，但却能险险避过程大雷的剑锋。
<br/>　　程大雷突然出剑，若倒刺流星，剑划过崔弄海的脸颊，画出一道血痕。
<br/>　　这是从交锋开始，程大雷第一次伤到对方，但并不是致命伤。
<br/>　　二人分开一段距离，互相对峙，程大雷稳稳提着剑，血从剑尖滴落。
<br/>　　人力有时而穷，哪怕真的是头野兽，也同样会有力疲的时候。而等他力疲之时，只要轻轻一剑就可以洞穿对方喉咙。
<br/>　　“结束了，给你说临终遗言的机会。”程大雷长剑横指。
<br/>　　崔弄海目光落在剑上，眼神流露出一抹忌惮。他突然暴起，跃到樊梨花身边，拔出钉在她掌心的刀，用刀刃抵在樊梨花的喉咙上。
<br/>　　程大雷一惊，想要扑过去，却被崔弄海以眼神制止。
<br/>　　“把你的剑放下。”
<br/>　　程大雷全身肌肉绷紧，身体如一张拉满的弓，却咬紧牙关，克制着愤怒。
<br/>　　“你也一定不想她死吧，现在丢开剑，丢得越远越好。”
<br/>　　“姓程的，不要听他的！”樊梨花怒容满面：“杀了他。”
<br/>　　程大雷握剑的手轻轻颤抖，显然在剧烈挣扎。他突然将剑抛开，赤手空拳立在崔弄海对面。
<br/>　　剑在空中抛出一道弧线，目睹着它远去，崔弄海脸部肌肉松弛下来，显然握着剑的程大雷要比赤手空拳的他少许多危险。
<br/>　　他长身而起，瘫在地上的樊梨花大口大口的喘气。
<br/>　　崔弄海咧开嘴，露出森然的微笑，然后猛地向程大雷扑了过来。
<br/>　　在其面前，程大雷节节败退，失去武器的他，很难匹敌这头人形猛兽。
<br/>　　拳头夯砸过来，五指成抓，握住了程大雷的手腕，随之便想丢樊梨花一样将程大雷丢出去。
<br/>　　然后，回应他的是更加磅礴的力量，崔弄潮的手腕被握住，紧跟着他整个人被拎起来，如丢沙包一样被丢了出去。
<br/>　　崔弄潮全身骨头像是已被摔断，他眼神里流露出一抹震撼，不敢相信程大雷的力气竟然可以和自己抗衡。
<br/>　　程大雷拦在樊梨花面前，活动着骨节，目光落在樊梨花身上。
<br/>　　“你再撑一会，等我收拾了这混蛋。”
<br/>　　
<br/>(本章完)
