　　“大道在上，吾名玄天，今日得创一道，名曰：‘武’！”
<br/>　　一声浩大的声音从不周山山谷出发，传向九天，震动洪荒！
<br/>　　轰隆隆！
<br/>　　天地开阖，混沌弥漫，整个山谷，一尊模模糊糊的身影立于虚空之上，不动如山，伴随着无量光，他仰头望天！
<br/>　　下一刻，于那九天之上，漫天雷光倾泻下来，竟然比成准圣还要恐怖，已经快要接近混元大罗金仙的威力了。
<br/>　　玄天的声音在这片大地上响起，轰动云霄，响彻八荒，比一般人成准圣还要恐怖！
<br/>　　“武之第一境：炼体境！”
<br/>　　一声大吼，天地四方俱颤，人们只看到，宇宙深处，有一尊宛如魔神般的身影缓缓苏醒过来，他全身弥漫着不朽金光，人们只感觉到一股强烈震撼的气血之力，肉身汩汩，如同雷震。
<br/>　　只见那人睁开双眼，金光闪烁，射向虚空，金光所过之处，虚空破碎。他挥起金色拳头，迎向九天神雷，雷光闪烁，撕天裂地，沐浴在雷光之中，他就仿佛一尊战天魔神！
<br/>　　“武之第二境：开脉境！”
<br/>　　这声吼出后，众生只见虚空中的那道沐浴在雷光之中的庞大金人全身变得晶莹剔透，在他的身上，出现一条条红色的线条。那线条从那金人口部开始，向下渐渐延伸到身下，直至四肢，向上延伸到头部，最后布满整个人身。
<br/>　　这些红色线条有粗有细，粗的如大江大海一般宽阔，细的如毫毛一般微小。若是仔细数数，便会发现，这些粗粗细细的线条，加起来正好有十二万九千六百条，正合一会元之数。
<br/>　　同时，与那金人腹部，一声雷霆般的爆炸，顿时出现一片混沌空间。那空间混混沌沌，透露着神秘的色彩，宛如一切之发源地。
<br/>　　这一日，闭关中的众人都被惊醒了，他们纷纷将目光投向虚空，想要知道这是怎么一回事儿？
<br/>　　特别是人族，因为这正是在人族的上空，而且玄天也正是在为人族布道，所以，人族之地中，他们感受最深，众人族脑海中宛若有一部道经在形成，诵经声悦耳动听，仿佛有一种大道在他们心间回荡。
<br/>　　这实在是太过恐怖了，怎会如此？
<br/>　　“武之第三境：炼脏境！”
<br/>　　“咚——咚——咚——！”
<br/>　　这声大吼发出后，只听到，于在那金人身体之中，传来五声震耳欲聋的跳动声。那声音如同心跳声，只不过像是被放大了数千倍，数万倍……
<br/>　　紧接着，白青黑赤黄五色之光于那金人身上弥漫出来，五行金木水火土弥漫周身，缓缓旋转，生生不息。
<br/>　　于那金人的头顶，凝聚出五道雷光，它们分别是甲木神雷、丙火神雷、戊土神雷、庚金神雷与葵水神雷。五道雷光闪烁，犹如五片华盖，传出各种赞颂之音，犹如诸天神佛在称赞祝福。
<br/>　　那金人头顶五色华盖，沐浴雷光，同时也在吸收着九天神雷，壮大自身，众人只觉得他的气息更加幽深浩瀚了。
<br/>　　“武之第四境：开窍境！”
<br/>　　“开窍有灵！”
<br/>　　巨大的吼声，惊天动地，那金人身上，渐渐开始出现星星点点，慢慢布满那金人的全身，这些星点有大有小，每一颗星点，都犹如一颗星辰一般。与此同时，九天之上，星河之中，星光如同瀑布般落了下来，落进那金人的身体之中，使得他身上的星点更加璀璨了。
<br/>　　“怎么可能，他竟然能够吸收星辰之力？而且，他竟然在创法？！”
<br/>　　洪荒之中，各处地界的大神通者都震惊了，道：
<br/>　　“他竟然在创法，完全走出一条自己的路，现在他言出法随，干预大道的轨迹，影响了别人参悟经文与修道的进程！”
<br/>　　一个个大神通者都被惊住了，这简直不可思议！
<br/>　　任何一种修行体系的初CD会有这种异象，让天地敬畏！
<br/>　　“赶紧趁此机会参悟大道，这个时候，大道是最活跃的时机！”
<br/>　　很快，他们就坐不住了，因为他们能够感觉到天地法则在剧烈抖动，大道气息在弥漫，此刻修行，简直是算得上是一日千里啊。
<br/>　　各大神通者们纷纷看着九天之上的金人，眼神一动不动，参悟着大道法则，顺带着参悟这种新的修炼体系。就连元始、老子等人也不例外。
<br/>　　“武之第五境：金丹境！”
<br/>　　“大道为峰！金丹为宗！”
<br/>　　再一声大吼后，天地间变得瞬间金光璀璨，大道金光闪亮无比，之前那些个星点，渐渐凝聚出一颗颗金光四射的元丹，宛若生命初育。
<br/>　　众多人族们，只觉得自己的生命就好似孕育其中，在他们的脑海之中，一篇篇金光四射的经文诵读出来，传遍全身，他们只觉得自己的身体一轻，好似有无数力量在其中，他们变得更加强大了。
<br/>　　这经文，就好似天生适合他们一般。
<br/>　　“武之第六境：神通境！”
<br/>　　“神通为王！”
<br/>　　只见那金人抬手就是一条金龙飞出，轰向神雷，神雷消散。之后，他五指律动，五座闪烁着五行之光的大山飞出，砸向神雷，将它们镇压，吸收，同化。之后，又是一拳挥出，拳势惊天，分碎苍穹。
<br/>　　他一举一动，都是莫大神通，震动乾坤，斗破苍穹！
<br/>　　“武之第七境：洞天境！”
<br/>　　“洞天称尊！”
<br/>　　轰！
<br/>　　一声大吼，金人的上中下三片闪烁着神秘色彩的丹田轰然将破碎倒塌，渐渐演化出三轮巨大的黑洞，黑洞衍生，金丹入内，演化出各种惊天异象。黑洞之中传出剧烈的吞噬之意，九天之上新凝聚出来的神雷，刚一靠近，便被吞噬一空。
<br/>　　惊世异象惊动所有人，所有人都跟着颤抖不已！
<br/>　　“武之第八境：法相境！”
<br/>　　“法武为相！”
<br/>　　轰！
<br/>　　金光闪烁，一尊足有数十万丈的浩大身影出现在洪荒之中，他全身散发着亘古不朽的气息，流露出无尽的威压，他只手摘星辰，一指破苍穹，脚踏山河，如同一尊九天十地的主宰，执掌着众生的生死。
<br/>　　他抬手便是一掌，轰向神雷，神雷破碎，天地具颤。
<br/>　　“武之第九境：转轮境！”
<br/>　　“我命由我，不由天！生死转轮，唯我称皇！”
<br/>　　一声大吼，那尊浩大的天地法相挥手间，便于虚空之中，召唤出一条无穷无尽的岁月长河，岁月沧桑，最是无情！他大手向着岁月长河一划，便截断岁月，一轮飞出，却是乳白色，散发着无尽岁月的气息。
<br/>　　这是岁月轮！
<br/>　　他在一拳轰出，又是一轮飞出，它散发着黑与白的的光辉，弥漫这生与死的气息，一轮划过天际，生死由我掌控！
<br/>　　这是生死轮！
<br/>　　轰！
