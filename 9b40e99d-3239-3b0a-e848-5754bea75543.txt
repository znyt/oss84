<br/>?    陈明抬头看向青狮妖王，危险指数：208（满值560）。
<br/>    
<br/>        青狮妖王现在的危险指数确实降下来了，但是却是有个满值560的提醒，如果说青狮妖王现在的危险指数就是208，那么后面的满值提醒就没有任何意义，所以青狮妖王的危险指数只剩下208，但是绝对不是208的战斗力，这还是一场恶战。
<br/>    
<br/>        陈明之前并没有插手，虽然他有着插手的实力。他就是要给青狮妖王一个错觉，他并不会亲自动手，或者说他亲自动手并不强，只有这样，青狮妖王在进攻其他五位妖王之时才会没有顾忌，不考虑后路，才能将自己弄成现在这副德行。
<br/>    
<br/>        陈明将血海魔刀扛在肩膀上，看向青狮妖王，话语淡淡，“你有权继续说话，因为你说的每一句话都将成为最后遗言。”
<br/>    
<br/>        青狮妖王明显也感受到了不对的地方，他能够清楚地看到血海魔刀之上发散而出的刀气，“你让我很惊讶，你的实力得到了我的尊重，或许，我们应该换另一个角度来看这些问题，你与我皆不是易于之辈，我们都有机会成就大妖王，只要你我联手，这十万大山，何处去不得？至于这五个废物，你又何必执着？”
<br/>    
<br/>        五位妖王都是看向陈明，是的，现在的陈明还有的选，陈明并不能说是惊才艳艳，那只是对于年轻人的说法，他们是妖王。
<br/>    
<br/>        能够用到陈明身上的形容词是深不可测，他的剑法凝聚剑心，他是人族，却是妖族功法先登王位，他的阵法鬼神莫测，一人便可布成大阵，而现在，他手握血海魔刀，至少已经是刀心的境界。
<br/>    
<br/>        就是他们，也不得不心动青狮妖王的提议，青狮确实没有说错，若是他和陈明联手，两人都有成就大妖王的机会。
<br/>    
<br/>        两位有望大妖王的妖王在十万大山的边缘联手，无人可挡。
<br/>    
<br/>        陈明笑了笑，横刀于前，道：“抱歉，我并不接受你这个建议。”
<br/>    
<br/>        陈明单臂拖刀，朝着青狮妖王跑去，血海魔刀之上魔气蔓延，在四象大阵之中凝聚成一团黑云，血红色的刀气狂舞，陈明一刀劈向青狮妖王，直取青狮妖王的胸膛，数百道刀气飞出，青狮妖王挥动手中的聚妖幡，黑色的烈焰与刀气相击于一处。
<br/>    
<br/>        “给我一个解释！”青狮妖王怒吼道。
<br/>    
<br/>        陈明手中握着血海魔刀逐渐下压，“你要我给你一个解释吗？那好，我便给你，我有三个叔叔，两个哥哥，都死在你的口中。”
<br/>    
<br/>        青狮妖王为之侧目，露出一抹警戒之色，“原来是处心积虑地复仇，等待在黑暗中发出致命一击，而现在，你等到了这个机会，就让我来看看，你到底有什么本事来复仇！”
<br/>    
<br/>        陈明道：“既然你想要看看，我就让你好好看看，这将是你完美的谢幕演出！”
<br/>    
<br/>        手中血海魔刀挥动，陈明此刻已经是刀心的境界，四象大阵之中，陈明如同唯一的主宰，他清楚地知道这种感觉，在大阵之中，他是天人合一境界。
<br/>    
<br/>        一刀落下，刀光蔓延十丈，融合数百道刀气，携带着整个大阵的力量朝着青狮妖王攻去。
<br/>    
<br/>        青狮妖王手中的聚妖幡舞动风云，在手中挥舞地呼呼作响，但是刀光之下，他还是被劈的后退三步，将聚妖幡插在地面的冰层上才得以稳住后退之势。“你并不是刀心境界，也不是剑心境界，你是天人合一！”
<br/>    
<br/>        陈明笑了笑，道：“你觉得是天人合一，那便是吧！”
<br/>    
<br/>        在四象大阵之中，他的状态确实很像天人合一，至少效果是一样的。
<br/>    
<br/>        陈明挥刀，每一刀之中都有着整个大阵的大势，携带着无边大势，陈明朝着青狮妖王攻去，借助着血海魔刀之威，他施展血海魔刀的最后一刀，未必弱于一指通玄，而且消耗地法力还比一指通玄少。
<br/>    
<br/>        开启！
<br/>    
<br/>        血红色的刀光汇聚成一片，陈明如同在顷刻间劈出了数刀，青狮妖王手中舞动聚妖幡，如同挥舞一根长枪一般，聚妖幡在手中挥舞成一个圆扇，聚妖幡之中飞，飞出一阵龙卷风，携带着黑色的烈焰，与陈明挥出的刀光撞在一起。
<br/>    
<br/>        陈明可不是凌仙那种小孩子，站在原地和对手远程对轰，静静地等待对手的下一招，只要能杀掉对方的，便是好的。
<br/>    
<br/>        脚踏刀光剑影而行，陈明不过顷刻之间八步冲到青狮妖王的身旁，携带着刀光的一刀直接劈向青狮妖王的下盘。
<br/>    
<br/>        出手果决狠辣之极，他要的只是杀人，陈明的血海魔刀被青狮妖王挡住，此时此刻，陈明打开储物戒，左手之中出现一柄灵剑，素王剑法，一剑长虹！
<br/>    
<br/>        剑光如同匹练，像是一条瀑布倾泻而出，陈明绕开了聚妖幡，直接攻在青狮妖王的腹部。
<br/>    
<br/>        青狮妖王哪里想得到陈明此时此刻的进攻居然有这么迅猛，被陈明一击轰得倒飞，撞碎了一块青石，而他的腹部，此刻也是鲜血累累，甚至看得到腹部的内脏。
<br/>    
<br/>        青狮妖王笑了笑，将自己腹部的内脏塞了回去，手中一团火焰，直接烧灼在自己的伤口上，瞬间封住流血的趋势，将血迹烧干，青狮妖王伸出舌头舔了舔嘴角的鲜血，一头青色的乱发狂舞，青狮看向陈明，“说实话，你给了我很多的惊喜，让我始料不及，陷空山一带，我青狮，愿意称你为我之下最强！”
<br/>    
<br/>        陈明一手刀，一手剑，青狮妖王可以感受到陈明在四象大阵之中的强大，那是天人合一的压力，那种感觉，就像是整个天地都在压迫你一般。
<br/>    
<br/>        “可我青狮，可不会这么容易的死去！”
<br/>    
<br/>        “想要我的命，那你就亲手来拿吧！”
<br/>    
<br/>        青狮妖王道完，手中聚妖幡挥动，“群魔乱舞！”
<br/>    
<br/>        一道道魔气化作一条条黑影，手中握着兵器朝着陈明冲去，如同一条黑色的钢铁长河，朝着陈明冲去。
<br/>    
<br/>        陈明立在数十道黑影之前，笑道：“只是这点程度的法术，可伤不到我！”
<br/>    
<br/>        四象大阵风卷天炎压下，将数十道黑影撞得粉碎，就在黑影之后，黑影不过刚刚破碎，陈明的身形便已经冲了出来，朝着青狮妖王的头颅一刀劈下！
