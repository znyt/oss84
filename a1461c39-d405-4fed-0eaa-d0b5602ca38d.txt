        童筱颖觉得，自己虽然是在大山里长大的孩子，但她却从未因为自己的出身而自卑过。
<br/>
<br/>        外婆在她的心里，一直都是她的超级大英雄。
<br/>
<br/>        不仅可以给她做好吃的，还会给她做好玩的，就连她病了，外婆还可以给她治病。
<br/>
<br/>        “那你外婆除了把医术传授给你了以外，还把医术传授给了谁？”傅夜沉试探性地问。
<br/>
<br/>        童筱颖不假思索地回答道：“外婆教给我的医术，只是一些平常的小病小痛的急救方子。然后教给我妈妈的是医理药理，所以我妈妈才会去开药店。然后是我小姨，她对医术没兴趣，所以，外婆什么也没教会她，而且她嫁得好，不需要跟着外婆学医。要说真正言传身教的话，那就属我二姨了。我二姨，是她们三姐妹里最有出息的。她不仅考上了医科大学，而且还去当了军医。虽然，小时候，我没见过二姨几面。但是，我听我外婆说起过。外婆在带我之前，一直带着二姨在外看病就诊。所以，我二姨的医术一定在我外婆之上。”
<br/>
<br/>        “你二姨会‘蛊术’吗？”傅夜沉不经意间地皱了下眉头，深沉地问道。
<br/>
<br/>        童筱颖点了点头，接着说道：“我们那儿管我外婆叫‘巫医’，我二姨以前是跟在我外婆身边的，自然也懂‘蛊’，这不足为奇。”
<br/>
<br/>        “那筱颖你会不会？”傅夜沉若有所思地问。
<br/>
<br/>        童筱颖微微摇了摇头，忽然间，她回想起那天大哥被下药的事情，蓦地心有余悸了。
<br/>
<br/>        很明显，那天大哥被人下了药，是来自她们那儿的一种“蛊”。
<br/>
<br/>        外婆的《行医日记》和《千草集》里有记载，所以，她才半知半懂。
<br/>
<br/>        童筱颖一边吃着马奶糕，顿时一边狐疑地看着傅夜沉。
<br/>
<br/>        她此刻心里想的是，大哥该不会是在怀疑她对他下了“蛊”吧！
<br/>
<br/>        而且，听昼景前晚那话的意思，大哥在和昼景暗中较劲。
<br/>
<br/>        倘若，大哥发生了意外，傅氏集团和傅家大庄园的继承人，自然就会落在昼景的身上。
<br/>
<br/>        所以，大哥很堤防昼景的话，那么堤防她也是理所当然。
<br/>
<br/>        但是，这种谋害他人性命的事情，她童筱颖可不会干。
<br/>
<br/>        “大哥，我外婆只教会我如何救人，从未教过我如何去害人！所以……”童筱颖欲言又止，突然间发现自己这种辩解，简直就是一种“不打自招，贼喊捉贼”的错觉。
<br/>
<br/>        “你想哪去了？”傅夜沉黑眸微眯，顿时忍俊不禁。
<br/>
<br/>        童筱颖尴尬地笑了笑，继续捏起盘子里的马奶糕往自己嘴里塞，她真希望是她想多了。
<br/>
<br/>        傅夜沉意味深长地看着童筱颖吃东西，觉得她不管什么时候，在他的眼里和心里，都是一副很可爱的样子。
<br/>
<br/>        “随便填填肚子就好，待会中午的午餐还有更多好吃的。”傅夜沉情不自禁地微笑着，抬起手来，宠溺地在童筱颖的鼻梁上刮了刮。
<br/>
<br/>        童筱颖下意识地躲开，睁大了眼睛，怔讷地瞪着傅夜沉。
<br/>
<br/>        傅夜沉僵硬地收回手，脸上笑容渐敛。
<br/>
<br/>        “大少爷，大少奶奶，你两怎么会在这厨房里？”
<br/>
<br/>        门外忽然传来一个纳闷的质问声，打破了两人之间的僵局。
<br/>
<br/>        童筱颖看到一个穿着白色厨师服的男人朝这边走过来，连忙将手里的盘子往傅夜沉手上塞。
<br/>
<br/>        所以，当这厨子走进来后，看到的就是傅大少爷正在“偷吃”他准备的午膳点心马奶糕。
<br/>
<br/>        童筱颖看着厨子咧嘴一笑。
<br/>
<br/>        看样子，这厨子错将她认成苏漫雪了。
<br/>
<br/>        傅夜沉淡定地将盘子放下，会心一笑：“易叔叔，马奶糕的味道不错。”
<br/>
<br/>        “你这小子，小时候进我这厨房里偷东西吃，倒是已经轻车熟路了。这长大了，带着媳妇上门来偷，也不害臊！我那门上那铜锁，这回，你没给我撬坏吧？”易厨师和蔼可亲地打趣道。
<br/>
<br/>        童筱颖怔了怔，有些难以置信地看着傅夜沉。
<br/>
<br/>        傅夜沉自然是知道易厨师认错了人，索性将错就错地微微一笑，云淡风轻地回答：“这不，我这带着媳妇来感受感受一下，我童年时在易叔叔的厨房里享受到的美好时光。”
<br/>
<br/>        “好啦！好啦！你两到别处玩去，我现在要开始准备午饭了。”易厨师笑呵呵地开始下逐客令。
<br/>
<br/>        傅夜沉礼貌性地向这中年男子微微颔首，然后伸手牵住童筱颖的手，拉着她往屋外头走去。
<br/>
<br/>        “大哥，你小时候为什么要进厨房里来偷东西吃啊？”童筱颖难以置信地小声问道。
