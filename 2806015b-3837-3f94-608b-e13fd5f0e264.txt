　　进入城池之中后，苏莫尽量往人流量大的地方走去。
<br/>　　随着走动，他的气息与容貌不断地变化，几乎是一个呼吸一个形态。
<br/>　　“嗯？”
<br/>　　极空城城主，跟着苏莫进入了城池之中，但是，眨眼之间，他却是找不到苏莫了。
<br/>　　“怎么回事？”极空城城主心中一惊，他的神念一直锁定着苏莫的气息，但是，怎么突然就消失了呢？
<br/>　　随即，他急忙寻找了起来，神念笼罩整座城池，但是根本就没有苏莫的身影。
<br/>　　“这下遭了！”极空城城主眉头紧皱，一个大活人怎么可能就这么消失了呢！
<br/>　　突然，他想到了苏莫变化成许焱之事，立刻明白了苏莫肯定再次变换了模样。
<br/>　　“不好！”
<br/>　　极空城城主面色大变，神念立刻锁定四方城门，想要看看苏莫有没有混出城。
<br/>　　但是，四个方向的城门，人来人往，人流量着实不小，他根本就分不清，哪个是苏莫。
<br/>　　而此地不是时空界域，他也没有权利，要求之城之主封锁城池。
<br/>　　“完了！”
<br/>　　极空城城主面如土色，他堂堂六品强者，跟踪一个二品武者，居然跟丢了。
<br/>　　无法找到苏莫，极空城城主也没有离开城池，因为，他要等待永恒神宫的强者来临。
<br/>　　消息已经传回永恒神宫，要不了多久，便有永恒神宫的强者来临。
<br/>　　……
<br/>　　高空之上，苏莫一路疾行，他在那繁华的城池之中，紧紧是逗留了片刻，便离开了。
<br/>　　“这下应该甩了吧！”
<br/>　　苏莫心中暗道，只要不是神王，想要再追踪到他，可能性很低很低。
<br/>　　飞行了一日之后，苏莫停了下来，等待了良久，确认没有人跟踪之后，他才稍稍松了一口气。
<br/>　　他暂时没有放了公孙嘉，没有绝对安全之前，此人还是带着比较好。
<br/>　　就这样，又过了一个多月，苏莫走走停停，辗转两个界域，彻底确认没有人跟踪之后，他才彻底的放心了。
<br/>　　苏莫身形隐藏在一处密林之中，暗暗思索接下来要去哪里？
<br/>　　是去天龙神朝找龙腾等人，还是去神剑山，先将元神给予绫裳。
<br/>　　他不确定，龙腾等人有没有来到神域？有没有在天龙神朝？也不确定回神剑山有没有危险？
<br/>　　沉思了片刻，他意念一动，便将公孙嘉放了出来。
<br/>　　光芒一闪，公孙嘉出现在了苏莫的身前。
<br/>　　“公孙嘉，你走吧！”望着眼前的公孙嘉，苏莫沉声说道。
<br/>　　“你真的让我走？”公孙嘉惊疑不定的望着苏莫，这一段时间，他一直提心吊胆。
<br/>　　不过，经过一段时间的修炼，他的经脉，已经恢复了一半，已经有反抗的力量了。
<br/>　　只不过，因为实力被封印，他想要强行破封，还需要一点时间。
<br/>　　“走吧！我说过，我们无冤无仇，擒你也只是为了脱身而已！”苏莫摆了摆手，意念一动，对方的体内，有浑厚的混沌源力流泻而出，涌进了他的体内。
<br/>　　混沌源力被撤回，公孙嘉的实力，立刻恢复了。
<br/>　　当然，由于经脉的原因，其实力并不能发挥到巅峰。
<br/>　　公孙嘉见此，微微松了口气，他没有立刻离开，而是凝视着苏莫，沉声道：“秦陨，说实话，你的实力非常逆天，战力超过了一般的万古天骄，算是我见过的最厉害的天才。”
<br/>　　公孙嘉目光凝视着苏莫，心中有些感慨，虽然被擒拿，之前让他很愤怒。
<br/>　　但是，现如今，苏莫遵守诺言放了他，他的怒气已经消了大半。
<br/>　　他生平最敬重天才，和很多万古天骄都相识，这也是他帮助慕容雷利的原因。
<br/>　　虽然苏莫让他难堪，他依旧佩服苏莫的天赋和过人胆识，真非常人可比。
<br/>　　“你想说什么？”苏莫望着对方，他已经放了对方，对方居然还不走？
<br/>　　“我想说，终极神王殿你还是不要回去了，加入我永恒神宫如何？”公孙嘉朗声道。
<br/>　　苏莫闻言，顿时双眸一眯，去永恒神宫，这不是找死吗？
<br/>　　对方被他擒拿，不仅不怒，反而好心邀请？
<br/>　　这能让人信服？
<br/>　　“你的好意我心领了！”苏莫淡淡的摇了摇头，他可不信任对方。
<br/>　　公孙嘉闻言，沉默了少许，突然语出惊人，道：“我知道，你叫苏莫。”
<br/>　　“哦？”闻听此言，苏莫顿时眼神一凛，对方居然知道他的真实身份？
<br/>　　“前一段时间，轮回神朝的无上神王天尘宇在追查你的下落，曾去了永恒神宫，如果我所料不错，天尘宇不止是去了时空界域，其它很多界域都去了。”公孙嘉继续道，此事知晓的人不多，他作为永恒神宫宫主之孙，才恰巧得知。
<br/>　　之前，他并不知道，眼前的秦陨就是苏莫，在被擒拿之后，他才猜到了苏莫的身份。
<br/>　　“是吗？”苏莫眉头一皱，他相信对方之言，因为一般人根本不知道天尘宇和他的情况。
<br/>　　“你还知道什么？”苏莫沉声问道。
<br/>　　“我不清楚你和天尘宇之间有何恩怨，居然劳烦其亲自找你，不过，以你现在的情况，在神域随时都会陷入险境。”公孙嘉道。
<br/>　　苏莫闻言，心中明白，对方应该不知道他是分魂，只知晓天尘宇在找他。
<br/>　　“神域这么大，九大神朝，天尘宇休想找到我。”苏莫面无表情的说道。
<br/>　　“呵呵，这你就错了！”
<br/>　　公孙嘉微微摇了摇头，道：“天尘宇作为无上神王，威望极高，在任何一个界域，都有人卖其面子，就算是在另外几大神朝，都与很多强者交好，找到你只是时间的问题。”
<br/>　　苏莫闻言沉默了，天尘宇有这么大的能量？敢大肆追查他的下落吗？不怕将消息泄露给地王宫？
<br/>　　“你为何如此好心？”
<br/>　　少倾，苏莫凝望着公孙嘉，疑惑的问道，他将对方擒拿，对方不是该视他如仇敌吗？怎么还如此好心的提醒他？
<br/>　　“我的意思，你加入了永恒神宫，方可保你性命！”公孙嘉郑重的说道，如此天才，若是能加入永恒神宫，那就是一大幸事。
<br/>　　他永恒神宫，一直以来的规矩，就是吸纳各方天才，才能不断的壮大。
<br/>　　“多谢你的好意！”苏莫再一次的拒绝了对方，他以后不会再加入任何势力了。
<br/>　　因为，这些势力，都不足以给他绝对的庇护。
<br/>　　“既然如此，那你好自为之吧！”公孙嘉叹了口气，他已经软言向邀，苏莫无动于衷，他只能作罢。
<br/>　　随即，公孙嘉身形腾空而起，就欲离去。
<br/>　　不过，这个时候，他又突然停在了半空之中。
<br/>　　“苏莫，三年之后，我会去终极剑域的禁地，万道剑冢，到时候希望能在哪里领教你的实力！”
<br/>　　公孙嘉朗喝一声，随即便不再多留，转身破空而去。
<br/>　　他想要领教苏莫的实力，只不过，现在他实力未全部恢复，不宜出手。
<br/>　　“万道剑冢？”苏莫闻言疑惑，他并不知道万道剑冢是什么地方。
<br/>　　虽然他在神剑山待过不短的时间，但并未接触过什么人，也不了解终极剑域。
<br/>　　少倾之后，苏莫也离开了这片山林，一边飞行，一边思索接下来的目的地。
<br/>　　他试着用传讯符，给龙腾传讯，但均是石沉大海，没有半点动静。
<br/>　　他又给司空炎、姜枫然、清雅等几人挨个传讯，均是同样的结果，没有任何的反应。
