<br/>　　“打败苏莫，啸天狼族威武！”
<br/>　　“打败苏莫，啸天狼族威武！”
<br/>　　众啸天狼族的子弟，喝声完全不比道宗弟子差，气势同样强大。

<br/>　　只见幽狼的身体，以肉眼可见的速度膨胀了起来，急速变大，并且开始了兽.化。

<br/>　　只见庞大的银色妖狼，扬天一吼，狂暴凶厉的气势，铺天盖地的散发而出，充斥九天十地。
<br/>　　天穹之上，方圆万里的空间，在这一声巨吼之下，都被瞬间震碎了。
<br/>　　苏莫心中惊讶不已，这幽狼化为了本体，气势之强，真的是他见过的最强的武帝境武者，比之巨摩好像都还要稍强一分。
<br/>　　“银月啸天狼！是银月啸天狼！”
<br/>　　“传闻幽狼的血脉，已经快要返祖，成为真正的银月啸天狼，原来是真的！”
<br/>　　“恐怖，好强大的气势！”
<br/>　　整个武斗场，嘈杂一片，很多人都被眼前这只银色的巨狼震惊了。
<br/>　　强大！
<br/>　　极度的强大！
<br/>　　这种气势，在武帝境之内，不敢说无敌了，但基本上应该无人能击败了吧！
<br/>　　“银月啸天狼？”苏莫眸光一凝，这好像是一种神兽吧！幽狼是神兽？
<br/>　　这好像不太可能，应该只是拥有神兽的一丝血脉，但距离真正的神兽远着呢！
<br/>　　不然的话，实力绝对不止如此。
<br/>　　“苏莫，你必败！”银月啸天狼口吐人言，声音如同闷雷炸响。
<br/>　　“这种话我已经听腻了，这是你最后的机会了，出手吧！”苏莫淡淡的说道，不管对方变成什么狼，对他都没有什么威胁！
<br/>　　吼！
<br/>　　庞大的银月啸天狼，再次扬天一吼，身形猛然一纵，庞大的身体，如同飞驰的大山，急速的向苏莫扑了过来。
<br/>　　这一幕极为惊人，仿若是一只庞大的太古神兽，在捕杀一只小蚂蚁一般，苏莫的身形对于庞大的银月啸天狼而言，实在是小的可以忽略不计了。
<br/>　　轰隆隆！
<br/>　　虚空一震爆响，银月啸天狼庞大的身体，瞬间碾碎了空间，急速临近苏莫。
<br/>　　而后，它巨大的狼爪再次探出，如同泰山压顶一般，向苏莫的身上镇压而去。

<br/>　　身形淡薄的苏莫，在巨爪裹挟的劲凤之下，衣裳猎猎作响，一头长发，也被吹的笔直。
<br/>　　“回去吧！”苏莫轻喝一声，再次出剑了，再次一剑笔直的刺了出去。
<br/>　　与此同时，浑厚、磅礴，狂暴的气息，伴随着一股浩瀚的剑威，从苏莫的身上冲天而起，锋芒无尽。
<br/>　　剑光撕裂虚空，瞬间便与狼爪狠狠的相击在了一起。
<br/>　　轰！
<br/>　　一声惊天爆响，震耳欲聋，恐怖的劲力炸开，如同天河倒卷，浩浩荡荡。
<br/>　　而后，所有人都是瞬间石化了，不可置信的望着眼前的一幕。
<br/>　　只见银月啸天狼庞大的身体，仿佛飞驰的小山，居然直接倒飞了回去。

<br/>　　被一剑击出了战台，彻底的败了！
