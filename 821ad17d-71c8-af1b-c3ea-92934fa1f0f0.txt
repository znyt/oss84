　　北海。
<br/>　　一处浩瀚的海域之中，坐落着一座庞大的岛屿。
<br/>　　这座岛屿方圆数十万里，一眼望去岛上青山连绵，郁郁葱葱，如同一条条青色的长龙，盘旋蛰伏。
<br/>　　此岛名为青俞岛，又称为青俞国，因为岛上有一个国家，叫青俞帝国。
<br/>　　青俞帝国的实力，在这一片海域非常强大，是依附于北海冰宫的势力。
<br/>　　青俞帝国，皇城。
<br/>　　宽阔的街道上，人来人往，车水马龙。
<br/>　　苏莫一袭月白色青衫，漫步在街道上，他的面容稍稍改变了，免得又引起轰动。
<br/>　　此地虽然距离苍穹大陆极远，但肯定有人认识他。
<br/>　　所以，他稍稍改变了容貌，否则若是被人认出来，到时候估计会打草惊蛇。
<br/>　　在来到皇城之前，苏莫在岛屿外，以及岛上其他地方转了一圈，用幽冥魔瞳武魂扫视了每一寸地方。
<br/>　　甚至，他连青俞岛四周的海底，都仔细的检查了一遍，没有任何发现。
<br/>　　他也没有抱太大的希望，或许黄泉魔宗之人，真的曾在此地藏身，但可能早已离开了。
<br/>　　最后，苏莫才来到了皇城，若是再没有发现，他便会返回苍穹神宫。
<br/>　　“嗯？”
<br/>　　走在皇城的街道上，苏莫双眉一挑，他果然在空气中发现了若有若无的尸气。
<br/>　　这股尸气极为淡薄，一般的武皇强者都很难发现，但苏莫精神力强大，感知力极为敏锐，还是发现了空气中弥漫的尸气。
<br/>　　“会不会是尸罗殿之人？”苏莫心中疑惑，这尸气也有可能是自然形成的尸魁所留。
<br/>　　不过，既然来了他自然会查清楚。
<br/>　　随即，苏莫灵识扫荡而出，覆盖了整座皇城，一边漫步而行，一边仔细感受尸气的源头。
<br/>　　“皇宫！”
<br/>　　很快，苏莫便发现了尸气的源头所在，居然是青俞帝国的皇宫。
<br/>　　“有趣！”苏莫来了兴趣，这散发尸气之人无论是尸罗殿的人，还是自然形成的尸魁，但身处皇宫之中，那就耐人寻味了。
<br/>　　而后，苏莫大步而行，一步数十丈，向青俞帝国的皇宫而去。
<br/>　　不多时，他便来到了皇宫之前。
<br/>　　青俞帝国的皇宫，占地万顷，宫殿连绵，各种建筑恢弘大气，尽显皇家气派。
<br/>　　皇宫前是一个辽阔的广场，占地方圆万丈不止，而此刻，广场上人山人海，喧嚣震天，好不热闹。
<br/>　　之前苏莫已经用灵识查探过了，此地好像在举行某种比武，所以才会聚集了这么多人。
<br/>　　只见广场中央，搭建了一座高达五丈，长宽各有近千丈的方形战台。
<br/>　　战台上，一对二十岁左右的年轻男女正在激斗，男子面容粗狂，身材高大魁梧，女子身姿纤柔，容貌娇美，身穿青色绒裙，一头黑发随风飘舞。
<br/>　　这两人均是武王境三重的修为，实力着实不弱，每一击都威力绝伦。
<br/>　　不过，明显那名女子的实力更强，其手中之剑青光耀眼，招招致命，将魁梧年轻人杀的节节败退。
<br/>　　战台之上以及四周均布置了阵法，能隔绝气劲波动，所以两人的战斗威势虽然不俗，却并不会波及战台之外。
<br/>　　而在战台下方数百丈之处，搭建了一座高高的看台，有数十名强者端坐在看台之上，正在观战。
<br/>　　这数十名强者之中，有一面容肃穆的中年人，身穿龙袍，不怒自威，显然便是青俞帝国的皇帝。
<br/>　　四周还有数百名身披甲谓的士兵，将战台和看台守护在内，显然是在维护秩序。
<br/>　　“青雅公主真是厉害，已经连败了九名挑战者！”
<br/>　　“不错，我看我们青俞帝国的年轻一代之中，很难有人能击败青雅公主！”
<br/>　　“哈哈！陛下为公主举行比武招亲，打败公主才能成为驸马，估计短时间内很难招到驸马了！”
<br/>　　“其它岛上也有天才闻讯而来，不知道谁最终能抱得美人归！”
<br/>　　嘈杂的议论声此起彼伏，广场上聚集了数十万人，声浪震天，直冲云霄。
<br/>　　人群后方，苏莫淡然而立，他没有去关注战台上的比武，而是在打量着那看台上的数十名强者。
<br/>　　他的目光，首先落在了那青俞帝国皇帝的身上，武皇境二重武者。
<br/>　　随即，苏莫目光转动，在另外数十名强者身上扫过，最后落在了一名老者的身上。
<br/>　　那名老者年约六十岁许，身穿藏青色长袍，身形消瘦，鹰钩鼻，面无表情的端坐在青俞皇帝的身边。
<br/>　　“就是此人！”苏莫眸中精光一闪，在他的感知之中，那淡薄无比的尸气，便是从此人身上逸散而出。
<br/>　　这就说明，此人要么是修炼了尸功，要么本身就是一具尸魁。
<br/>　　随即，苏莫立刻穿过人群，准备绕过战台，直接去将看台上的那名老者擒拿。
<br/>　　对方是否是黄泉魔宗之人，他直接逼问便可。
<br/>　　不过，就在苏莫刚刚走到战台旁边之时，异变陡生。
<br/>　　“你败了！”
<br/>　　只听得一声娇喝，便见战台上那名青衣女子一剑飞快的斩出，一道青色的剑气，瞬间斩在了那名魁梧年轻人的胸膛上。
<br/>　　啊！
<br/>　　一声惨叫，鲜血飚射，那名魁梧年轻人立刻身形倒飞而出，直接砸在了苏莫的身前。
<br/>　　嘭！
<br/>　　魁梧年轻人身躯砸在地面，将地面都砸的龟裂开来。
<br/>　　好在那名女子有所留手，魁梧年轻人才得以保全性命。
<br/>　　“哼！草包一个，居然还想当本公主的驸马！”
<br/>　　战台上，那名青衣女子傲然而立，面上带着傲气与刁蛮，不屑的冷哼一声。
<br/>　　随即，女子美眸流转，望向了接近战台的苏莫，傲气的说道：“你也想挑战？没有实力的话，就不要浪费本公主的时间了！”
<br/>　　苏莫没有理会青衣女子，抬脚跨过魁梧年轻人的身躯，就欲前往看台。
<br/>　　青衣女子见此一怔，完全没有想到，苏莫居然不理会她！
<br/>　　青衣女子心中怒气顿生，脚下一跨，直接飞下了战台，拦在了苏莫的身前。
<br/>　　“本公主问你话呢！”青衣女子美眸含煞，冷冷的凝视着苏莫，厉声娇喝。
<br/>　　她作为青俞帝国的公主，从来还没有人敢无视她，而今日面前的这个家伙，居然无视她的问话？真是岂有此理！
<br/>　　“没兴趣！”苏莫停下了身形，淡淡的瞥了青衣女子一眼。
