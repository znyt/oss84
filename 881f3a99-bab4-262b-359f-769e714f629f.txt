　　　　“叮...宿主彻底融合人类觉醒者能力，获得冰霜之力”
<br/>　　涂小安猛然一个似有所感，三十米巍峨的蛇躯下，一股深蓝色的气焰迸发而出，沸腾了起来，紧接着周边的温度骤然下降，透骨奇寒。
<br/>　　这种冷，不同蛇类气息下的阴冷，而是冷到骨髓，冻到灵魂，能冻结凝固一切。
<br/>　　哈哈...冰霜之力，又得到了一种元素属性能力。
<br/>　　涂小安难免激动起来，他现在可是三种元素能力加身，强的无法想象。
<br/>　　元素属性能力，一个人能拥有一种，修炼到极致，就可以站到这个世界的巅峰。
<br/>　　随着涂小安的心随意动，身上迸发的深蓝色气焰朝着地面蔓延，所到之处，寸寸冻结，寸草不生。
<br/>　　方圆几十公里，冷的跟南极一样，不知道冻死了多少生物。
<br/>　　这冰霜之力，在自己的手上，貌似比冷霜手中更强，更具杀伤力。
<br/>　　这是实力，他的实力本来就碾压冷霜。
<br/>　　“由于宿主突破自身束缚，杀戮果断，吞噬系统全面启动...”
<br/>　　忽然，涂小安的脑海中，想起了系统的声音。
<br/>　　“全面启动？”
<br/>　　还未待涂小安想明白，脑海中就出现了光芒闪烁的虚拟界面。
<br/>　　宿主数据显化...
<br/>　　宿主：涂小安。
<br/>　　身份：伪蛇王（登基坐殿）
<br/>　　等级：二十八级。
<br/>　　进化能源值：000/1000.
<br/>　　信仰点数：29800000
<br/>　　天赋属性：（意识洞察）（统治能力）
<br/>　　技能：（雷电之力）（火焰之力）（冰霜之力）
<br/>　　涂小安看了看自己的新数据，明显发现了不同，进化能量值，变成了能源值。
<br/>　　这是什么意思，说明涂小安现在碰到的都是高强的精怪或许是修武者，不管是生物体内的强大生命力，还是人类，都用能源来定义。
<br/>　　这是升级版。
<br/>　　所谓的全面启动，涂小安还不是很懂，不过之后体验就知道了。
<br/>　　他也没有过多的想这个，三十米的巍峨紫金巨蟒消失不见，一位翩翩少年郎出现。
<br/>　　信仰点数多，不心疼。
<br/>　　对了，三阶虺蛟。
<br/>　　涂小安骤然朝着原来的地方找回去，过去之后，地面一趟血液，可却不见虺蛟的踪影。
<br/>　　看来他是趁着自己对付冷霜，恢复了一点气力，逃跑了。
<br/>　　“逃...”
<br/>　　不存在的。
<br/>　　涂小安真正的收获可在这条三阶虺蛟的身上，它若跑了，自己不是白忙活。
<br/>　　他的意识力铺天盖地的席卷出去，不过半响，涂小安淡淡浅笑了起来，心中有数。
<br/>　　平安村经过一场蛇与蛟的战役，已经一片狼藉，在加上冷霜的一手冰封千里，将这个村所有的房屋彻底摧毁。
<br/>　　可以说这里已经没有平安村了，简直就是拆迁的废墟。
<br/>　　涂小安也不管这些，朝着一个方向走去，走着走着，一片废墟中，一口水井暴露了起来。
<br/>　　这口水井应该是平安村现在唯一保存的文化古迹了，并没有在战斗中被破坏。
<br/>　　里面阴森森，阴冷的风不时的冒出，井水丰盈。
<br/>　　涂小安抬头看了一眼慢慢钻出来的月亮，一屁股到了井口处，嘴角微勾，道：“天师，出来吧，别躲了”
<br/>　　井内静悄悄，只有一股股阴冷的风冒出。
<br/>　　“我知道你藏匿在里面，你也应该知道，要是我想，这口水井可以顷刻间不复存在”
<br/>　　“你到底想怎样”井内深处，传来张天师气急败坏的声音。
<br/>　　涂小安闻言，笑了笑：“就想请天师出来一见，你说你堂堂三阶虺蛟，还怕见我这条二阶蛇精吗”
<br/>　　“莫非，被我打怕了”
<br/>　　“哼，休得胡言，你想战，那么改天本天师奉陪”
<br/>　　涂小安一头黑线，改天？
<br/>　　等你恢复了伤势在打，涂小安闲的啊。
<br/>　　“天师，我的耐心是有限的，别浪费时间了，出来吧”
<br/>　　之前的战斗，由于冷霜的偷袭，让涂小安跟虺蛟都受了很重的伤。
<br/>　　但是涂小安有蛇莓，立刻恢复了，而且他的肉身强度本来就很变态。
<br/>　　可虺蛟就未必了，能爬到水井这里，算不错了。
<br/>　　它不出来也是正常，出来，被涂小安吃吗。
<br/>　　水井里面干脆连声音都没了。
<br/>　　涂小安眼眸精光一闪，伸出一只白皙的手，一团炙热的火焰冒出，直接丢到了水井内。
<br/>　　倏然地，白茫茫的雾气升起，里面的井水瞬间被蒸发，成了一口干枯的井。
<br/>　　井内很深，涂小安忽然想起了男娃被投井，村花跳井，都没有被人找到尸体，而且这口井是曾经虺蛟修炼的地方。
<br/>　　那么里面应该别有洞天，没有那么简单。
<br/>　　“还不出来吗，想我进去找你？”涂小安说着心中却在思忖。
<br/>　　想着如何处置这条三阶虺蛟，到底是收服了好，还是直接吞噬了好处更大。
<br/>　　有一条三阶的虺蛟为宠物，逼格简直不要太大，要是它以后进化成龙，那么一条蛇拥有龙为宠物，应该足够说明一切了。
<br/>　　况且三阶生灵不易，直接吞吃了未免有点暴殄天物。
<br/>　　几番思索下，涂小安决定到时候在说，只是隐藏在下面的虺蛟依然不见出来。
<br/>　　“天师，你真想我毁了这口水井不成，死者为大，我只是不想打扰了小凤跟你孩子的安息”
<br/>　　此话一出，干枯的井内一股骇人的恐怖气息扑面而来。
<br/>　　紧接着，一条不过三米多的虺蛟翻滚而出，直接悬浮到了涂小安的面前。
<br/>　　“蛇精，你不要欺人太甚”张天师的声音从虺蛟的口中冒出。
<br/>　　涂小安笑道：“你现在连化为人形都做不到吧”
<br/>　　虺蛟沉默，遍体鳞伤的伤口，已经不用多说，它现在并没有多少的战斗力。
<br/>　　涂小安深深看了一眼悬浮的虺蛟。
<br/>　　“叮...三阶虺蛟，等级三十级，吞噬可以获得3000的进化能源”

<br/>　　照成之前的进化能量值，这样的三阶生灵，起码能获得数千万的能量值了。
<br/>　　将这条虺蛟吞了，貌似涂小安顶多在升一级多。
<br/>　　两级恐怕都办不得。
<br/>　　这数据未免有点不对吧。

<br/>　　还不如一颗蛇王左眼。
