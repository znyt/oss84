的功效。<br/>　　<br/>　　    这个快要死亡的人，服用这药，竟然能够延长其生存的时间，而且病情还不至于恶化，这样的药物，他都想来一打！<br/>　　<br/>　　    “好！这个药丸你拿好！另外，陈默有没有告诉你，关于这个药物的其他信息？”南宫承宣问道。<br/>　　<br/>　　    “没有！”南宫雪直接说道。<br/>　　<br/>　　    对于药物中的一些药材，陈默是说过的，但是现在这么人，谁能保证什么呢？所以还是直接说没有好了！<br/>　　<br/>　　    再说了，她看到父亲震惊的表情，就能明白这药有多么好，所以这才隐瞒了下来！<br/>　　<br/>　　    “好吧！既然你有如此的把握，而且也有这种药物的话，那么就按照你说的办吧！”南宫承宣毕竟是一家之长，对于有些事情，还是有决断的。<br/>　　<br/>　　    他也想救自己的老婆，但是没有希望以后，自然也就绝望了！但是有了女儿这些说词后，他还是决定赌上一赌！<br/>　　<br/>　　    “谢谢父亲！也谢谢大家的关心！”南宫雪对自家人说道。<br/>　　<br/>　　    现在出现在大厅中，也是十几号人，都是南宫雪的亲人。他们阻止南宫雪，也是有着各自的打算。但是无论是好心还是私心，南宫雪都是要感谢的。<br/>　　<br/>　　    南宫雪感谢完毕后就要起身，却被父亲叫住了，带着她来到了书房中。<br/>　　<br/>　　    “阿雪，这个药丸是陈默炼制的还是其他人炼制的，你知道不知道？”<br/>　　<br/>　　    “这个倒是不清楚，陈默只是将药丸给了我以后，说了一下用法和功用以外，就没有说过其他的。”南宫雪说道。<br/>　　<br/>　　    “那好！你这次送你母亲去医治的时候，记住打听一下这个药丸！一个是谁炼制的！另外一个就是还有没有了，要是有就全部买下来，钱不是问题。如果有其他药丸，也要全部买下！”南宫承宣说道。<br/>　　<br/>　　    对于药丸，他作为家族族长来说，自然是有多少就要买多少，不会嫌多的。而且这种救命药丸，也是多多益善，并且陈默哪里有这种药丸，那么说不定也有其他药丸！<br/>　　<br/>　　    &lt;/content&gt;<br/>　　<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
