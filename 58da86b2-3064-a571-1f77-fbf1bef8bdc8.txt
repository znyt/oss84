　　“前辈，你在此等待，我去杀几个人！”苏莫向秦不死说了一句，便闪身离开。
<br/>　　索性无事，那些巫族的试炼者既然来了，苏莫自然不想放过。
<br/>　　零散的几名巫族天才，很快便被击杀，苏莫再次获得了几枚神丹，以及一些资源宝物。
<br/>　　目前，剩下的巫族试炼者，还有四百多人。
<br/>　　不过，这些人也是被苏莫杀怕了，西海一役之后，全都分散开了，并没有再聚集在一起。
<br/>　　杀死了几人之后，苏莫便和秦不死汇合，继续等待。
<br/>　　随后的三天时间里，陆陆续续又有二十余名巫族试炼者来到了平原上，尽数被苏莫斩杀。
<br/>　　而到得这个时候，苏莫将幽冥魔瞳武魂催动到极致，能发现平原的四周，极远之处有很多巫族天才隐藏。
<br/>　　看来很多人都来到了此地，但发现苏莫守在平原上，根本不敢靠近。
<br/>　　数万里外，天穹之上，一团浓密的云层中，百余名巫族天才藏身与此，巫天御亦在其中。
<br/>　　“天御哥，这次麻烦了，那个土著守在平原上，我们如何离开？”一名魁梧壮硕的巫族青年，脸色难看的说道。
<br/>　　“是啊，我们不会都要死在这里吧？”
<br/>　　“这个土著太强大了，我们完全不是对手啊！”
<br/>　　“空间通道即将开启，此人必定会守住空间通道，我们必须想个办法？”
<br/>　　“……”
<br/>　　其他巫族天才纷纷开口，个个面上充满了担忧与惊恐。
<br/>　　所有人都后悔了，后悔来参加这次试炼。
<br/>　　现在已经有大半族人惨死，他们真的是恐惧不已。
<br/>　　一众巫族试炼者，虽然不是温室里的花朵，但在外界之时，作为巫族天冥星支脉的后辈天才，在天冥星是无人敢惹的存在，何曾遇到过如此挫折！
<br/>　　从来都是顺风顺水的他们
<br/>　　一时间，很多人都是心中慌乱，没有了注意。
<br/>　　“都慌什么？”
<br/>　　巫天御黑着脸，轻喝一声，冷声道：“若是你们怕死，那就永远的留在苍穹世界吧！”
<br/>　　众人闻言心中一震，随即个个脸色变幻，留在苍穹世界？这怎么可能？
<br/>　　“我宁愿死，也不会留在这个鸟不拉屎的地方！”
<br/>　　“我未婚妻还在等我，我必须要回去！”
<br/>　　“留在这里很难突破到武帝之境，只会耗尽寿元而死！”
<br/>　　“我们一定要出去！”
<br/>　　众巫族天才收起了心中的恐惧，个个满脸坚毅之色，他们必须要出去！
<br/>　　巫天御见此微微点头，道：“那好，你们去招集所有人，到时候所有人一起联手！”
<br/>　　“是，天御哥！”
<br/>　　“是，天御族兄！”
<br/>　　“我们这就去办！”
<br/>　　众巫族天才纷纷应声，随即四散而开，前去召集人手。
<br/>　　转瞬间，云团中便只剩下了巫天御一人。
<br/>　　望着远处的平原，巫天御的眼眸微眯，双拳不自觉的紧握了起来。
<br/>　　他一定要出去，即便舍弃所有人，他也要出去。
<br/>　　可惜上次的地煞晶，没能杀死苏莫，现在他已经没有了这种强大的攻击手段。
<br/>　　……
<br/>　　土丘之上，苏莫和秦不死并肩而立。
<br/>　　“苏莫小友，两月之期已到，通道将会在今日开启！”秦不死满脸凝重的说道。
<br/>　　“嗯！”
<br/>　　苏莫重重的点了点头，道：“准备战斗吧！杀光所有人，我们再离开！”
<br/>　　对于这些试炼者，苏莫是绝对的狠辣无情，因为这些试炼者就是将苍穹世界的人当场猪狗，随意屠杀。
<br/>　　所以，他心中没有丝毫的怜悯。
<br/>　　秦不死闻言，心中暗叹一声，按照他的想法，通道开启便立刻离开，但苏莫坚持要杀光所有人，让他很是无奈。
<br/>　　不过，他也没有反对，反正苏莫实力强大，也无需他帮什么忙！
<br/>　　两人静静的等待，时间仿佛变得很慢，每一个呼吸都变的很是漫长。
<br/>　　苏莫和秦不死，均是心中沉重到了极点，他们能不能彻底逃离苍穹世界，就看今日了！
<br/>　　一个时辰又一个时辰，在苏莫两人的等待中，四个时辰的时间，悄然而逝。
<br/>　　轰隆隆！
<br/>　　就在此时，虚空中突然传来一阵沉闷的声响，如同闷雷炸响。
<br/>　　霎时之间，万里之外的虚空，剧烈的颤动了起来，波及八方。
<br/>　　“来了！我们走！”苏莫眼眸中顿时精光爆射，璀璨无比。
<br/>　　唰！
<br/>　　毫不犹豫，苏莫身形一闪，如同一支利箭划过长空，向那空间波动最剧烈之处冲去。
<br/>　　嗖！嗖！
<br/>　　秦不死紧随其后，片刻之间，两人便来到了空间波动的源头。
<br/>　　随后，两人伫立在虚空中，静静的望着眼前的一幕。
<br/>　　他们的前方十余里处，虚空不断的震动、扭曲，闷响连连。
<br/>　　强烈的空间波动席卷天地，横扫八方，大半个苍穹世界，都能感受的到。
<br/>　　虚空中炸雷般的声响，低沉无比，以苏莫两人的修为，耳朵都被震的嗡嗡作响。

<br/>　　因为他们知道，试炼者将要离开了，他们安全了。

<br/>　　所以，很多苍穹世界的强者，得以保命。
<br/>　　被试炼者斩杀的人，十不足一。
<br/>　　轰隆隆！！
<br/>　　闷响不断，巨响连连，直到半个时辰之后，虚空开始向内塌陷，渐渐形成了一个方圆数十丈的虚幻通道。

<br/>　　巨大的通道幽深无比，其内发出淡淡的白光，仿佛通向遥远的未来，又好像通向未知的神秘之地。

<br/>　　通道乍一形成，苏莫和秦不死便身形一闪，飞进了通道中。
<br/>　　而后，两人立刻转身停了下来，伫立在通道口，将通道牢牢的守护在了身后。

<br/>　　“这些试炼者人呢？难道都不想回去了？”秦不死皱眉问道。
<br/>　　“他们肯定会回去，只不过见我们在此，不敢现身而已！”
<br/>　　苏莫摇了摇头，继续道：“我们不能掉以轻心，小心他们有强大的手段！”
<br/>　　对于这些试炼者，苏莫想的是能杀多少是多少，杀不尽也无妨。
<br/>　　毕竟，说不定就有人有恐怖的手段，比如上次的那种地煞之气。
<br/>　　他最主要的目标，还是巫天御，因为宏青璇还在对方的手上。
