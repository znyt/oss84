　　习大人每次回来的时候，都会很亲切的和他们这些佣人打招呼，从来不像今天这样，爱搭不理的。
<br/>　　而且，习大人怀里抱着的那个男孩，就是金木研吗，他听月山习描述过很多次，这还是首次见到真人呢。
<br/>　　看起来，也很一般啊，不知道为什么习大人对他那般上心。
<br/>　　紫发男孩低着头颅，偷偷打量着，心中略微升起一些情绪，但很快就被他掩埋了下去。
<br/>　　作为习大人身边最忠心的仆人，习大人就是他的一切，他不能在心中对习大人有任何不满和异议。
<br/>　　一行人匆匆忙忙的来到大厅，月山习将金木放在沙发上，吩咐让人找私人医生过来。
<br/>　　像他们这种家族，自然会有隶属于自己家族的医生，毕竟身为喰种，他们如果受了伤也没办法去医院治疗。
<br/>　　虽说如此，以月山家的实力，即使是私人医生也比一些所谓专家什么的高明多了。
<br/>　　刘哔站在旁边，有些无所事事，对于金木的死活，他一点也没有担心。
<br/>　　身为未来的喰种之王，这点伤势应该不会有什么影响，光是靠他自身超越一般喰种的再生能力，也足以恢复过来。
<br/>　　只需要花费为数不多的一点时间而已。
<br/>　　相对而说，他现在更关心的，是自己刻意留下的那张名片，到底有没有发挥作用。
<br/>　　等候了一段时间，医生提着医疗箱来到了众人面前，月山习紧张的交代了一下，便让医生抓紧时间给金木研治疗。
<br/>　　最终，果然如同刘哔所料，医生的检查结果并没有太大问题，金木身上的伤势，只需要静养就能恢复。
<br/>　　为了增加伤势的恢复速度，最终医生还是开了一幅药方，嘱咐了一下注意事项，提箱走了出去。
<br/>　　确认金木不会有什么问题之后，月山明显松了口气，吩咐佣人将金木送到房间，方才转过来面对着刘哔。
<br/>　　“这位先生，难道不介绍一下你的身份吗？”没有了往日常挂在脸上的和睦笑容，月山表情有些阴冷。
<br/>　　事关月山家的安危，他不敢有丝毫大意，他也有一点好奇，不知道这家伙是如何得到他联系方式的。
<br/>　　为了避免家族的消息泄露，他很少将自己的身份透露出去，很多喰种知道他是美食家，但却不知道他也叫月山习。
<br/>　　能知道他真实姓名的，基本上都是和他熟识的喰种，譬如古董，或者几个关系紧密的合作伙伴。
<br/>　　但月山习很确定，他今天是第一次认识这个男人，在此之前，连见都没见过。
<br/>　　“修罗。”脸上挂起笑容，刘哔静静的看着面前的月山习，他的身份，没有什么隐藏的必要。
<br/>　　听到这个名字，月山习眉头微微一皱，对于前段时间将20区引的大乱的修罗，他自然有所了解。
<br/>　　但多方打听之下，也没有谁真正知道修罗的真正实力，目前为止，也没有有关他战斗的消息流露出来。
<br/>　　“原来是修罗先生，不知道为何金木会和你在一起，修罗先生能否告知？”月山习语气变的客气了许多，对于未知实力的喰种，他也不想过多得罪。
<br/>　　稍稍思索了一下，刘哔将库克利亚中发生的事情简单讲述了一遍，当然，其中很多事情他都隐瞒了下来。
<br/>　　告诉月山习的版本，也只是金木率领青铜树众人入侵，然后被CCG埋伏，后被他就出来了而已。
<br/>　　听刘哔讲完之后，月山习眼中露出一抹思索，他不是傻子，自然察觉到对方告诉他的事情有所隐瞒，但刘哔有意不说，他也没什么办法。
<br/>　　思索片刻之后，他还是决定先将刘哔留下来，试探出他的具体实力之后再做打算。
<br/>　　但刘哔的下一句话，却让他心中猛然警觉了起来。
<br/>　　“听说月山家和小丑组织经常往来，恰好，前段时间我听说小丑组织要进行一场拍卖会，不知道习大人能否帮我搞到一张邀请函？”
<br/>　　脸上带着人畜无害的笑容，刘哔似乎没意识到他的话语有多凝重，月山习以及身边的那个紫发少年，都猛地将目光对准了他。
<br/>　　耸了耸肩，刘哔对这两人的反应之大有些意外，小丑组织，这在喰种之间并不是什么秘密吧。
<br/>　　“你是小丑的人？”月山皱起了眉，语气中带着戒备，想了一下，又改口道：“不对，你不是，但你怎么知道拍卖会的消息？”
<br/>　　哪怕是和小丑组织极为亲密的月山家，也只是知道小丑组织在策划一场拍卖会，可具体的时间和地点，根本就没有确定下来。
<br/>　　更不要说，那还没开始制作的邀请函了。
<br/>　　这个‘修罗’，到底是什么身份，居然知道小丑内部的事情，而这个消息，也肯定不会是正常手段打听到的。
<br/>　　如果他真的和小丑组织为同一战线，也不会连一张邀请函都搞不到了。
<br/>　　大脑疯狂旋转着，月山习借助着几言片语，寻找着可能存在的蛛丝马迹，但最终，他还是失望的叹了口气。
<br/>　　实在想不通，这个家伙是从哪里冒出来的，在此之前，他甚至都没听说过。
<br/>　　看着面前满脸笑容的刘哔，月山习突然也笑了起来，随后，似是放下了戒备一般，月山习伸出了自己的右手。
<br/>　　“月山习，请多赐教，如果修罗先生不急的话，可以在这里居住几天，毕竟CCG此时还在严查，此时外出活动未免太过于危险。”
<br/>　　“好的，那就劳烦习大人了。”沉吟了一声，刘哔没有拒绝，留在此地，也正合他意。
<br/>　　两人的手掌握在一起，月山习的笑容更加灿烂了，随后，月山为他在三楼安排了一个房间，并嘱咐他晚上七点准时开饭后，从楼梯走了下去。
<br/>　　关上自己的房门，刘哔脸上的笑容逐渐消失，随后，似乎想到了什么，他仔细检查了一下房间，并没发现摄像头之类的监听仪器，方才放下心来。
<br/>　　月山习之所以留下他，恐怕原因并没有那么简单，如果他的猜测没错，小丑组织很快就能得到他的消息。
