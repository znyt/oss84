　　从这三名武者身上得到的宝藏，比起楚风眠自己寻找，倒是要快得多。
<br/>　　吞噬了这三名武者空戒身上之中的宝物，楚风眠感觉到他的力量，明显是更近了一步。
<br/>　　与其是辛苦寻找宝藏，倒不如去猎杀其他武者。
<br/>　　在这赤红大陆之中的武者，应该都有着不小的收获。
<br/>　　现在既然是其他三大势力，已经是都彻底撕破脸面了，那楚风眠倒不如先出手，先杀一下三大势力的锐气。
<br/>　　而且还可以得到他们从这赤红大陆之中得到的宝藏，一举两得。
<br/>　　“就这么做！”
<br/>　　楚风眠思考了一下，便是下定了决心。
<br/>　　以楚风眠现在的实力，就算是碰到九劫古帝，也足以从容而退，这三大势力之中的圣子，能够达到九劫古帝的，都是无比罕见。
<br/>　　大部分都是一群七劫古帝，八劫古帝左右实力的武者，楚风眠想要杀了他们，不需要浪费多少的力气。
<br/>　　一名古帝，培养起来所花费的资源都是一个天文数字，纵然是大罗天星域是四大势力，每一方势力之中的古帝也并不多。
<br/>　　楚风眠杀一个，也就杀一个，可以削减三大势力的实力。
<br/>　　而且现在楚风眠要追杀人，还有着一个无比方便之物。
<br/>　　楚风眠手心一动，一枚铜镜便是出现在楚风眠的手心之中。
<br/>　　窥神镜，楚风眠从严家众人的手中得到的天道宗至宝，这窥神镜，可以查出方圆百里武者的气息。
<br/>　　武者的气息，跟修炼的功法息息相关，楚风眠手心一动，突然间模拟了一下五行天的功法，凝聚一股气息，打入到了这窥神镜之中。
<br/>　　楚风眠已经是跟其他三大势力的强者交手过了不少次了，对他们的功法，熟悉无比。
<br/>　　虽然不足以精通，但是模拟几分气息，倒是容易。
<br/>　　这一股气息，打入到了窥神镜之中，窥神镜中当即显现出了几道影子。
<br/>　　就在这方圆百里之内，居然就有着三名五行天的圣子，三人皆是古帝实力，如今正结伴同行，寻找宝藏。
<br/>　　“果然有效果！”
<br/>　　楚风眠看到这效果，脸色一喜，随即又将沧海都，齐林山功法的气息，打入到这窥神镜之中。
<br/>　　在这窥神镜之上，又是多显现出了几道影子。
<br/>　　虽然身为五行天圣子，也有可能修炼的不是五行天的功法，不过楚风眠目的并不是要将三大势力的圣子一网打尽。
<br/>　　毕竟赤红大陆之大，楚风眠一个人想要将三大势力的圣子全部一网打尽，根本是在天方夜谭。
<br/>　　现在楚风眠只需要是去大量的猎杀便可。
<br/>　　看着窥神镜上面显现出的几道身影，楚风眠身形一闪，向着最近的几道身影的方向飞了过去。
<br/>　　就在不远处，赤红大陆之上，三名武者正结伴同行，他们身上的衣着，皆是显现出他们来自于同一方势力，五行天。
<br/>　　“这赤红大陆，正是满地都是宝藏，这一次我收获的许多圣药，许多都是中古，甚至还有不少上古级别的古圣药了，拿出去足以卖一大笔钱了。”
<br/>　　一名五行天圣子，手中正拿着一株古圣药，脸上露出了几分喜色说道。
<br/>　　“这一次我们是赚大了。”
<br/>　　“这算什么，据说这赤红大陆，还有几处大型的宝藏，即将开启，那些大型宝藏，据说可是天人留下的洞府，那里面的机缘，才是真正的宝藏，我们得到的这些，根本不算什么。”
<br/>　　另一名五行天圣子，却是撇了撇嘴的开口道。
<br/>　　“只可惜我们没有资格去争夺，只有我们五行天的太上长老，才有资格去争夺那些大型宝藏。”
<br/>　　“大型宝藏想要争夺，谈何容易，据说这几日，各大势力之中的九劫古帝，都陨落了好几位了，以我们的实力前去，只怕是凶多吉少，还不如安安稳稳得到些圣药。”
<br/>　　又一名五行天圣子开口道。
<br/>　　“对了，不过除了大型宝藏，你们可知道，小天主的悬赏？”
<br/>　　“你是说？那个叫楚风眠的小家伙？”
<br/>　　听到这小天主的悬赏，另外两名五行天圣子都是冷笑一声道。
<br/>　　“也不知道那个小家伙，怎么引起小天主注意的，居然是给他下了天价悬赏，谁能杀了那小子，才是走大运了。”
<br/>　　“据说那小子只是一个古帝，不如我们去找找他，能找到他，得到小天主的悬赏，我们三人都可以实力再进一步，有着冲击九劫古帝的机会。”
<br/>　　听到冲击九劫古帝，这另外两名五行天圣子的眼神都是炙热兴奋无比。
<br/>　　他们三人都是八劫古帝，八劫古帝跟九劫古帝虽然只差一重境界，但是地位却是截然不同。
<br/>　　在五行天内，圣子最高也就是八劫古帝，一旦是踏入到了九劫古帝，可就不是圣子了，而是太上长老了。
<br/>　　不过突破九劫古帝，谈何容易，十位八劫古帝之中，都未必有一人可以踏入九劫古帝。
<br/>　　除了绝世的天资之外，还需要大量的资源。
<br/>　　他们三人虽然是从这赤红大陆之中得到了大量的宝藏，足以换取惊人的财富，可是距离冲击九劫古帝，还想差得远。
<br/>　　“想要找那小子，谈何容易，一个小小的准帝，没准早就死了。”
<br/>　　一名五行天圣子，摇了摇头道。
<br/>　　现在在这赤红大陆之中，古帝都遍地都是，真帝更是满地走，一个准帝，来到这让赤红大陆，早就是不知道怎么死的了。
<br/>　　“你们想要来找我？那好，倒是给你们一个机会。”
<br/>　　突然之间，一道声音从天而降，随即一道身影，来到了这三名五行天圣子的面前。
<br/>　　“什么人？”
<br/>　　看到这身影无声无息的出现，三人的脸色都是露出了几分谨慎，当看清这身影样貌的时候，三人突然眼神之中皆是无比兴奋，同时开口道。
<br/>　　“楚风眠？”
<br/>　　一名五行天圣子盯着，楚风眠兴奋的说道。
<br/>　　“楚风眠，你居然还敢出现？看来这一次正是上天让我们走大运了，发财的机会到了。”
<br/>　　“杀了此人，免得此人跑了！”
<br/>　　另一名五行天圣子，则是没有丝毫的犹豫，直接出手，一把对着楚风眠抓了过去。
