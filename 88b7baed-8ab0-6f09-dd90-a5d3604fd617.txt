　　以剑堂长老破玄八重的修为，这道澎湃剑气，足以要了萧逸的命。
<br/>　　更别说如今萧逸正在参悟状态，毫无防备。
<br/>　　嗖的一声，易老瞬间出现。
<br/>　　愤愤地一挥手，剑气瞬间消散。
<br/>　　剑堂长老的澎湃剑气，在易老面前，竟是如此不堪一击。
<br/>　　甚至于，没有发出半点声响，便消散须弥。
<br/>　　不仅没有伤到萧逸。
<br/>　　更没有影响萧逸的参悟状态。
<br/>　　数秒后，大长老和其余长老才相继出现。
<br/>　　……
<br/>　　放置极界碑的楼阁，本就在剑堂之内。
<br/>　　故剑堂长老来得最快。
<br/>　　十大长老，感觉到极界碑的异动，都是怕极界碑有闪失，而立刻赶来。
<br/>　　只有易老，知道极界碑的异动，必然与萧逸有关。
<br/>　　担心之下，全力赶来。
<br/>　　外门宝库，位于外门。
<br/>　　距离内门有很长的一段距离。
<br/>　　来到内门，还要去剑堂；而后穿过剑堂，才来到楼阁，最后才进入灵气屏障内。
<br/>　　这么长的路程，易老却几乎是瞬间赶到，并救下萧逸。
<br/>　　可想而知，他的实力，绝对比剑堂长老强得多。
<br/>　　这时，刚刚来到的大长老等人，愣愣地看着极界碑上的那道虚影。
<br/>　　“先辈虚影，那是先辈虚影…”大长老语气中带着无比的激动。
<br/>　　“数百年了，整整数百年了。”
<br/>　　“极界碑终于被完全参悟了。”
<br/>　　“裂天剑派，终于要出现剑主了。”
<br/>　　大长老甚至没有发现自己，竟已然热泪盈眶。
<br/>　　其余各长老，也是激动得无以复加，脸上一片潮红。
<br/>　　数百年，是一段不短的时间。
<br/>　　这曾是剑派许多先辈的夙愿，今日，在他们眼中实现。
<br/>　　自然不可能不激动。
<br/>　　当然，长老们都是见惯大风大浪之辈。
<br/>　　很快就稳住了心情。
<br/>　　并立刻发现了面前的情况。
<br/>　　易老，正牢牢地守护在萧逸身边，满脸杀意地看着剑堂长老。
<br/>　　“十一，怎么回事？”大长老连忙出言问道。
<br/>　　易老看了眼还在参悟状态的萧逸，并不希望萧逸受打扰。
<br/>　　身影一闪，瞬间来到大长老等人面前。
<br/>　　而后大手一挥，一道澎湃的真气，彻底横亘在各位长老以及萧逸的中间。
<br/>　　“十一，你做什么？”大长老皱起了眉头。
<br/>　　“只是不想再有小人害那小子。”易老的身影恢复了淡然，但却淡然得毫无情感色彩。
<br/>　　简单来说，那是一种淡漠。
<br/>　　“你什么意思？”大长老疑惑道，“领悟极界碑十成。”
<br/>　　“这是剑派渴望了数百年的事，谁会害他？”
<br/>　　“你说呢？”易老没有反问一声，而后不语。
<br/>　　众长老都是人精，一听便知易老的意思。
<br/>　　目光纷纷看向了四长老，也就是剑堂长老。
<br/>　　“四长老，你想杀萧逸？”大长老冷声质问道。
<br/>　　他虽迟来了几秒，没有看到剑堂长老发出的剑气。
<br/>　　但空气中，却残留着剑堂长老剑气上的气息。
<br/>　　而且，这股气息的轨迹，显然是从剑堂长老手中，一直延伸到萧逸背后。
<br/>　　对于真正修为高深的武者来说，要发现这些气息并不难。
<br/>　　剑堂长老闻言，点了点头。
<br/>　　他知道不可能瞒得过一众长老。
<br/>　　他脸上的不甘之色一闪而过。
<br/>　　但很快，他就恢复了镇定自若的神色。
<br/>　　沉声道，“想来你们也是感知到极界碑有异动，才立刻赶来的。”
<br/>　　“我最先赶到，却看到极界碑中力量暴走，灵气屏障极为不稳。”
<br/>　　“那时，先辈虚影还未出现。”
<br/>　　“我也并不知道，力量暴走是因为要凝聚先辈虚影。”
<br/>　　“只以为是那小子不自量力，强行参悟，触犯了极界碑上禁制。”
<br/>　　剑堂长老说着，冷冷地看了萧逸的背影一眼。
<br/>　　“你们很清楚，极界碑对剑派有多重要，不容有失。”
<br/>　　“在那样千钧一发的境地，我只能做出这样的选择，杀了那小子。”
<br/>　　“以免极界碑出现意外。”
<br/>　　“这…”众长老，将信将疑。
<br/>　　剑堂长老继续道，“那小子不过是控火兽武魂，如此低劣的资质，你们相信他能彻底领悟极界碑？”
<br/>　　“起码，在这先辈虚影出来前，我是不信的。”
<br/>　　这时，大长老点点头，道，“十一，或许真的是误会。”
<br/>　　“再者，萧逸如今并没有受伤，还是等他彻底从参悟状态醒来再说吧。”
<br/>　　“哼。”易老冷哼一声。
<br/>　　但没有多言。
<br/>　　或许说，他不屑于多言。
<br/>　　淡漠地转过了身，负手而立，眼睛，紧紧地注视着萧逸。
<br/>　　有他在，萧逸绝不可能再有危险。
<br/>　　其余长老，则目光纷纷看向极界碑上的先辈虚影。
<br/>　　在他们眼中，那道虚影是定格的。
<br/>　　但，在参悟状态的萧逸眼中，却不是。
<br/>　　这也是他还未醒来的原因。
<br/>　　那道虚影，正在舞着剑。
<br/>　　一剑又一剑，玄妙而莫测。
<br/>　　每一剑，似乎都蕴含着他一生的武道知识。
<br/>　　每一剑，似乎都是对霸道之剑的最好诠释。
<br/>　　萧逸看得入神了。
<br/>　　他知道，那不是真正的裂天剑魔。
<br/>　　只是极界碑上残留的力量凝聚而出的虚影。
<br/>　　无论是玄界碑、元界碑、还是极界碑。
<br/>　　除了记录有这位武道先辈一生的武道知识外；
<br/>　　还残留有他生前的一丝力量。
<br/>　　待得参悟者将石碑完全参悟，这位先辈的虚影就会自动凝聚。
<br/>　　或是寓意着这位先辈想看看是哪位后辈领悟了自己的武道知识。
<br/>　　又或者，代表着这块武道石碑已经完全认可了参悟者。
<br/>　　当初萧逸在武道碑林参悟玄界碑时，除了得到武技‘电光火石’外，也看到了那位先辈的虚影。
<br/>　　只不过，那位先辈生前只是破玄九重武者。
<br/>　　凝聚出来的虚影，也极淡，几乎看不到。
<br/>　　那次的内门执事，并未能感知出来，只有萧逸自己能看到。
<br/>　　这次的极界碑，力量极大，凝聚的虚影，连旁人都能肉眼可见。
<br/>　　言归正传。
<br/>　　数个时辰后，先辈虚影渐渐变淡，而后消失。
<br/>　　刚才的舞剑，原来是极界碑上武道知识的再一次诠释。
<br/>　　如果说，之前萧逸对霸道之剑只是了解，现在则是融会贯通的地步。
<br/>　　如果说，之前对崩山斩、覆海斩和裂天斩，只是单纯领悟。
<br/>　　现在，则是如同他自己曾经都施展过了一般，了然于胸。
<br/>　　崩山斩和覆海斩自不必多说。
<br/>　　裂天斩，待得他有足够的真气时，便可立刻施展。
<br/>　　虽然这一天，不知要到什么时候。
<br/>　　但终究是极大的收获。
<br/>　　半晌，萧逸渐渐从参悟状态中醒来。
<br/>　　一醒来，立刻感觉到背后不远处有人。
<br/>　　一惊之下，回头一看，待看到是十大长老以及易老后，吓了一跳。
<br/>　　“参见长老们。”萧逸拱拱手，行了一礼。
<br/>　　“免礼。”大长老欣喜若狂，满脸慈祥地过来扶起萧逸。
<br/>　　其余长老，除了剑堂长老外，也纷纷面露欣慰笑意地走了过来。
<br/>　　只有易老，默不出声，身影一闪，瞬间消失在原地，已然离开。
<br/>　　有大长老等人在，他知道剑堂长老不可能再伤害萧逸。
<br/>　　“易老。”萧逸一愣。
<br/>　　大长老摆摆手，道，“不用管他，你这师傅，性格古怪。”
<br/>　　萧逸点点头，道，“不知长老们可有什么事要找小子？”
<br/>　　“有是有，不过不急。”
<br/>　　“你可知，你参悟这极界碑多久了？”
<br/>　　萧逸皱了皱眉头，回答道，“至少三个月了吧。”
<br/>　　“四个半月。”大长老笑道。
<br/>　　“四个半月？”萧逸一惊，道，“那岂不是说，全派大比已经开始了？”
<br/>　　“对。”大长老道，“所以，你先去参加吧，之后我们再去找你。”
<br/>　　恰在此时，一道反对声响起。
<br/>　　“不可。”
<br/>　　剑堂长老喝道，“萧逸不曾参加抽签，不曾参加之前的比武，岂可现在忽然加入，打乱全派比武的规则。”
<br/>　　“他参悟极界碑，以致耽误了全派大比的时间，是他自己不重视所致。”
<br/>　　“鱼与熊掌，不可兼得。他既觉得参悟极界碑更重要，不惜错过全派大比。”
<br/>　　“后果，自然有他自己承受。”
<br/>　　“裂天剑派立派无数年，全派大比，乃最大之盛事，从未为什么人开过特例。”
<br/>　　剑堂长老脸色严肃，一副执法严明的样子。
