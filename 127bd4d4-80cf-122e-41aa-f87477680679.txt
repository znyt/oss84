        &lt;h3&gt;第三卷第一百四十三章  贵州三人行&lt;/h3&gt;
<br/>
<br/>        凌辰接到宁雅电话的那一瞬间，心里是开心的、激动的、但是宁雅那公事公办的不带一点温度的声音却让他的心里在抓狂。
<br/>
<br/>        原来，她是因为有事情才找他。
<br/>
<br/>        呵，女人！无情！
<br/>
<br/>        既然决定了不谈感情，既然人家都有了男朋友，他再也没有必要和她牵扯不清，他不是那种一辈子都想当备胎的人。
<br/>
<br/>        放下，是对自己的救赦。
<br/>
<br/>        这几天他们准备回家，但是厂家要求他们留下来，配合厂家的工作，一定要做好年后的工作，况且新年来临，对他们这个行业来说也，也是销量旺季。
<br/>
<br/>        他们几个正在商量着最近的工作计划，宁雅的电话打来，大家都静静的看着凌辰接电话，董亚子在就坐在凌辰的身边，紧靠着凌辰，她注意着凌辰的表情，凌辰先是一喜，接着就是语气冷冰。
<br/>
<br/>        然后她适时的在旁边催了一把，凌辰被她催着挂了电话。
<br/>
<br/>        刘诚之听着董亚子那娇滴滴的声音就来气，直接走到他们中间，将董亚子推开：“董总，凌总，你们看看这个营销方案可行？”
<br/>
<br/>        凌辰并不知道他挂了宁雅的电话，宁雅有多难过，他也来不及感受自己的心里疼不疼，就和董亚子、刘诚之他们讨论起了工作。
<br/>
<br/>        厂里一直要等到大年三十这一天才放假。其实一些人已经陆陆续续的走了，绝大多数员工的家都在附近。董亚子她们家世代定居在这，有今天龙头老大的地位也是靠几代人辛苦打拼出来的。在这个地界是无人不知无人不晓，连带着整个镇子形成了一条供给工厂运转的产业链。
<br/>
<br/>        几个人讨论完了工作，都不约而同的想上街转一转，散散心。
<br/>
<br/>        “凌辰哥哥，要不要去街上转转？”董亚子看着凌辰那张万年不变的脸问。
<br/>
<br/>        “诚之，去不去？”凌辰问刘诚之。
<br/>
<br/>        刘诚之感觉浑身起了鸡皮疙瘩，他怎么感觉老板今天怪怪的？天啊，过了这个年，赶紧回家，还是家里好。
<br/>
<br/>        “去，转转也好，我们明天就回家了。”刘诚之大学毕业，跑出来实习，这一实习就在凌辰身边呆了两年，跟着凌辰学了不少东西，尤其是在人际交往上。
<br/>
<br/>        凌辰对人际关系是把控的恰到好处，上上下下对他都很佩服，当然除了何少华之外，何少华是仗着自己在甸城有点背景，但是井底之蛙总堪大志？甸城虽然山高路远，毕竟是在地球上。
<br/>
<br/>        董亚子见凌辰那么亲密的问刘诚之，心里也有点不舒服的感觉，本来是跟着凌辰的，但是又故意的拉开了和凌辰的距离。
<br/>
<br/>        “刘诚之，你们老板转性了？”董亚子拉拉刘诚之的衣袖，刘诚之一下子蹦老高。
<br/>
<br/>        “你别碰我，我可不想被你逼着负责。”
<br/>
<br/>        “刘诚之？”董亚子感觉再一次被刘诚之嫌弃了。
<br/>
<br/>        “干嘛？你别用你的身份来压我，我告诉你，我天不怕地不怕。”刘诚之被董亚子拽着衣服。
<br/>
<br/>        董亚子手中拽着刘诚之的衣服，感觉衣服的面料和手感非常好，不由用力又拽了一下，将刘诚之拽到了身边，翻起刘诚之的袖子看了看。
<br/>
<br/>        “刘诚之，你......”
<br/>
<br/>        刘诚之捂住了董亚子要喊出口的话：“不准说出去，你要是说了：i  will  kiss  you！”
<br/>
<br/>        董亚子睁着大眼睛，嘴里“呜呜呜”，看着董亚子好像还不肯罢休，刘诚之长臂一挥，搂着董亚子的肩膀，直接将董亚子圈了起来。
<br/>
<br/>        “不准说，听见了没有？”刘诚之盯着董亚子威胁。
<br/>
<br/>        董亚子被刘诚之圈住，嘴又被捂上，没有那么大的力气，委屈的只想哭，眼泪在眼睛里只打转。
<br/>
<br/>        凌辰听见身后的动静，转过身，看见刘诚之圈着董亚子:“我好像成了多余的？”
<br/>
<br/>        刘诚之：“平时总是被你喂狗粮，今天也让你吃一次。”
<br/>
<br/>        凌辰：“不错，这狗粮我吃定了，哎呀，不用担心你们俩个嫁不出去了。”
<br/>
<br/>        刘诚之就那样圈着董亚子朝前走着，等董亚子不挣扎了刘诚之才放开她，默默的跟着刘诚之走着。
<br/>
<br/>        凌辰都两次看见她和刘诚之的“亲密”举动了，要说是不误会谁都不会相信的。
<br/>
<br/>        刘诚之这家伙一定有故事。
<br/>
<br/>        “亚子妹妹，告诉哥哥，你是什么时候喜欢我们小诚之的？”凌辰慢下来，和他们并排走着。
<br/>
<br/>        “你哪只眼睛看见我喜欢他了？”董亚子被刘诚之松开了，离刘诚之远了一点。
<br/>
<br/>        “你不喜欢他？不喜欢他让人家负责？啧啧”凌辰那意思“你是不是太渣了？”
<br/>
<br/>        “你......我。”董亚子是百口莫辩。
<br/>
<br/>        “小诚之，你是什么时候喜欢上她的？”凌辰又问刘诚之。
<br/>
<br/>        “你是哪只眼睛看见我喜欢她了？”刘诚之学着董亚子的语气。
<br/>
<br/>        “不喜欢就别搞什么暖昧了，小心都嫁不出去。”凌辰开心的说，身边跟着这俩个活宝感觉天气都是好的。
<br/>
<br/>        “我们嫁不出去关你什么事情，还是管好你自己吧。”刘诚之和董亚子同时说。
<br/>
<br/>        “你不准抢我台词。”刘诚之语气有点不爽。
<br/>
<br/>        “是你抢了我的......”
<br/>
<br/>        “first  kiss！”凌辰接过董亚子的话哈哈大笑，竟然觉得心情特别好。
<br/>
<br/>        小雅要是在一起是不是也很开心？
<br/>
<br/>        董亚子和刘诚之相互看了一眼，这是受刺激了？竟然哈哈大笑？
<br/>
<br/>        “刚才是不是小雅姐姐给你打电话？只有小雅姐姐给你打的电话的时候你才很开心。”刘诚之不怕死的问了一句凌辰。
<br/>
<br/>        天气立马阴转多云，一股冷风而来。
<br/>
<br/>        凌辰：“刘诚之，看来你是在我身边呆得不耐烦了。哪儿温暖哪儿呆着去。”
<br/>
<br/>        “董总，我感觉你这里比较温暖，求收留。”刘诚之将董亚子推到了身前，躲过了那马上就要袭击过来的冷空气。
<br/>
<br/>        “你还是不是男人？”董亚子被刘诚之推到身前。
<br/>
<br/>        “你试试不就知道了，问什么问？”凌辰又突然好心情的提醒了一下董亚子。
<br/>
<br/>        “刘诚之，你快告诉我，他是不是受刺激了？”董亚子说，在她的印象中，凌辰一直都是冷冰冰的。
<br/>
<br/>        “最近失恋了，早上接了个电话，心情又大好，这种行为，算了，不说了，我不懂，你也不懂！”刘诚之离凌辰有一米远。
<br/>
<br/>        “早上是他女朋友打的电话？”俩个人当着凌辰的面直接讨论起凌辰来了。
<br/>
<br/>        “估计是和好了，你没戏了。做个董妹妹其实也挺好的。小雅姐姐在一个公司上班，那个公司名气也很大，说不定你还能认识。”刘诚之笑着解释。
<br/>
<br/>        “什么公司？”董亚子问。
<br/>
<br/>        “某某信息化有些公司。”
<br/>
<br/>        “他们公司的老板是个女人，陈子昂！”
<br/>
<br/>        “我就说你知道吧。”
<br/>
<br/>        “陈子昂曾经被我们导师拿来做了一次课题，不过老板厉害，不代表你小雅姐姐厉害。”董亚子不服气的说。
<br/>
<br/>        “如果小雅干起工作来，也不输陈子昂！”凌辰冷冰冰的声音飘过来。
