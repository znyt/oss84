　　半小时后，狼牙三十九小队全员出发，乘坐了投放物资的直升机离开，脱离了航道改为飞往阴阳谷方向。
<br/>　　作为埋骨大战中唯一脱离大部队的一百人队伍，第三十九小队全体人员都抱着必死的决心，前往阴阳谷这一任务没有战略方案，没有侦查员的各方面支持甚至没有指挥官。
<br/>　　他们所有人都知道这次行动过后，说不定第三十九小队就不复存在……
<br/>　　从神秘家族中抢人？
<br/>　　异想天开！
<br/>　　他们要做的只能是拖延时间。
<br/>　　“全体准备！”第三十九队的队长大声下令：“一二三，跳！”
<br/>　　哗啦啦！
<br/>　　连续着一百人从高空跳下，没有降落伞，直升机只是略调高度后，这一百人就全体跳了下来，下方是满是各色植被的阳谷外森林，那些植被地便是俯冲地带。
<br/>　　刷刷刷！
<br/>　　高速的穿过丛林植被之中，在中途翻滚落地，这种可怕的空手跳机后一百人连轻伤都没有。
<br/>　　此处充满了野外最大的危险，刚落下便有一人遇到一只巨蟒张大了嘴咬了过来。
<br/>　　噗！
<br/>　　锋利的小刀瞬间刺入巨蟒脖颈，快速的反手用力一拉。
<br/>　　哗啦！
<br/>　　划破血肉的声音伴随着飙血声一起，这条巨蟒便被一下子割断半个喉咙，放血放的吓人。
<br/>　　而那名三十九小队成员，则在这条巨蟒挣扎的时间远离，快速的跟上大部队，丝毫不浪费时间。
<br/>　　降落地就在阳谷入口不远处，很快一百人便全员到达入口平台。
<br/>　　——————
<br/>　　这时候的埋骨之地，狼牙也已经打响了第一场进攻！
<br/>　　“左侧准备，引爆！”何峰的声音沉稳响起，指挥着这场战役。
<br/>　　埋骨之战的第一场战役简单粗暴，由冲锋队直接扛着大量炸药避开高空的飞行丧尸群，前往埋骨之地的丧尸集中地埋下炸药，在此之前的极长时间都是准备期，计算投放点位置。
<br/>　　何峰的话音落下后，一队成员集体向后撤离，很快在前方便听得一声闷爆声响起，紧接着就是一大·片的残值断臂飞上高空，挥洒着漆黑的丧尸血。
<br/>　　腐臭的气味在四处蔓延，炸药味都被掩盖。
<br/>　　而那处引爆的地方，更是直接被炸出了好几个坑，靠近了还能看到没炸死的丧尸在那里爬。
<br/>　　“右侧准备，引爆！”何峰的声音再次响起。
<br/>　　轰！轰隆隆！
<br/>　　又是一连串的闷爆声响，紧接着便是好几个深坑出现，埋葬着大量丧尸的尸骨，以及部分爬不出来的苟延残存者。
<br/>　　何峰的命令每隔一段时间就响起，闷爆声就一连串的接上，从外向内将这处地带炸出了一个深坑圈，将整个埋骨之地的核心地围了起来。
<br/>　　这是一项大工程，一次性就消耗了狼牙所带的一半以上炸药，作用凶猛也快刀斩乱麻。
<br/>　　当这大深坑圈出现之后，不管深坑是否一致也不管丧尸死亡数量多少，何峰及全体在一线的狼牙战团成员都松了一大口气。
<br/>　　因为这次战役的第一步，已经完成！
<br/>　　“清理。”何峰下令，比起之前的果断，这回他的声音更为沉稳，显得尤其慎重。
<br/>　　围绕在各个地点的狼牙战团成员开始缓慢朝前移动，所有人都分散而开隐藏在丧尸的尸骨堆中，用腐臭掩盖着人类气味，一步步的靠近那一圈深坑地带。
<br/>　　他们要做的是占领这个圆圈，然后与后方的大部队接应，紧接着步步攻下圆圈范围内的所有地带！
<br/>　　第一批冲锋陷阵的成员由全体三战队及部分精英队成员组合而成，人员数量比起整个狼牙战团来说并不多，但却任务最为艰巨又危险，虽然炸出了一个圈的深坑，但前方的大·片埋骨中心地全是丧尸，四周更有不断游走的丧尸群会出现，一不小心就会引发难以挽回的可怕局面。
<br/>　　所以所有人都尤其的小心，前进速度也空前缓慢。
<br/>　　徐锋带领的第一小队在前进中停下了好几次，他们无声的相互打着狼牙特有的手势，力保稳妥。
<br/>　　“停。”徐锋又一次下令停下。
<br/>　　周围一片安静看起来什么都没有，但五十人集体匍匐在地上一动不动，他们会一直维持这样的状态直到徐锋下达新的前进命令。
<br/>　　但这一次的等待时间极长，五十人在原地呆了整整半小时都没有动静，没有人动弹更没有人发出声音，安静充斥着四周让人无端紧张。
<br/>　　就这样又过去了十分钟后……
<br/>　　咔嚓！
<br/>　　一道树枝折断的声音响起，让五十人一下子屏气凝神。
<br/>　　果然有情况！
<br/>　　徐锋紧握金枪，双目犀利的看着前方，余光更是不断注意周围，他没有扭头看向旁边，将自身存在感降到了最低。
<br/>　　咔嚓！咔嚓！
<br/>　　声音越来越近，显然是放松了警惕，踏着地面的枯枝走来，并且数量不是一个。
<br/>　　徐锋的耳朵动了动，轻轻扭头看向了旁边的声音来源处，只见那里已经出现了好几个高大身躯，体型全部都在两米高，紧接着其后一只只的丧尸逐步出现。

<br/>　　这样的显然都是超级丧尸！
<br/>　　“一批超级丧尸队。”徐锋冷笑着，缓慢捏紧了金枪。
<br/>　　队伍中的其余人更是紧握自身武器，望着那批超级丧尸群越来越近，战斗一触即发。
<br/>　　埋骨之地随处可见超级丧尸，而超级丧尸队的出现更说明对方来自埋骨中心地，刚才那一连串的爆破动静可不小，会出现一批超级丧尸队前来查看，一点也不意外。

<br/>　　在这时候这个地段遇到超级丧尸，还是一批以队伍形式出现的超级丧尸，危险程度已经到达了任务最高峰。
<br/>　　这里距离埋骨中心太近了，如果这批超级丧尸队不能一次性解决，之后的所有行动，他们必将遇到数不尽的麻烦！
