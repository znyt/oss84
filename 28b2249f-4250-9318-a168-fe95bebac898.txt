　　猎王的强势霸道让现场沉静了良久，几名高级猎者都扬眉吐气的看着段氏基地的人，这件事已经从猎王的话中透露的明明白白，太明显的针对和阴谋，让猎者联盟损失惨重，甚至还是在猎者联盟众人自发前来支援的情况下。
<br/>　　实在太令人心寒！
<br/>　　若不给出一个说法，先不论猎王的态度如何，这些高级猎者首先就会采取武力行动。
<br/>　　猎者联盟的确是个自由中立的组织没错，但并不代表他们没有归属感和凝聚力，为了保持这种自由中立的绝对存在，相互扶持才是正道，否则哪里有这样良好的环境给他们自由发展？
<br/>　　这些道理但凡在猎者联盟内呆的久的人都明白，更默契的守护着。
<br/>　　看到段江伟没有出声，鲁初雪自然不会放任这种状态持续下去，便打破沉寂道：“猎王，我等真诚的说一句，虽然这件事看上去像是一场针对猎者联盟的阴谋，我等也看出了倪端，但我保证这绝对跟段氏基地没有关系。”
<br/>　　“请您试想，如果这一切都是段氏基地所为，那也太明显太过于表面了吧？”鲁初雪说着，瞬间将问题提高到深处：“或许您知道我们段江伟上将，但应该不知道他的擅长在于谋略，只不过因为楚涵的太过妖孽变·态，才让我们上将在这方面的才能鲜为人知。”
<br/>　　“如果这件事是我们所为，凭借我们上将的才能，怎么可能暴露的如此明显，更将段氏基地推上这般境地？”
<br/>　　“这样谁都能看出的问题，对我们有什么好处吗？”
<br/>　　鲁初雪的一番话出口，让几名高级猎者都冷静下来，反复一想之后觉得的确是这么回事，就是因为太像是段氏基地所为，才更加可以排除在外。
<br/>　　毕竟没有一个上将会这么蠢！
<br/>　　几名猎者都将目光看向猎王，等待他的回复。
<br/>　　猎王则是依然那副游神在外的样子，除了之前的那一刻爆发，至始至终他似乎都不在状态。
<br/>　　此时众人的目光看过来，他也不过是微微偏头，一副心不在焉的样子，却说出了一句让现场气氛骤然一冷的话：
<br/>　　“有句话叫最危险的地方就是最安全的地方，最高端的谋略也可以是绕一个大圈回到原点。”
<br/>　　鲁初雪瞳孔一缩，有些微怒：“您还是在怀疑我们？”
<br/>　　“不存在怀疑，就事论事而已。”猎王轻巧开口：“的确如你所说，这一切都太像是一场段氏基地针对猎者联盟的谋杀，还是借刀杀人的法子，但因为段氏基地和猎者联盟向来无争端，根本没有做出这件事的动机，所以才更像是有人刻意如此。”
<br/>　　鲁初雪松了一口气，道：“您明白就好，我们的确没有针对猎者联盟的意思。”
<br/>　　“那么想一想，跟猎者联盟有仇的势力是哪些呢？”猎王看着眼前的水杯，看似不经意的问道。
<br/>　　此时的段江伟已经将思绪拉回，他复杂的看了眼猎王，没有任何犹豫的脱口而出：“神秘家族。”
<br/>　　四个字一出口，现场众人一下子呼吸都沉重起来。
<br/>　　猎王嘴角轻轻一勾没有说话。
<br/>　　鲁初雪却是猛然一惊：“神秘家族？！怎么会……他们？对猎者们下手的可是丧尸群啊！”
<br/>　　“唯有神秘家族才有这个本事吧？”段江伟也是个说起话来不怕吓死人的存在，当场就说道：“一个能和丧尸交易的势力看起来不存在，丧尸是这世上所有人的敌对一方，一般人没有道理更没有那个能力与丧尸交易，可如果存在一个智慧与人相当的丧尸，比方说丧尸王？然后又有了神秘家族的授意呢？”
<br/>　　鲁初雪咽了咽口水：“神秘家族会为了打压猎者联盟，而与丧尸合作？”
<br/>　　段江伟深思道：“神秘家族到底是什么，又有谁知道？我们把他们当做是同类，可对方不一定这么想吧？”
<br/>　　猎王轻轻转了转手中水杯，道：“如果是神秘家族动的手，他们更应该针对的是总部，坐镇长老才是他们真正畏惧的人，我们这些猎者可入不了他们的眼。”
<br/>　　段江伟眯了眯眼睛，有些不能跟上猎王的思维节奏：“你想说什么？”
<br/>　　猎王摇头：“我只是猜测，会不会有人在其中搅浑水？”
<br/>　　“无端猜测没有必要。”段江伟打断他，道：“眼下更应该注重真实情况，但这次事件已经发生无法挽回，我想知道猎者联盟接下来的打算？如有需要我们段氏基地一定会帮忙。”
<br/>　　正当众人将会议展开到最后阶段的时候，忽的会议大门被人急促敲响。
<br/>　　“上将！”一名汇报成员慌张的跑进大厅，将一份情报递交上。
<br/>　　现场众人也都纷纷中止讨论，神情严肃，在这种会议进行中的当下能被放进来的消息，一般都是非常紧急的情报，不容闪失。
<br/>　　段江伟快速将消息看完，而后便彻底愣住，眼神有些意外和无措的看向了猎王。
<br/>　　“又是和我猎者联盟有关？”猎王也皱起了眉，深觉事件有些不简单。
<br/>　　“猎者联盟总部，出事了……”
<br/>　　——————
<br/>　　在距离狼牙地界北面的一处山谷里，一条溪水从高山上流淌而下，山上是雪峰，这溪水便是融化的雪水，带着冰冷彻骨的寒意。
<br/>　　这条溪水的旁边是大·片的青石滩，与水边的杂草混在一起，高耸而上遮住了大·片的视野，也将旁边并不起眼的一座矮房遮挡。
<br/>　　吱呀——
<br/>　　矮房的门从内被推开，一名身穿红衣的女子走出，她的装扮在这片山野中显得很奇怪，但颜色分明的对比，又似乎理应如此。
<br/>　　女子似乎闻到了不同寻常的气味，皱了下眉走向前方小溪边，她拨开茂密的杂草后，便看到一个浑身是伤的人躺在那里，不知是死是活。
<br/>　　女子嘴唇勾了勾，笑容有些冷魅：“这么巧，大总管？这是被人追杀，还是被人抄家？不过竟然能被水流冲到我这里来，难道是山底密道被开启了？”
<br/>　　想到这里的女子一顿，眼神幽幽的往远处雪山上看了一眼：“总部出事了么？”
