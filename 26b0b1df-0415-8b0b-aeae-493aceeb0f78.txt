        &lt;h3&gt;第三卷第一百六十九章  梦中有你&lt;/h3&gt;
<br/>
<br/>        陈子澜在甸城出了车祸，让闫欢心急如焚，甸城是不是更加凶险了？
<br/>
<br/>        谷强可好？
<br/>
<br/>        她已经等不及陈子昂了，她决定先去甸城看看。
<br/>
<br/>        她对陈子悦说：“子悦，我想提前去甸城看看情况。”
<br/>
<br/>        “不行，你一个人不能过去，等子昂过去了再说。”
<br/>
<br/>        陈子悦坚决不同意，三姐曾经阻止过陈子昂去甸城，她可是听见了，现在她们的生意也不错，她和闫欢这么多年的感情比亲姊妹还要好。陈子悦是无论如何都不同意闫欢一个人去甸城的。
<br/>
<br/>        闫欢去意已决，准备过完新年就去甸城，闫欢在年前接到了家里的电话，说妹妹要回来过新年，让闫欢早点回家。
<br/>
<br/>        闫欢比闫芈大了两岁，从离开学校之后，他们就很少见面，闫芈结婚后偶尔回一趟家也是来去匆匆。
<br/>
<br/>        这么几年大概能回来个两次，但是都是一个人，匆匆来匆匆去。闫欢也就中间见了一次闫芈，她想知道闫芈过得怎么样，闫芈很少谈论家里的情况，感觉不是很好。
<br/>
<br/>        这一次闫芈竟然能回家过年还真是稀罕。
<br/>
<br/>        闫芈回来了她可以多问问甸城的情况，毕竟秦少卿是甸城的老总，闫芈自己现在也是甸城一个部门的老总，说不定还能认识谷强。
<br/>
<br/>        闫欢对陈子悦说：“过年我得回老家一趟，我妈打电话给我，妹妹要回来，商场的事情你多操份心，尤其是过节期间，特别要注意。”
<br/>
<br/>        “好了，妈婆，我知道了，好好回家去休几天假吧。”陈子悦知道闫欢的妹妹很少回家的。
<br/>
<br/>        “你可以将妹妹接到西城来住几天，让她看看我们现在的事业。”
<br/>
<br/>        “嘿，你还想显摆给我妹妹看呢？人家也是一个大老总呢。”闫欢撇着嘴。
<br/>
<br/>        “哪里是显摆啊，是让她安心，有这么一个有能力的姐姐，家里就不用她操心了。”陈子悦敲了一下闫欢的头，俩个人并排着走进电梯。
<br/>
<br/>        其实陈子悦说的也对，这些年闫欢将家里照顾得很好，让她父母亲来城里住，父母亲不愿意，闫欢就出资将家里的房子盖成了小别墅，冬天有电暖气，夏天有空调，哥哥在农村收购苹果，将苹果供到他们商场，小日子过得不错。
<br/>
<br/>        “说真的，这些年我们也算是有成就了，要不是现在的政策好，就算是我们有通天的本事也不行，说来说去，还是我们赶上了好时代，”闫欢穿着黑色的毛衣、黑色的直筒甩裤，灰色的大衣，显得成熟而时尚。
<br/>
<br/>        陈子悦是黑色高领毛衣，短款羽绒服、蓝色牛仔裤，黑色马丁靴。
<br/>
<br/>        俩个人因为工作的原因，衣着总是简单大方。
<br/>
<br/>        “是啊，原来我都没有想过我会过上今天这样的生活。”陈子悦拿出车钥匙，“你今天开那辆红色的车回去，我开宝马。”
<br/>
<br/>        “我开那车回去干嘛啊，我开那辆商务，回去人多。”那辆红色车是陈子悦最钟爱的，她可是狠下心买回来的。
<br/>
<br/>        “也行。”陈子悦也不再坚持，闫欢回去肯定会有家里人坐车，商务车坐的人多。
<br/>
<br/>        “王老师来西城过年吗？”闫欢问王浩楠，陈子悦愿意过这种生活她也没有办法，每个人都有每个人的想法。
<br/>
<br/>        “应该会来吧，可能他爸爸妈妈也会来。”
<br/>
<br/>        王浩楠在他们单位是什么情况她也不问，王浩楠回来了他就是家里的一个人，不回来家里就没有这样一个人。
<br/>
<br/>        王浩楠倒是每个礼拜都回来，只是他们再也不在一起了。
<br/>
<br/>        陈子悦什么都不说，冷处理，弄得王浩楠不知道怎么办才好，陈子悦人漂亮又能干，如果和他离婚了，那他也是得不偿失，外面的女人再好，也比不上陈子悦的好。
<br/>
<br/>        可是和外面的那个女人也多少是有点感情，纠缠了很多年，校领导也是知道了这件事情，将他们俩个人的工作进行了调整，俩个人因为距离远了，而且那个女人总是闹着要和他结婚，思来想去，和陈子悦离婚的成本比较大，王浩楠收了心，慢慢的和外面的女人淡了下来，再加上每个礼拜都得到西城，一来二去王浩楠也慢慢的将心思放在了家庭上。
<br/>
<br/>        陈子悦只生了一个女孩，但也是他王浩楠的骨肉，每次回来俩个女儿围着他，王浩楠周末回来会下厨做饭，吃得俩个女儿每周都盼着王浩楠周末回家做饭，闫欢有时候也会蹭蹭饭。这让王浩楠有成就感。
<br/>
<br/>        慢慢的一家人在外人看起来也是其乐融融，可是陈子悦心里的伤痕却无法融合。
<br/>
<br/>        告别了陈子悦，闫欢开着车回到了陵县，她到家的时候其实哥哥都已经把该准备的都准备好了。父母亲只是希望孩子能在家里多呆几天，闫欢失踪的那几年，父母亲为了寻找她，头发都白了，家里也没有心思打理，妹妹上学都是申请的补助。
<br/>
<br/>        好的是现在一家人都平平安安。
<br/>
<br/>        “闫芈说什么时候到机场？”闫欢问哥哥，她平时忙和闫芈联系的也少。
<br/>
<br/>        “后天到。我们一起去西城机场接她。”哥哥话少，有一句说一句。
<br/>
<br/>        家里的生活安宁而惬意，二楼的阳光充足，闫欢窝在阳台上的摇椅上睡着了，太阳暖融融的透过玻璃晒在闫欢的身上，洒下一层金黄。
<br/>
<br/>        闫欢在睡梦里梦见了谷强被人追杀，跳进了熊熊的火光里，随后传来了剧烈的爆炸声，闫欢的心也裂开了，她在梦里大喊:“陈子寒，你回来，你快回来。”
<br/>
<br/>        她大声的喊着、哭泣着、挣扎着，可是突然又发现陈子寒站在她的身边，冷冰冰的看着她：“你哭什么呢？”
<br/>
<br/>        她又欣喜不已，原来他还活着，她抱着他又哭又笑。
<br/>
<br/>        “欢欢，欢欢，你怎么啦？”哥哥摇醒了闫欢，闫欢的泪水还挂在脸上。
<br/>
<br/>        “哥，怎么是你？”闫欢半醒之间看见了哥哥。
<br/>
<br/>        “你做梦了吗？”哥哥关切的问，“我听见你在喊一个人的名字，你没事吧？”
<br/>
<br/>        “我做梦了，梦见一个朋友出事情了。”闫欢还没有从悲伤里走出来。
<br/>
<br/>        “梦都是反的，你的那个朋友一定过得很好。”哥哥安慰着闫欢。
<br/>
<br/>        “但愿吧！”闫欢揪心的疼。
<br/>
<br/>        回来这么多年，她从来都没有忘记过谷强，也知道了谷强就是陈子寒，似乎和陈子悦在一起就是提醒她要时时刻刻记着谷强，记着那个时候的谷强就是她活着的希望，现在的谷强就是她努力的动力。
