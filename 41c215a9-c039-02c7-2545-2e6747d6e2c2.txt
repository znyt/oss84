        &lt;h3&gt;第三卷第七十七章  甜蜜的爱恋&lt;/h3&gt;
<br/>
<br/>        寒伯安和安男在国外，国庆节七天假，对他们来说也是难得清闲的，寒伯安自从毕业之后就开始着手家族的事业，然后还要干自己的小事业，忙的满天飞。
<br/>
<br/>        而安男呢，自从开始上班之后也是搞学术研究，他们家都是文化人，父母亲都是教授，安家祖祖辈辈都是读书人，安男是生在一个文化底蕴很深厚的家庭。
<br/>
<br/>        安男可以用：满腹经纶来形容了。
<br/>
<br/>        安家祖籍湖北蕲州，蕲州的“家学渊源”，“书香门第”、“家世因缘”为历代国人所看重。家学看重气节、品行、门风，立身做人是第一要紧，学还在其次。古代名门望族留下了家学，也留下了诸多“家训”。有道是“观名扬四海的一代大儒，衮衮诸公，一门风雅，于人于学于家于国，皆成楷模。”
<br/>
<br/>        安家只有安男一个女孩儿，安男的父母亲将孩子培养得知书达理，满腹经纶，留学归来就已经是国学领域里的顶尖人物。
<br/>
<br/>        腹有诗书气自华，安男不但长得美，气质绝伦，尤其是那一对丹凤眼，水灵剔透，像一潭秋水，更像是深海碧波。
<br/>
<br/>        寒伯安和安男正在吃饭，俩个人结婚证已经领了，干脆当这次是蜜月旅行，等回去的时候补办一场婚礼也是可以的，他们所接受的教育基本一样，俩个人也有共同话题，虽然寒伯安回家继承了家族的事业，但是不否认他也是一个有知识的人，而且寒伯安喜好广泛，枪、表、烟、酒、雪茄、西装、皮鞋、车啥的他都懂，可以说是热爱生活，知识面很广的一个人，和这样一个有趣的人在一起，安男是幸福的也是满足的，安男注重学术研究，对于生活上有现场的导师适时指点，何乐不为呢。
<br/>
<br/>        对于安男时常崇拜的眼神，寒伯安是非常受用的，尤其是被自己的老婆崇拜，而且自己的老婆还是之乎者也、满腹经纶的人。
<br/>
<br/>        寒伯安的手机qq提示音响起来了，寒伯安拿起手机看了一下，是江俞轩的。
<br/>
<br/>        “轩轩的信息。”寒伯安经常会给安男讲他们之间的事情，虽然安男不是一个做生意的人，但是该知道的还是让她知道。
<br/>
<br/>        安男很喜欢他们最近的相处方式，寒伯安经常会给她讲一些他们生意上的事情，虽然她不太懂，但是寒伯安给她说了她就知道，也能了解很多。
<br/>
<br/>        帅得掉渣的轩轩：“大哥，我最近怕是一直要在西城了，这边项目已经开始着手了，要参与西城的改造中，后期竞争也比较激烈，父亲让我在西城帮忙。”
<br/>
<br/>        “嗯，我和你嫂子安男在一起。”寒伯安简短的回了几个信息。
<br/>
<br/>        “你安心的做你手头上的事情，我们这边一切进展顺利。后期还会和陈子昂有合作。回去联系。”寒伯安放下手机，专注的看着安男，突然觉得安男的眼睛和陈子昂的眼睛有些像。
<br/>
<br/>        “看什么呢？”安男看着寒伯安沉思的表情。
<br/>
<br/>        “看你的丹凤眼。”
<br/>
<br/>        “丹凤眼不是人人都有的，我妈也是丹凤眼。”安男骄傲的说。
<br/>
<br/>        “有一个人的眼睛和你的眼睛特别像。”
<br/>
<br/>        “谁？”
<br/>
<br/>        |“陈子昂。”
<br/>
<br/>        “陈子昂？银烛吐青烟，金樽对绮筵。离堂思琴瑟，别路绕山川。明月隐高树，长河没晓天。悠悠洛阳道，此会在何年。”安男随口吟了一首诗。
<br/>
<br/>        “同名同姓不同时，一男一女俩不知，碧波寒潭剪秋水，芙蓉不及三两脂。”寒伯安微微一笑，对陈子昂作出了评价，也算是回答了安男。
<br/>
<br/>        “你这样说，我都好奇了，是什么样的女人？”
<br/>
<br/>        “回去带你认识一下，也许你们会相互欣赏。”
<br/>
<br/>        “在自己的老婆面前夸别的女人，也怕只有你了。”安男笑着说
<br/>
<br/>        “优秀的人是不会对自己没有信心的。”
<br/>
<br/>        “寒伯安！”
<br/>
<br/>        “在！”
<br/>
<br/>        “看你回答这么干脆利索就放过你了，我吃好了。”
<br/>
<br/>        “吃好了，我们就走，回去可以加餐。”
<br/>
<br/>        “不，我不能再吃了，我怕胖。”
<br/>
<br/>        “我们加很甜的餐，不会胖的。”
<br/>
<br/>        “你......”安楠脸红了
<br/>
<br/>        “人呢，活在世上最重要的就是一日三餐，这个是人幸福指数最重要的保证。吃饱喝足了才有心情干工作，你说是不是？”
<br/>
<br/>        “你说的好像很有道理。”安男就这样看着寒伯安大谈人生哲理而无法反驳。
<br/>
<br/>        “所以呢，加餐是有必要的。”寒伯安长臂一挥，搂着安楠回去加餐了。
<br/>
<br/>        安男伸出手，握着寒伯安的手，雪白的手臂上一只玉手镯泛着莹莹的光。
<br/>
<br/>        “这手镯真漂亮。”寒伯安赞叹着
<br/>
<br/>        “妈妈的，妈妈说这是她身份的唯一象征，本来不想给我的，但是我觉得太好看了，便拿过来带上了。”
<br/>
<br/>        “喜欢玉的话我到时候给你淘一个。”
<br/>
<br/>        “这个不是普通的玉，我找人鉴定了一下，说是很久以前的，现在可以说是无价之宝了。”
<br/>
<br/>        “那我岂不是得到了两个无价之宝了？我要香香。”
<br/>
<br/>        “贫嘴，我只是很喜欢，还是要还给妈妈的，说不定她会找到她的亲人呢。”
<br/>
<br/>        “有你这一个无价之宝就已经够了，我不贪的。”
<br/>
<br/>        “你要什么没有啊？”
<br/>
<br/>        “认识你之后，世界上所有的存在都不及你的存在，所有的美好都不及有你时的美好。”
<br/>
<br/>        风，低声的呢喃着，温柔的在俩个人身边围绕，花，悄悄的开放，羞涩的花瓣儿悄悄的半遮半掩，太阳在云层里时而羞赫，时而欢快，空气散发着甜蜜的芬芳。
<br/>
<br/>        江俞轩看着寒伯安发的信息，也就不再回信息了，原本他是想问问后期的情况，但是寒伯安的信息透露了两点：
<br/>
<br/>        一，人家和老婆在一起，不想被打扰。
<br/>
<br/>        二，后期是会和陈子昂合作的。
<br/>
<br/>        江俞轩拿着手机翻了一下qq空间，看到了寒伯安晒出来的照片，照片下面配着文字：空气散发着甜蜜的芬芳，有玫瑰花的气息在流淌。
<br/>
<br/>        江俞轩在下面评论了一句：“你们幸福的样子是这个世界上最美好的风景。”
<br/>
<br/>        每个人都很幸福，他也要和张倩楠成为这个世界上最美好的风景。
<br/>
<br/>        愿每个人的空气都散发着甜蜜的芬芳，愿我们都成为这个世界上最美好的风景！
<br/>
<br/>        江俞轩看了一眼忙碌的江俞琪，收回心思，又低着头开始忙碌，这个项目是他真正接手的一个项目，一定要拿下来。
