陆司爵这半辈子都活的冷静睿智，杀伐果断，如高高在上的帝王，但是在陆寒霆和夏夕绾的问题上，他一再的迟疑和犹豫。

<br/>        他也不知道自己怎么了，也许，他…羡慕了吧。

<br/>        回到自己的房间里，陆寒霆没有去洗澡，而是身高腿长的伫立在落地窗边，他修长的两指里燃着一根香烟，不停的吞云驾雾，他很想麻痹自己，很想甩掉脑袋里那张巴掌大的绝色小脸。

<br/>        一包烟很快就抽空了，陆寒霆去了沐浴间冲了一个冷水澡，然后上床睡觉。

<br/>        他很快就睡着了，回到帝都后，他的睡眠一直很好。

<br/>        但是，他做梦了。

<br/>
<br/>        那是一个女孩，身段纤柔妙曼，她身上穿的是吊带睡裙，看着像是裸粉色的，本来牛奶白的肌肤衬的越发赛雪。

<br/>        那画面，简直是限制级的活色生香。
<br/>
<br/>        梦里的他眼角猩红，喉头里像滚过的火炭，大手握住了女孩那双S曲线的柔软腰窝，他嗓音沙哑到不行，“绾绾，你干什么？”
<br/>
<br/>        女孩伸出小手，将他推倒回去，贝齿咬着红唇，一字一句道，“你说我干什么，干…你呀。”

<br/>        陆寒霆倏然睁开了眼，醒了。

<br/>        入眼都是黑暗，现在是凌晨，他在自己的春.梦里醒来。

<br/>        凸起的喉头漫不经心的滚动着，陆寒霆颓废的敛上俊眸，抬手遮了一下自己满是情欲的猩红眼角，他将另一只手探到下面，裤子全湿了。

<br/>        特妈的！
