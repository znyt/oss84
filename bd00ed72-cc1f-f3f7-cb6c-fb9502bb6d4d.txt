这天夏夕绾接到了一个电话，是厉老夫人打来的。
<br/>
<br/>        夏夕绾按键接通，“喂，厉老夫人。”
<br/>
<br/>        “绾绾丫头，”厉老夫人慈爱的声音迅速传递了过来，“过两天就是我八十岁的生日了，到时厉家会办一个生日party，绾绾丫头你一定要过来哦，过来给我庆祝生日。”
<br/>
<br/>        厉老夫人邀请她去厉家参加生日party。
<br/>
<br/>        夏夕绾双眼一亮，她正愁不知道用什么办法进入厉家寻找钥匙，现在这个机会就来了。
<br/>
<br/>        虽然她利用了厉老夫人对她的喜欢，但是生日那天她也会真心诚意的给厉老夫人祝寿的！
<br/>
<br/>        “厉老夫人，那提前祝你生日快乐哦，既然你已经邀请我了，那我一定会准时去的。”
<br/>
<br/>        厉老夫人迅速心花怒放，“太好了绾绾丫头，我就知道你会过来的，我已经派人给你送了晚礼服，记得查收。”
<br/>
<br/>        什么？
<br/>
<br/>        厉老夫人将晚礼服都送过来了？
<br/>
<br/>        这速度也太快了吧？
<br/>
<br/>        这时蓝烟捧着一个精致的盒子走了进来，“绾绾，这几天怎么经常有人给你送东西，刚才又有人送来了这么一个盒子。”
<br/>
<br/>        这一定是厉老夫人送来的晚礼服了！
<br/>
<br/>        “蓝妈，我来拆开看看。”夏夕绾站起身走到了床边，打开了那个精致的盒子。
<br/>
<br/>        盒子里放着一件吊带款的深蓝色星星长裙，简单而精致的设计，坠着一颗又一颗的星星，美的令人窒息。
<br/>
<br/>        蓝烟惊叹道，“天哪，这件晚礼服也太好看了趴，绾绾，这个尺寸都像是为你量身定制的。”
<br/>
<br/>        夏夕绾也惊叹这件晚礼服的美丽，要量身定制一件晚礼服是需要很长时间的，难道厉老夫人早就开始准备了吗？
<br/>
<br/>        夏夕绾更加愧疚了，她该怎么回报厉老夫人这样的厚爱？
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾去了电台，远远的就看到了厉嫣然，厉嫣然身边围了一群女孩子，她们叽叽喳喳一脸兴奋羡慕的看着厉嫣然，
<br/>
<br/>        嫣然，过两天就是你奶奶的八十大寿了，作为厉家的小公主，这一次你的晚礼服一定很好看吧？
<br/>
<br/>        厉嫣然一扫前两日的阴霾，漂亮的小脸上露出了骄傲的笑容，“是啊，偷偷告诉你们，我奶奶请了皇家御用绣娘给我量身定制了一件星空晚礼服！”
<br/>
<br/>        “星空晚礼服？哇！”这些女孩子们两眼放光。
<br/>
<br/>        “我奶奶并没有将星空晚礼服告诉我，是我无意间在奶奶的房间里发现的，我奶奶应该是想给我一个大大的，我奶奶可疼我了！”
<br/>
<br/>        厉家老太君80岁大寿可是轰动整个帝都城的大事，厉嫣然作为厉家的小公主自然再度回归到了璀璨的聚焦灯下，以往这些party她的晚礼服也是私人订制的，昨天她推开奶奶的房间，就发现了星空晚礼服！
<br/>
<br/>        这时厉嫣然一抬头就看到了已经走过来的夏夕绾，她当即得意的勾唇，“夏夕绾，我奶奶80岁的生日party你也会来参加的吧？”
<br/>
<br/>        厉嫣然十分期待夏夕绾来参加这个生日party，前段时间奶奶去夏家给夏夕绾长脸了，那这一次她就要所有人看清楚究竟谁是野鸡，谁是凤凰！
