        她的尾音就像一缕妖精的魔法，声线娇软温柔，像一根羽毛轻轻在他心上撩拨。
<br/>
<br/>        听着季思妍这般热情又深情的告白，傅昼景忽然感觉自己的身体莫名其妙地燥热了起来。
<br/>
<br/>        他的耳根发烫，渐渐地，俊脸上也烫得浮泛起两片红霞，原本淡淡的温热气息也在不知不觉中变得粗气浑浊。
<br/>
<br/>        季思妍暗自得意地微微扬起了红唇，唇瓣从他的耳鬓，滑过他硬朗的面颊，再次落在了他的唇角，轻轻一吻，慢慢地覆上他的薄唇，轻柔而试探着一触即开。
<br/>
<br/>        他体内的药效，终于上来了！
<br/>
<br/>        起初，她还在担心，这药下的分量轻了，会对他没效果。因为，她起初在他粥里下的那种药并不多，如果下多了，来势汹汹的话，她怕他会发现事情不对，而因此厌恶他。
<br/>
<br/>        其实，她并不想这么做。只是，她对自己不再像以前那样自信了。
<br/>
<br/>        因为，自从那个“筱颖”成为阿景的妻子后，阿景对她的一言一行都很矜持。
<br/>
<br/>        阿景一直不肯越界，是为了那个“筱颖”吧！
<br/>
<br/>        所以，那个“筱颖”在阿景的心里，一定有与众不同的地位。
<br/>
<br/>        阿景为了那个“筱颖”，不敢越雷池一步。
<br/>
<br/>        季思妍想要动摇“筱颖”在傅昼景心里的地位，既然无法从“筱颖”身上下手，那么就先从傅昼景身上下手好了。
<br/>
<br/>        她的手段是卑鄙了点，但也是跟阿景的母亲学的。
<br/>
<br/>        与其得不到他的心，那不如得到他的身体，让他愧对于她。
<br/>
<br/>        “阿景，你要报仇吗？只有我才能帮你啊！”季思妍不停地扭动着自己的身体，轻吻着傅昼景的耳垂，每一个动作里都透着满满的风骚，双手一丁点一丁点地在傅昼景身上点火。
<br/>
<br/>        傅昼景一直在努力地克制自己，但意识却越来越模糊，甚至在恍惚间，他眼前的季思妍，变成了童筱颖的样子。
<br/>
<br/>        这是怎么一回事？
<br/>
<br/>        筱颖……
<br/>
<br/>        最终，傅昼景身体上最后一道防线彻底崩塌，他抬起手来扣住她的后脑勺，猛地攫住她双唇，发泄着自己内心这压抑了许久的情愫。
<br/>
<br/>        忽然，暗夜里的一声惊雷，震得床上纠缠中的男人不住侧了眸望窗外风雨飘摇的夜景。
<br/>
<br/>        就像是老天爷在警醒他，让他看清楚面前的这个女人到底是谁！
<br/>
<br/>        季思妍见傅昼景的思绪开了小差，于是愈发情动。
<br/>
<br/>        她不想给他任何思考和犹豫的机会，联翩似的亘动，取代所有未曾出口的语言。
<br/>
<br/>        傅昼景皱了眉不语，微眯着眼睛看女人，思绪却早飞得没了踪影。
<br/>
<br/>        可他看着她的模样还是不住的恍惚，明明热情至此，眼前飘过的，却是那个甜美纯净对他说着“以后，我什么都听你的”童筱颖。
<br/>
<br/>        筱颖的身体，他虽未完全见过，但是那次因为含有麝香的香水惹出来的意乱情迷，差点就让他真真切切地要了筱颖的身体。
<br/>
<br/>        筱颖的温香玉软，让傅昼景那次隔着衣料就能抚摸到她那少女般含苞待放的肌肤，实实在在地扎入了他的心头。
<br/>
<br/>        筱颖……
<br/>
<br/>        筱颖……
<br/>
<br/>        筱颖……
<br/>
<br/>        他的心里，脑海里，想的人，全是筱颖。
<br/>
<br/>        此时此刻，他最想要的人是筱颖……
<br/>
<br/>        房间里的热气氤氲不断，直到窗外瓢泼的大雨侵袭。
<br/>
<br/>        季思妍柔柔地说道：“阿景，别担心，我不在排卵期。所以，不会怀孕的。以后，没你的允许，我是不会怀上你的孩子的。”
<br/>
<br/>        傅昼景靠在床头，从床头柜最上面的抽屉里拿出并不常抽的女士香烟，为自己点上一支，便始终安静地靠在那里。
<br/>
<br/>        季思妍仰起头看着傅昼景若有所思的侧脸，惊讶地问道：“你怎么知道我那儿放了香烟？而且……你……从不抽烟的……”
<br/>
<br/>        傅昼景依然沉默着，边抽烟边看向了窗外漂泊的大雨。
<br/>
<br/>        他刚刚是怎么了，他不想再去追究了。
<br/>
<br/>        现在的他，已经被季思妍牵着，迈向了另一个无法回头的深渊。
<br/>
<br/>        他以为她是筱颖，等到真正清醒时，才发现她是思妍。
<br/>
<br/>        所以，这，才是他真正的一面吗？
<br/>
<br/>        他看不清自己的心，也看不透自己的人。
<br/>
<br/>        傅昼景眸光微沉。
<br/>
<br/>        他其实也有熊熊野心，只不过他一直压抑着自己，压抑着内心深处的那头雄狮。
<br/>
<br/>        季思妍眉目轻闪了一下，还是迅速向上爬了一爬，伸手拽住他拿烟的那只大手，小唇咬了他咬过的滤嘴，轻轻一吸，然后冲着他的方向烟雾缭绕。
<br/>
<br/>        他在迷蒙的光景里看她，她便俯了唇去吻他，弱弱地问道：“阿景，告诉我，你在想什么？”
<br/>
<br/>        傅昼景沉默了几秒侧开头去，将手中的烟蒂在床头柜上的烟灰缸里摁了摁，再推她到被子里，接着翻身而起。
<br/>
<br/>        季思妍拢着身前的被子焦急唤他：“这么晚了，你……”
<br/>
<br/>        “我回我的公寓去了！”傅昼景一边说，一边将散落在地上的衣服捡起来穿好。
<br/>
<br/>        季思妍微微张了下唇，还是快速从床上挪动到床边。
<br/>
<br/>        “不要走，留下来陪我！”季思妍一把从身后抱着傅昼景的虎腰，阻止他正扣衬衫的动作：
<br/>
<br/>        傅昼景淡淡地说：“我真的该走了！”
<br/>
<br/>        季思妍眼见着他拾起掉落在地上的钱包和手机，她慌忙爬到床脚看他：“阿景，你是不是想筱颖了？”
<br/>
<br/>        傅昼景收拾妥当所有的东西，站在床边勾了下她的下巴，落唇的时候却只到她颊畔，连声音都有些微冷：“早点睡，我只是有些认床而已。”
<br/>
<br/>        “阿景！”
<br/>
<br/>        季思妍惊得慌忙伸手去拉他，可他已经旋身，当着她的面拉开卧室的房门向外，任了她的叫唤，他就是义无反顾拉开大门走了出去。
<br/>
<br/>        傅昼景从季思妍的公寓里出来后，上了一辆出租车。
<br/>
<br/>        噼里啪啦的雨点打在车身上，满城寂静的夜色里，好像都只剩下雨点摔落地上时碎裂绽开的声音。
<br/>
<br/>        傅昼景下意识地掏出手机来看了看，才发现上面有一条筱颖发过来的短信。
<br/>
<br/>        昼景，你到家了吗？——筱颖。
<br/>
<br/>        是一个小时以前发的。
<br/>
<br/>        傅昼景回了一条短信过去。
<br/>
<br/>        嗯，到家了，只是这边下起了大雨，回家的路上堵车了。你早点睡，晚安。——昼景。
<br/>
<br/>        临海城，傅家大庄园。
<br/>
<br/>        童筱颖缩在床上，直到手里的手机微微震动，她心里一惊，立即点亮了屏幕看了看。
<br/>
<br/>        昼景终于给她回短信了！
<br/>
<br/>        太好了！！
<br/>
<br/>        嗯，他平安到家了，她就放心了。
<br/>
<br/>        童筱颖情不自禁地微微扬起嘴角，立即回拨了一通电话过去。
<br/>
<br/>        这回，电话通了，傅昼景那边也接听了电话。
<br/>
<br/>        “昼景，你平安到家，我就放心了！”童筱颖柔柔地说道。
<br/>
<br/>        她的娇羞，她的青涩，让他回想起了当年也是十八岁的季思妍。
<br/>
<br/>        傅昼景一边接听电话，一边望向了车窗外，云淡风轻地转开了话题：“滨江城下起了很大的雨！”
