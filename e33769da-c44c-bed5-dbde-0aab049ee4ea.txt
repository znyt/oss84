        &lt;h3&gt;第三卷第二百五十七章  我们总是会相见&lt;/h3&gt;
<br/>
<br/>        寒伯安和李长卿飞回了上海，而沧岚雪和他们一趟飞机。
<br/>
<br/>        李长卿对沧岚雪说：“沧总，是先到上海还是先回甸城？”
<br/>
<br/>        “我可以先去上海签章吗？”沧岚雪问李长卿。
<br/>
<br/>        “你可以先回去好好商量一下，让你们老大签字，我只认你老大的签字，你就算了吧。”李长卿很开心啊。
<br/>
<br/>        “傻屌，先让你开心一下下，然后在慢慢的磨磨你，嘿嘿，这种感觉还不错。”李长卿心里想着沧岚雪后期的那种惊讶的样子嘚瑟的神情溢于言表，他故意的靠近了一下寒伯安。
<br/>
<br/>        “安男不来接机，真是让我好开心啊。伯安，你开心吗？”李长卿笑眯眯的看着寒伯安。
<br/>
<br/>        “你这是想让我翻车的节奏吗？话说，安男不来接机，轩轩应该来，文文应该来啊。”寒伯安有些不高兴了。
<br/>
<br/>        沧岚雪听得浑身直冒鸡皮疙瘩，寒家年轻的总裁嗜好真的是有些让人无法接受，这些个轩轩和文文，大概是他这个好什么，那个好什么来着，他摇了摇头。
<br/>
<br/>        “你还在想他们？他们都不理你了，你想他们干什么？”李长卿责怪寒伯安。
<br/>
<br/>        “行，行，不想了，不想了，那俩个没良心的，最近这一段时间都不和我联系了，真是让人伤透了心，回去看我如何收拾他们。”俩个人完全无视身边的沧岚雪。
<br/>
<br/>        李长卿像突然发现新大陆一样：“沧总，你怎么还跟着我们？想必你也听到了，我们寒总回去要收拾他们，这一段时间顾不上工作了，我也得陪着他，你先回去吧。反正我把字都签了。”
<br/>
<br/>        沧岚雪有点迟疑。
<br/>
<br/>        李长卿低声说：“你过一个礼拜再来，这几天他顾不上，你跟的太紧，如果他生气了，连我也哄不好的那种，你就没有希望了。就因为你挨了我的那一拳，让我心生愧疚，沧总，记得给我账户上转点零花钱啊，我可是费了很大的劲为你说好话。”
<br/>
<br/>        “哦，好，好，你们年轻人的世界我真的搞不懂。”沧岚雪点点头。
<br/>
<br/>        “搞不懂就对了，你搞懂了我们怎么和你玩？”寒伯安嗤笑一下。
<br/>
<br/>        “你还要和他玩？”李长卿的眼神像刀子一样在沧岚雪的身上转了一个圈。
<br/>
<br/>        “不，不，不，你们年轻人好好玩，我年龄大了，不作陪，我先回甸城了，改天见，改天见！”沧岚雪急匆匆的转身离开。
<br/>
<br/>        “册那，沧岚雪还想和我们玩，就这智商看我们不玩死他？”寒伯安冷笑一声。
<br/>
<br/>        “对，让他在我们手中活不过三集。”
<br/>
<br/>        “不，让他活得久一点，我们还有用处。”放长线钓大鱼，寒伯安想看看后面他们的下一步怎么走。
<br/>
<br/>        “我很久没有联系安男了，不知道这几天她好不好。”寒伯安拿起一支雪茄在手中捻着。
<br/>
<br/>        “沧岚雪说安男姐姐和陈子昂走了，你还是打个电话问问，看看她在哪里。”李长卿拿下寒伯安手中的雪茄，催促寒伯安打电话。
<br/>
<br/>        安男和陈子昂已经到达西城机场，安林曦和代凉能比他们早半个小时，在机场等她们。
<br/>
<br/>        安男接到寒伯安电话的时候，刚下飞机。
<br/>
<br/>        看到是寒伯安的电话，安男的心里一松，眼眶发热。
<br/>
<br/>        “伯安。”安男的声音有些颤抖。
<br/>
<br/>        “男男，你在哪里？”寒伯安心里一疼，安男一定是担惊受怕了很久，可是他都不能够陪在她的身边。
<br/>
<br/>        “你在哪里？你还好吗?你回来了吗？”安男觉得她自己都有点语无伦次了，天知道她最近内心是有多么的煎熬，面上却还要带着笑。
<br/>
<br/>        “我回来了，在回家的路上。你在哪里？我去接你。”寒伯安不知道安男在哪里，但是安男被陈子昂带走，绝对不会是在家里。
<br/>
<br/>        “我在西城，刚下飞机，和陈子昂一起。”安男看了一眼陈子昂，陈子昂也在打电话。
<br/>
<br/>        “你去西城了？那我也去。”安男人在哪里，寒伯安的心就在哪里。
<br/>
<br/>        “你要来吗？爸爸妈妈也到西城，陈子昂的父亲生病了。”安男解释着。
<br/>
<br/>        “那我也过去，现在马上！”寒伯安听说安男在西城，立马就想飞到西城。
<br/>
<br/>        “那好，我们西城见。”安男脸上飞起了两朵红云，寒伯安来西城他们一家在一起就皆大欢喜了。
<br/>
<br/>        陈子昂一直都是风风火火的样子，包括走路都带着风，脚下的高跟鞋走出了万千气象。因为她边走便打电话，将安男拉下一截，看了看身边没人，回头看见安男一脸甜蜜蜜的样子，红扑扑的脸上流光溢彩。
<br/>
<br/>        “寒伯安回来了？”陈子昂站着等安男跟上来。
<br/>
<br/>        “嗯，已经到了机场，他说他也过来。”安男有点害羞的样子。
<br/>
<br/>        “哈哈哈，这是有多么的迫不及待啊。你看看你一副小女人的模样，哎哟，我的天，这还没有见面呢，要见面了，啧啧啧，不可描述。”陈子昂看着安男那副小女人的模样，起了逗安男的心。
<br/>
<br/>        “讨厌，再这样我不跟你走了。”安男被陈子昂一笑，脸越发的红了。
<br/>
<br/>        “好了好了，安男小姐姐，我不说了，赶紧的，咱老爸老妈还在机场等着呢。”陈子昂和安男并肩走出机场。
<br/>
<br/>        “大哥什么时候来，房间还没有定呢。”陈子昂不知道寒伯安突然会回来，这是事情处理完了？
<br/>
<br/>        “不用定了，我爸爸和我妈妈住一间，我和他住一间。”安男想也没想的说。
<br/>
<br/>        陈子昂“噗嗤”笑了，然后一本正经的说：“我还准备联系江俞轩，让江俞轩晚上接待大哥呢，你这就自己安排上了。”
<br/>
<br/>        “我怎么能将他安排给别人呢，三哥虽然很好，但是那能比得上我的好，你呀就别操那份心了。”安男开着玩笑。
<br/>
<br/>        “那好吧，既然大哥也来西城了，等你们卿卿我我结束了，我们一起聚聚。我要陪我爸爸两天，你们先玩着，我安排司机给你们。”陈子昂对安男说，她确实是没有时间陪安男的，父亲的病也不知道怎么样，她心里就像是压着一块大石头。
<br/>
<br/>        “行，你不用管我们的。”俩个人说说笑笑的朝代凉和安林曦等待的咖啡厅走去。
<br/>
<br/>        “我给江俞轩打电话通知一下吧，他在西城也很长时间了，看看他有没有时间陪你们逛逛。”陈子昂说。
<br/>
<br/>        江俞轩最近也挺忙的，听说要进军高新开发区和某风景旅游区，那个旅游风景区要盖别墅群，项目很大，不知道江俞轩能不能拿下来。
<br/>
<br/>        “等伯安来了再说吧，说不定伯安没有时间逛的。”安男心里说，我们不在乎逛风景，我们想要见亲人。
<br/>
<br/>        人生就是很奇妙，兜兜转转中，我们总是会相见，会在一起。
