　　走在苍穹神宫白玉铺就的小道上，苏莫心中沉重无比。
<br/>　　接下来他的路，将更加艰险。
<br/>　　不过，从水一的口中，他了解道的消息并不多，关于巫族的情况，对方所知甚少。
<br/>　　不过，距试炼者到来还有近十年的时间，考虑此事还为时尚早。
<br/>　　现在，他的当误之急，是提升实力，一统苍穹世界。
<br/>　　当然，一统苍穹世界之前，首先就要灭了黄泉魔宗。
<br/>　　黄泉魔宗到底与巫族有什么联系？
<br/>　　这也是值得深思的事情，不过，不管如何，黄泉魔宗一定要毁灭。
<br/>　　“还是先突破修为吧！”片刻之后，苏莫叹了口气，想太多反而会给自己带来压力，唯有提升自己的实力才是王道。
<br/>　　很快，苏莫便回到了宫殿中，而后进入虚界神图，准备冲击武皇境三重境界。
<br/>　　这一个多月的时间，他耗损的精气神也恢复了不少，虽然远没有彻底恢复，但对修炼的影响并不大。
<br/>　　神图空间中，苏莫盘膝而坐，将身上的所有神丹，尽数取了出来，包括鲁桂的那枚神丹。
<br/>　　他身上的神丹着实不少，灭了黄泉魔宗十几处据点，足足得到了数千枚神丹。
<br/>　　只不过，这数千枚神丹，九成九都是武王境的神丹，全部加起来也抵不上鲁桂的一枚神丹。
<br/>　　数千枚神丹堆积在苏莫身前，堆成了一座小山。
<br/>　　“吞！”
<br/>　　苏莫轻喝一声，立刻释放出了吞噬武魂，开始了吞噬。
<br/>　　他可没有耐心一部分一部分的吞噬，而是一次性全部吞噬。
<br/>　　霎时之间，无尽的玄力滚滚涌动，浩荡不休，将苏莫完全淹没。
<br/>　　苏莫的修为早已达到了武皇境二重的巅峰，上次也是因为资源不够，才没有突破成功。
<br/>　　现在沉淀了近两个月的时间，又有如此多的神丹、灵石，他的修为已经到了水到渠成的地步。
<br/>　　一个时辰！
<br/>　　两个时辰！
<br/>　　三个时辰！
<br/>　　但即便如此，这一次冲击境界，苏莫也足足耗费了近五个时辰的时间。
<br/>　　不仅如此，数千枚神丹被他耗尽，而且，他身上的灵石也被消耗了大半。
<br/>　　要知道，苏莫身上的灵石也是天文数字，因为他诛灭十数个黄泉魔宗的据点之时，也得到了大量的灵石。
<br/>　　可以这么说，苏莫现在冲击境界的消耗，若是全部用灵石的话，没有几个势力能支撑的住他的消耗。
<br/>　　呼！
<br/>　　苏莫睁开眼眸，张口吐出一口浊气，终于达到了武皇境三重了！
<br/>　　现在，他才是真正的站在了苍穹世界的巅峰，就算是准帝强者，他也不惧了。
<br/>　　当然，苏莫也没有小看准帝修为的强者，毕竟准帝要比一般的武皇境九重武者，强大的多。
<br/>　　这从二魔老身上就能看出来，对方的实力，比之阎泰以及被他杀死的鲁桂等人，强大太多太多了！
<br/>　　又巩固了一日的修为，苏莫离开了住处，前往了苍穹神殿。
<br/>　　苏莫召集了九名傀儡人，询问了一番大婚的安排。
<br/>　　关于大婚的事宜，他已经全部交由了几名傀儡人，几人也都已经安排妥当。
<br/>　　随后，苏莫又在各大宫逛了逛，去看望了父亲、韦长老、清平子、月群等熟人。
<br/>　　因为苏莫婚期将近，众人也都结束了闭关修炼，在等待苏莫大婚之日的到来
<br/>　　苏洪、清平子等人，虽然功法、资源都不缺，但修炼的速度，还是没有达到苏莫的预期，修为最高的清平子，也才堪堪的达到真罡境九重境界。
<br/>　　他们的天赋都不高，而且最佳的修炼时间已过，就算有再高等级的功法，再多的资源，将来成绩也是非常有限。
<br/>　　而一些年轻年轻一代，反而后来居上，超越了老一辈高手。
<br/>　　比如，洛千帆、凌慕辰费狂三人，修为赫然达到了真玄境，尤其是洛千帆，修为已达真玄境三重境界。
<br/>　　苏莫在光耀宫和一众相熟之人待了很久，详细了解了一番众人的修炼情况，稍稍指点了一下之后，便离开了光耀宫，独自前去练剑。
<br/>　　他的大苍穹剑法，作为帝级上品武技，无比的强大，不修炼精深实在是浪费。
<br/>　　他虽然才修炼到入门的阶段，但威力已经不俗，将来若是练到大成，必然强大无比。
<br/>　　时间缓缓流逝，一日又一日，距离婚期越来越近。
<br/>　　在距婚期还有两日之时，帝玄宫和天涯海阁的人，率先来到了苍穹神宫。
<br/>　　帝玄宫宫主帝释，带着数十名强者，亲自来到了苍穹神宫。
<br/>　　而天涯海阁带队之人，是一名身形枯瘦，老态龙钟的老者，此人正是天涯海阁的阁主，海明江。
<br/>　　至于钱副阁主，倒是没有来苍穹神宫，因为其要坐镇在天涯海阁。
<br/>　　不过，如胡副堂主等和苏莫相熟之人，倒是来了。
<br/>　　苍穹神殿之中。
<br/>　　苏莫高座于帝座之上，下方是帝玄宫和天涯海阁的众强者。
<br/>　　“欢迎各位光临我苍穹神宫，有失远迎！”苏莫面带微笑，向两方势力之人抱拳。
<br/>　　“苏宫主客气了！”
<br/>　　“两月不见，苏宫主居然再次突破了修为！”
<br/>　　“真是可喜可贺！”
<br/>　　一众帝玄宫和天涯海阁的强者，纷纷抱拳回礼，不少人眼眸中都隐隐有震撼之色。
<br/>　　每一次见到苏莫，苏莫的变化都翻天覆地，这才不到两个月的时间，居然又突破了修为！
<br/>　　这实在是惊世骇俗！
<br/>　　若是苏莫是武王境修为，两个月时间突破一个境界，还在情理之中，但到了武皇境修为，居然还这么快，众人不得不震惊！
<br/>　　正常情况下，一名武皇境二重的武者，突破到武皇境三重，起码需要二、三十年的时间。
<br/>　　就算是最顶尖的天才，也需要一两年的时间，而苏莫只需不到十分之一的时间。
<br/>　　连帝释和天涯海阁阁主，眸中都是带有深深的震撼。
<br/>　　“苏莫，恭喜了！”帝释向苏莫笑着说道，不过他心中却是有些忧虑。
<br/>　　他让苏莫和夕儿成亲，并且和苍穹神宫结盟，也有自己的打算，那就是从苍穹神宫得到无穷的好处。
<br/>　　不说其它好处，他若是能得到一门帝级中品功法，亦或是帝级上品功法，将来突破到帝境，就有极大的希望了。
<br/>　　帝释年轻之时，曾经来过苍穹神宫，也得到了一个不错的传承，还有一个苍穹神宫弟子的身份。
<br/>　　所以，他清楚的知道苍穹神宫的底蕴。
<br/>　　可是，苏莫的成长速度太快了，让得帝释欣慰的同时，亦是有些许担忧。
<br/>　　如此的话，过不了多久苏莫就天下无敌了，到时候帝玄宫怕是也会被苍穹神宫吞并。
<br/>　　这不是帝释想看到的，但他为了得到自己想要的，却不得不这么做。
<br/>　　帝玄宫，与他自己的前途相比，显然后者更为重要。
