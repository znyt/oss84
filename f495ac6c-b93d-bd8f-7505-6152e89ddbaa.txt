        夏海东有九个儿子，傅夜沉母亲的闺蜜，也就是他傅夜沉的干妈，是夏海东最宠的小儿子夏阳枭的妻子。
<br/>
<br/>        而夏海东的孙子是“凌”字辈，孙女是“玄”字辈，有孙子十五个，孙女三个，本来一共有十八个，但最小的那个小孙女，也就是他傅夜沉的干妈和夏阳枭的小女儿，在刚出生的时候被人给抱走了，至今下落不明。
<br/>
<br/>        不过，听说，傅夜沉的干妈因为思女过度精神失常，夏阳枭为了安抚妻子的情绪，又收养了一个与亲生女儿同年同月同日生的女孩，并且还把自己亲生女儿的“夏玄灵”这个名字给了这个养女。
<br/>
<br/>        换句话说，傅夜沉的干爹是夏阳枭，虽然夏阳枭有个亲儿子，但是能力却不及傅夜沉，所以夏阳枭在某些方面还是比较重视傅夜沉。
<br/>
<br/>        连华生所知道的也就这些。
<br/>
<br/>        对于夏家的事情，傅夜沉略知一二，以前倒是听爷爷和母亲说过，但除了干妈一家子的事，其他的事，他并未记在心上过。
<br/>
<br/>        “话说回来，阿沉，那个落以柔的出生年月日刚好和夏局长的那个被人抱走的女儿一样，你说，落以柔会不会是夏局长的那个失散多年的女儿？要不要拿她一根头发去验一验？”连华生忽然联想到这件事情，一本正经地问。
<br/>
<br/>        傅夜沉不以为然地回答道：“不要多管闲事。”
<br/>
<br/>        夏阳枭要是想找回女儿，早就动用他的权利去找了，但是这么多年过去了，他却一直没派人去找，这说明了什么？
<br/>
<br/>        具体是什么原因，傅夜沉猜不到，也揣摩不出来，但是一种直觉告诉他，夏阳枭似乎就不太想找回这个女儿。
<br/>
<br/>        “诶，真搞不懂。”连华生感慨不已。
<br/>
<br/>        傅夜沉将手套、拖把、纸帽和围裙一一递给连华生，微眯起黑眸，扬起下巴，冷冷地命令：“剩下的活，交给你了，我去找筱颖。窗户抹不完，这个月的工资，你就别想要了。”
<br/>
<br/>        “我去！你这黑心的资本家！”连华生苦憋着脸抱怨，“我还没吃早饭的！”
<br/>
<br/>        “现在已经十一点钟了！”傅夜沉一边点了点手腕上的名表，一边转身往楼下走去。
<br/>
<br/>        他一路从楼梯上下去并未见着童筱颖的身影，于是去了厨房。
<br/>
<br/>        而童筱颖果然在厨房里忙活。
<br/>
<br/>        明净整洁的厨房里，瓦罐电锅里“咕嘟咕嘟”直冒着热气，里面的鲍鱼海参汤熬得又浓又香。
<br/>
<br/>        以刘婶和童筱颖的厨艺，在家里都可以开大厨餐厅了。
<br/>
<br/>        “筱颖，我觉得，我很幸福。”傅夜沉走到童筱颖的身后，双手搂在了她的腹部上。
<br/>
<br/>        他高大的身体，将她娇小的身躯安全笼罩在自己怀中。
<br/>
<br/>        童筱颖一边择菜，一边忍俊不禁道：“大哥若是不结婚，也可以跟连医生过日子。”
<br/>
<br/>        刚刚，他俩那画面，CP感实在是太强太养眼。
<br/>
<br/>        “如果你不嫁给我，我真打算光棍一辈子。”傅夜沉附到童筱颖的耳边，轻轻地咬了咬她柔嫩的耳垂，呢喃细语道。
<br/>
<br/>        童筱颖缩了缩肩头，只觉自己的耳廓好像稍稍有点发烫，嘴角情不自禁地微微上扬，心里莫名其妙的甜丝丝的。
<br/>
<br/>        他的一只大手，拨开她颈后的长发，薄唇从她的耳垂游移到颈项，另一只手则从她的腹部慢慢往上爬去。
<br/>
<br/>        她衣襟上的纽扣，被他不着痕迹地一粒粒解开，修长的大手像水蛇般溜进了衬衫里。
<br/>
<br/>        “大哥，快停下来，这里是厨房……”只是片刻功夫，童筱颖就感觉有些喘不上气来，小脸涨得通红。
<br/>
<br/>        从昨晚起，他对她像是上瘾了一般，没有让他“吃”透，他心里就老是惦记着这事儿。
<br/>
<br/>        一边隐忍、一边矜持、一边还不忘试图引诱她越线……
<br/>
<br/>        他没有说话，身体紧紧地贴着她的后背。
<br/>
<br/>        她被他弄得衣衫不整，酥胸半露，圆润的香肩上有他烙下的印记。
<br/>
<br/>        童筱颖感觉到是傅夜沉的金属皮带扣抵到自己的腰后，两人又贴得太紧，金属皮带扣太硬，让她很不舒适地抱怨了一句：“大哥，你的皮带扣戳到我了。”
<br/>
<br/>        傅夜沉似笑非笑地扬眉，重新覆到她的耳边，暧昧低沉地挑逗：“筱颖若是嫌我的皮带碍事，不如筱颖来帮我解了，如何？”
<br/>
<br/>        “大哥，别闹了。”童筱颖敛了敛差点被傅夜沉诱惑进入到意乱情迷里的理智，制止住他的那双乱动的大手，“不然，我真生气了！”
<br/>
<br/>        “宝贝，乖，不生气了。”傅夜沉及时收手，炙热的薄唇，吻了吻童筱颖微烫的脸颊，接着说道，“我要出去一趟，今天中午，不会在家吃饭。”
