夏夕绾从床上坐起了身，她将他的回信来回看了三遍，最后颤抖着指尖发给他：流氓！
<br/>
<br/>        看到“流氓”两个字的陆寒霆挑了一下剑眉，喉头里溢出了低低而愉悦的笑声。
<br/>
<br/>        她性格聪慧从容，很少有生气炸毛的时候，他知道他将她给惹急了。
<br/>
<br/>        陆寒霆慢条斯理的回了一个骂我？回去再收拾你！
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾又躺回了床上，她现在整个人就像是煮熟的大虾，他最后那句“回去再收拾你”简直让她无法直视。
<br/>
<br/>        用力的闭上眼，夏夕绾将脑海里陆寒霆那张可恶的俊脸给甩掉了，然后开始考虑正事。
<br/>
<br/>        很明显苏希这一次回来就是跟她作对的，他现在是海城四大豪门，有权有势有地位，很难对付。
<br/>
<br/>        这种情势对她十分的不利。
<br/>
<br/>        还有有一点很奇怪，为什么苏希说她脏了，还说她是别人穿过的破鞋？
<br/>
<br/>        夏夕绾跟叶翎聊了这个话题，叶翎变态希在放p，我家绾绾还是处呢。
<br/>
<br/>        夏夕绾可是他为什么这样说？
<br/>
<br/>        叶翎去问他。
<br/>
<br/>        夏夕绾觉得十分有道理，自己在这里猜来猜去是没有用的，她要找机会当面问清楚。
<br/>
<br/>        叶翎绾绾，说真的，变态希没有那么难搞，他喜欢你，对待男人，如果你愿意，你就可以。
<br/>
<br/>        夏夕绾没有再回，她当然明白叶翎是什么意思，对待喜欢自己的男人，其实有很多办法可以化被动为主动的。
<br/>
<br/>        ……
<br/>
<br/>        夜晚，苏希将夏小蝶带到了酒店的总统套房里。
<br/>
<br/>        苏希刚洗过澡，身上披了一件白色浴袍，他往高脚杯里倒了一点红酒，优雅的抿了一口。
<br/>
<br/>        这时沐浴完的夏小蝶从后面抱住了他，“苏希哥哥，你说我爸是不是很爱夏夕绾的妈咪，这么多年都念念不忘？”
<br/>
<br/>        苏希看着落地窗外的霓虹灯，勾了一下薄唇，“怎么这么说？”
<br/>
<br/>        “我知道我爸的皮夹里一直放着夏夕绾妈咪的照片，那张照片他宝贝的很，从来不让别人碰，而且这么多年了，我爸不许任何人说夏夕绾妈咪的坏话，今天我妈说了，我爸就像变了一个人，他那么爱面子的一个人，绝对不会在公开场合让别人看笑话的，但是他竟然在酒店外面就打了我妈一巴掌。”
<br/>
<br/>        苏希脑海里浮现出一张绝丽的容颜，淡淡道了一句，“夏夕绾妈咪是一个传奇女子。”
<br/>
<br/>        夏小蝶没有见过夏夕绾的妈咪，十年前夏夕绾妈咪去世了，这个女人就好像人间蒸发了一样，很难再听闻她生前的那些消息。
<br/>
<br/>        夏小蝶也想过，想要在这座四九城里抹去一个人全部的信息，这需要多大的一只手，多么强悍的力量。
<br/>
<br/>        每当想到这里，夏小蝶都觉得害怕。
<br/>
<br/>        “苏希哥哥，我妈有一点没有说错，夏夕绾勾引男人的本事真的好厉害，她嫁到幽兰苑去冲喜，又在外面包养了一个小白脸，现在又跟帝豪酒店扯上关系，她不知道有过多少男人了！”
