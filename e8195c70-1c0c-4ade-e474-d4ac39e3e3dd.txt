　　古魔太子非常的会把握时机，就在苏莫出最后一剑，准备将霸云公子击出大阵范围之时，他便出手了。
<br/>　　当然，他并不是救霸云公子，而是要擒拿苏莫。
<br/>　　就在苏莫刚刚击出最后一剑之时，他的周身，瞬间出现滚滚魔雾。
<br/>　　这滚滚魔雾，仿佛是魔气的海洋，瞬间就将苏莫淹没在其中。
<br/>　　“什么？”
<br/>　　“不好！”
<br/>　　“是古魔太子！”
<br/>　　叶凌天和龙腾等人见此，顿时面色一变，他们太过于专注苏莫和霸云公子的战斗，对于古魔太子倒是没有太过留意。
<br/>　　只见滚滚魔气淹没苏莫之后，瞬间遁入了崩溃的虚空深处，消失无踪，速度快到极致。
<br/>　　而这个时候，苏莫最后的一击，也击中了霸云公子，直接向对方击到了九霄云外，飞出了阵法气罩的笼罩外围，被彻底的淘汰了。
<br/>　　“苏莫！”龙腾大惊失色，身形一动，便向虚空深处追了过去。
<br/>　　但是，进入虚空深处，哪里还有古魔太子和苏莫的踪迹。
<br/>　　虚空深处，没有方向，没有源气，而且很容易迷失在其中。
<br/>　　“这下糟了！”龙腾的脸色凝重无比，不过，想到苏莫的实力，他稍稍安心，古魔太子，也未必能奈何的了苏莫。
<br/>　　不过，他还是要追下去，以防万一。
<br/>　　他的神念，散热而出，在虚空中查探到了古魔太子的气息，然后急速追了过去。
<br/>　　叶凌天和公孙长绝等人，并未去追逐古魔太子，有龙腾去追就够了，他们要趁着第五城和第八城群龙无首之际，击败两城之人。
<br/>　　“击败他们！”
<br/>　　叶凌天一指第五城和第八城之人，朗声大喝，声震八方。
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　众人纷纷暴喝，浩浩荡荡的向第五城和第八城之人杀了过去，他们都知晓，现在霸云公子和古魔太子不在，正是出手的最好时机。
<br/>　　“杀！”
<br/>　　第五城和第八城之人，也没有退避，立刻开始迎战，因为他们若是退入城中，就是被个个击破的下场，倾力一战或许还有机会。
<br/>　　随后，大战彻底的爆发了，第五城和第八城之人，因为没有霸云公子和古魔太子，再加上人数较少，完全不是第一城、第二城和第三城之人的对手，被杀的节节败退。
<br/>　　不过，第六城之人，马上便出来支援了。
<br/>　　第六城，便是楚天墟之前所在的城池，此城之中并没有公子级别的天才，加之上次折损了很多人手，算是最弱的一城了。
<br/>　　有第六城之人加入，第五城和八城之人，才勉强能挡住叶凌天等人。
<br/>　　不过，落败也是迟早的事情，毕竟，他们没有太强的天才。
<br/>　　而叶凌天、阎魔鬼圣、公孙长绝等人，均是公子级人物，战力超级强大，手下几乎没有一合之敌。
<br/>　　……
<br/>　　神昆小世界之外，帝一魂的眸中，露出了一丝若有若无的笑意。
<br/>　　这一次，古魔太子应该能生擒苏莫了吧！
<br/>　　古魔太子的实力，不是霸云公子所能比拟的，在做足了准备之下，出其不意攻其不备，擒拿苏莫的可能性很大。
<br/>　　只要古魔太子擒拿了苏莫，那一切就稳妥了，至于龙昌和龙族之人，自然能轻易打发。
<br/>　　现在，就看古魔太子的本事了。
<br/>　　帝一魂心中，稍稍有些激动，因为，他有一种直觉，自己只要融合了苏莫之魂，必然会立刻突破境界，跨入高级神王之境。
<br/>　　而且，他的天赋，也会进一步提升。
<br/>　　到那个时候，整个神域，将没有人是他的对手。
<br/>　　另一边，龙昌和一众龙族之人，个个面色凝重，不过，大多数人并不知到古魔太子的意图，他们只是认为，古魔太子偷袭苏莫，是想要将苏莫踢出局。
<br/>　　只有龙昌等少数几人，心中隐隐有了猜测，猜出了古魔太子的真实意图。
<br/>　　而此刻，人王宫的一群人，则没有龙族之人这么淡然了。
<br/>　　人德宫主的面色，极为难看，他也是猜出了古魔太子的意图，毕竟，无论是贺兰殇，还是霸云公子，亦是古魔太子，貌似都有些不太正常。
<br/>　　“父亲！苏莫不会有事吧？”姬月沁俏脸上挂满了担忧，向身边的玄天神王问道。
<br/>　　玄天神王默不作声，只是微微的摇了摇头。
<br/>　　他自己也不清楚，苏莫会不会有事，虽然知道苏莫逆天，无比的逆天，但是，古魔太子也不是一般的天骄。
<br/>　　在这近千名天骄之中，古魔太子的实力，绝对是傲视群雄的存在，也就叶凌天能勉强抗衡一二。
<br/>　　当然，还有一人，就是混沌神朝的混天陵，但是此人比较低调，目前还在第九城之中没有出手，具体实力不得而知。
<br/>　　之前，古魔太子曾和叶凌天交手，看似只是略胜叶凌天一分，但是，估计并没有用出全力。
<br/>　　此刻，亘古魔国之人，倒是面色平静，大国师魔长烽背负着双手，静静的观看者神昆小世界中的战斗，好似对于古魔太子和苏莫之事，毫不关心一般。
<br/>　　……
<br/>　　虚空深处。
<br/>　　一团浩大的魔雾，在飞速的穿梭，速度如风似电。
<br/>　　这团魔雾漆黑无比，充满了森然的气息，其内更是传出阵阵轰鸣。
<br/>　　“苏莫，你别想逃出我的亘古魔域！”古魔太子的声音响起，声音中充满了冷漠。
<br/>　　魔雾之中，苏莫长身而立，眼前漆黑一片，唯有他手中终极神剑爆发出璀璨的剑光。
<br/>　　“亘古魔域？”苏莫低声自喃，他之前一时大意之下，被对方困在了这魔雾之中。

<br/>　　此亘古魔域，一般情况之下，只有神王级别的古魔才能施展，而他天赋异禀，本身修为只有八品，但也勉强能施展而出了。
<br/>　　“你就这么自信？”苏莫冷着脸道，言语之间，他在仔细的观察这亘古魔域。
<br/>　　此地的魔威极为浓郁，充满了浩大的魔性，但是，貌似只有困人之效。
<br/>　　“你就好好待着吧，待我甩开龙腾，再将你炼化！”古魔太子冷笑一声，并未与苏莫闲聊，因为后方正有一个尾巴在跟随，这是龙腾在追逐他，他要尽快将其甩开，不然外面没有他的统领，第八城很容易被彻底击败。
