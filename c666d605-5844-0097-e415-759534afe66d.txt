　　“看来要先回大罗天星域了。”
<br/>　　楚风眠眼神冰冷无比，他的几大敌人，九域的七大宗门，大罗天星域之中的三大势力，还有圣龙秘境之中的那些老古董，以及魔族的少帝。
<br/>　　魔族的少帝，得到了血邪魔帝骸骨，现在隐藏是已经躲藏在了遥远的魔域之中。
<br/>　　纵然是楚风眠的境界踏入了九劫古帝，实力保障，他也不敢贸然深入到魔域之中去。
<br/>　　据说最为古老的魔域中，可能是有着天人境的魔头，哪怕是天人境强者，都不敢深入其中。
<br/>　　更何况少帝得到了血邪魔帝骸骨，一旦是炼化，他的实力也会暴涨，楚风眠一时半会也难以对付他。
<br/>　　而剩下的两方。
<br/>　　九域之中的七大宗门内，可都有着隐藏极深的秘密。
<br/>　　从荒天王还在人世来看，只怕七大宗门之中现在还存活于世的老古董，比起楚风眠想象中的还要多。
<br/>　　楚风眠现在去攻打七大宗门，他还没有这个底气。
<br/>　　更何况是在七大宗门的背后，还有一个神秘的天九域。
<br/>　　楚风眠可没有忘记剑凌的提醒。
<br/>　　以当年七大宗门的实力，根本无法毁灭剑道门，围杀剑道之主，而是神秘的天九域出手，才做到了这一切。
<br/>　　楚风眠在没有足够的实力之前，也不敢去打草惊蛇。
<br/>　　毕竟曾经的剑道之主，可是仙人九阶的巅峰，一尊仙帝，就这样的陨落了，楚风眠可不敢小看天九域。
<br/>　　剩下的，最好对付的，便是大罗天星域的四大势力了。
<br/>　　这四大势力，远远没有七大宗门的底蕴，对他们内部隐藏的实力，楚风眠已经是无比清晰了。
<br/>　　这四大势力，无数次的对付楚风眠，楚风眠可没有忘记。
<br/>　　楚风眠一直以来隐忍，不是不报时候未到，现在这时候终于到了。
<br/>　　楚风眠身形一动，刹那间消失在了云羽天宫之中。
<br/>　　在剑道门中心大殿之内，寒月漓，洛汐，青梦，燕皇等人，还在商量着许多事情，来恢复剑道门的运转。
<br/>　　对于管理宗门，楚风眠一窍不通，便是交给他们来做，主要是燕皇。
<br/>　　燕皇曾经也是燕州大陆之主，有着管理一方势力的经验，他做起这些事来，比起楚风眠要出色的多。
<br/>　　剑道门在他的管理之下，也在不断的恢复这力量，无数剑道门弟子，经历这接连的生死之战，如今更是对力量无比渴望。
<br/>　　在疯狂修炼，提升这实力，整个剑道门，蒸蒸日上。
<br/>　　在众人议论之时。
<br/>　　突然一道身影，撕裂空间走了出来。
<br/>　　无数目光，便是看了过来。
<br/>　　“门主，你出关了。”
<br/>　　“刚刚那一道手臂是？”
<br/>　　看到楚风眠现身，寒月漓急忙询问道。
<br/>　　刚刚那一道荒天王出手，撕裂剑道门，一时间如同天崩地裂一样。
<br/>　　幸好是楚风眠借助剑源界的力量，才将荒天王逼退，但是还是让人有些胆战心惊。
<br/>　　“那人是荒天王，万年前荒古门的宗主，一身实力已经达到了半步天人境的巅峰，不好对付，不过我已经是击伤了他，还得到了他一道化身的力量。”
<br/>　　楚风眠平静的开口道。
<br/>　　“击伤了他？”
<br/>　　听到楚风眠的话，大殿之内的众人都是倒吸了一口凉气，不可思议的看向楚风眠。
<br/>　　刚刚那一道手臂突然撕裂开剑道门，爆发出强横到极点的力量来，那种力量，哪怕是身为九劫古帝巅峰的青梦，都无法反抗。
<br/>　　这样的强者，居然是被楚风眠击伤了？
<br/>　　无数人的心中都是无比震撼，楚风眠的实力，如今到底是强横到了什么地步了？
<br/>　　自从楚风眠踏入了九劫古帝之后，楚风眠的实力便是远远超越了所有人。
<br/>　　之前青梦还可以看出楚风眠的实力来，但是现在青梦在看向楚风眠，便是如同一团迷雾一样，根本无法看出楚风眠的一分一毫。
<br/>　　“借助了这剑源界的力量罢了，不过在外，我也应该有实力可以逼退他。”
<br/>　　楚风眠再度开口道。
<br/>　　现在随着楚风眠神树吞噬了无数的至宝，楚风眠实力也提升了不少，这一次哪怕是在本源界之外，碰到那荒天王。
<br/>　　楚风眠杀不了他，逼退他，应该也没有问题。
<br/>　　不过荒天王展现出来的实力，只是冰山一角，楚风眠也只是在猜测推断罢了，真正结果，必须要交手之后才知道。
<br/>　　“本以为九域已经没落，没想到九域中，居然卧虎藏龙。”
<br/>　　青梦不由的喃喃自语道。
<br/>　　她看到剑道门的实力，本是对九域的武者，不以为然，绝对早已经没落。
<br/>　　可现在看来，九域的确没落了，可是真正的强者，却是要比起大罗天星域还要多得多。
<br/>　　单单是这一个荒天王，就无比恐怖了。
<br/>　　从大罗天星域之中，都少有能够跟荒天王抗衡的人物了。
<br/>　　“九域没落，只是在这万年之间，但是还有一些万年前的老古董，留了下来，不过比起万年前，强者却是少得多了。”
<br/>　　楚风眠开口道。
<br/>　　万年之前的九域，可是还有着天人境的强者了。
<br/>　　剑道之主先不谈，就连楚风眠的大师兄剑凌，其实都是仙人九阶的第二阶，仙士境界的强者。
<br/>　　万年之前的大战中，大部分九域强者，都已经陨落了，活下来的只是小猫两三只。
<br/>　　“看来在九域中建立宗门，的确是无比危险了，七大宗门中，有着如此强横的人物，只怕是下一次攻打，我们就难以应对了。”
<br/>　　洛汐也开口道。
<br/>　　接连两次攻打，一次攻打比起一次攻打的要猛烈，七大宗门的实力，还远远没有展现出全力来。
<br/>　　现在剑道门之中，真正可以跟半步天人境抗衡的强者，只有楚风眠了，其他人根本没有办法抗衡。
<br/>　　像是如果没有楚风眠出手，那荒天王一个人，都可以横扫了现在的剑道门。
<br/>　　当然这是剑源界不出手的结果，剑源界出手，逼退一两位半步天人境强者，还不成问题。
<br/>　　“短时间内，七大宗门应该不会来攻打了，现在九域混乱，七大宗门也不敢伤筋动骨来攻打剑道门了。”
<br/>　　楚风眠沉稳的开口道。
