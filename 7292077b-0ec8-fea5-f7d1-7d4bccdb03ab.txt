崇文变得小心翼翼的，深怕自家少主把他当成出气包，“少主，你真的不打算去隔壁房间看一看，徐少已经进去了，我看都有半个小时了，万一他们真的…”
<br/>
<br/>        这时陆寒霆突然侧了一个身，那双幽沉危险的狭眸无比凌厉的扫了崇文一眼，“最近你话很多。”
<br/>
<br/>        “…”崇文迅速闭嘴，不，我是哑巴！
<br/>
<br/>        陆寒霆又抽了一口烟，然后将嘴里的一口烟雾幽幽的吐了出来，尼古丁的味道并不能麻痹他心里的阴郁烦躁，他满脑子都是夏夕绾那张巴掌大的绝色小脸，她现在跟徐少南在房间里干什么？
<br/>
<br/>        将指尖的烟蒂掐灭在了烟灰缸里，陆寒霆转身就出去了。
<br/>
<br/>        崇文迅速跟上去，小声道，“少主，这件小事就交给尚武去办吧，我让尚武将夏小姐偷过来，然后洗香香送到你的被子里。”
<br/>
<br/>        陆寒霆脚步顿了一下，薄唇里溢出一个冰冷的字眼，“滚！”
<br/>
<br/>        “…”少主你就端着吧，总有一天你会让尚武这么干的！
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆出了房间门，来到了隔壁。
<br/>
<br/>        她就在隔壁房间，两个人一墙之隔，如果她发出点动静，反抗挣扎的话，他会第一时间进去把她救出来的。
<br/>
<br/>        但是他在房间里等了等，她那里一点动静都没有。
<br/>
<br/>        他看得出来，她跟徐少南是旧识，但是他不知道她和徐少南是什么关系。
<br/>
<br/>        陆寒霆来到了隔壁房间，他站在门口，迅速听到里面发出了不正常的声响。
<br/>
<br/>        里面的大床在“吱呀吱呀”的响，夏夕绾清丽的嗓音已经变得娇媚柔糯，“徐少，你慢点…你将我弄疼了…”
<br/>
<br/>        很快，徐少南粗重的喘息声也传来了，“丑丫头，别乱动…绾绾…”
<br/>
<br/>        陆寒霆高大的身躯倏然一僵，狭长的眼角瞬间覆上了一层可怕的猩红，他拽着拳，将指关节拽着咯噔作响。
<br/>
<br/>        她跟徐少南在里面…
<br/>
<br/>        她竟然敢！
<br/>
<br/>        陆寒霆当即搭上了门把，想要一脚踹开门，但是英俊的眼睑快速的敛了下来，他克制住了胸膛狰狞的喘动，不让自己闹出什么动静来。
<br/>
<br/>        他不是一直知道她是怎样的女孩吗，他怎么还对她抱有幻想？
<br/>
<br/>        陆寒霆额角的青筋都在暴跳，他恨她，恨她的自甘下贱，跟什么样的男人都可以上.床。
<br/>
<br/>        但他更恨自己，因为他又将自己送上门让她羞辱让她践踏，现在站在这里听着她的叫.床声，他才真的死心！
<br/>
<br/>        陆寒霆不愿意将自己变成一个“泼妇”，现在安静的离开，是他留给自己最大的骄傲和尊严。
<br/>
<br/>        以后，他再也不会被她骗了！
<br/>
<br/>        陆寒霆转身离开。
<br/>
<br/>        ……
<br/>
<br/>        豪华房间里，徐少南一个人躺在床上，他面颊潮红，已经中了夏夕绾调制的迷情香。
<br/>
<br/>        夏夕绾坐在床边，用脚踩着床，让床发出那种令人面红心跳的吱呀声，为了以假乱真，她还假装叫了几声。
<br/>
<br/>        很快床上的徐少南低低哼了一声，睡着了。
<br/>
<br/>        终于将这个人给应付了，夏夕绾快速的起身，离开了这里。
