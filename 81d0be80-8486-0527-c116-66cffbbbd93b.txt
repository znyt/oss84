　　　　身为猫精，身为从小看着陈高阳长大的海棠，知道自己不告而别的消失，对他是怎样的打击。
<br/>　　好险，真的好险，要是自己晚来了一步，他们就天人永隔了。
<br/>　　纵然是活了一百六十二岁的猫精，此时也满是心有余悸，如临深渊噩梦。
<br/>　　“高阳哥哥，海棠在也不会离开你了，一步都不会离开，你原谅我好吗”
<br/>　　她像是在恳求，像是在哀嚎，楚楚动人。
<br/>　　陈高阳激动不已，也紧紧的抱住海棠：“真的吗，你真的不会再离开了吗”
<br/>　　这是他记忆中，唯一一次这样抱着海棠，抱着跟他相恋多年的女友。
<br/>　　说来可笑，这真的是第一次。
<br/>　　她的娇躯很柔，但却很冰冷，陈高阳暗暗自责，一定是自己刚才吓坏了她。
<br/>　　吓得她全身没有一点温度。
<br/>　　就这样，一男一女，立于地井旁，紧紧的拥抱，谁也不舍得放手。
<br/>　　像是一松手，对方就会瞬间消失。
<br/>　　涂小安远远的看着，这现实版的人妖恋到真如聊斋里面的爱情故事一样，感天动地。
<br/>　　记得小时候，涂小安第一次看聊斋故事，主人公聂小倩是一个美貌女鬼，生前只活到十八岁，死后葬在城北的荒凉古寺旁，不幸被妖怪夜叉胁迫害人。
<br/>　　后书生宁采臣暂居寺院，小倩受妖怪指使，前来谋害，却被采臣的正气打动，便以实相告，助采臣转危为安。
<br/>　　宁采臣也不负小倩重托，助她逃脱魔爪，并收留她侍奉母亲和久病的妻子，小倩勤劳善良，深受宁家人喜爱。
<br/>　　随后小倩因长期接触活人，逐渐犹如常人，宁家人亦绝口不提她的身份，宁妻病逝后，小倩嫁给宁采臣做鬼妻，之后，又指点采臣除掉前来报复的金华妖怪。
<br/>　　几年后，宁采臣考中进士，小倩也生下一子，长大后也成了有名望的人。
<br/>　　宁采臣跟小倩算是一个完美的人鬼恋故事。
<br/>　　涂小安现在也由衷的希望眼前这对人妖恋，也能有一个好的结局。
<br/>　　海棠还没开通灵智，就被陈家收养，从此生活在宅院中，上百年守护陈家人。
<br/>　　如今爱上陈家后代，到也算一段奇缘。
<br/>　　半晌后，涂小安走过去，笑着打趣：“高阳，我将你的女朋友给你找回来了，你准备怎么谢我”
<br/>　　陈高阳听到这个声音，立刻认了出来：“小安，原来是你帮我找回了海棠，你真是我的救命恩人”
<br/>　　说罢，陈高阳激动了朝着涂小安跪了下来，海棠没有阻止，也由衷的跟着下跪。
<br/>　　怎么又跪？
<br/>　　都是现代人，一个个还动不动就跪。
<br/>　　涂小安心中一阵无语，他都快有古代皇帝的待遇了，人人见他就跪，在白镇，他是蛇王，享受过被几十万人同时下跪。
<br/>　　看着下跪的两人，涂小安意念一动，无形的意识力将两人硬生生的给托了起来。
<br/>　　“高阳，我们是朋友，我可是受不起你的跪，你要真的想感谢我，我到有一事需要你帮忙”
<br/>　　陈高阳连忙道：“只要我能做到的，我一定照办”
<br/>　　“今晚夜已深，我初来温城，还没有找到落脚的地步，恐怕需要你收留收留我”
<br/>　　此话一出，陈高阳跟海棠同时一愣，这算什么大事，摆明了就是想给他们两个人一个台阶下。
<br/>　　别说让他现在收留涂小安住，就是他现在要求陈高阳将整个四合院送与他，陈高阳都不会蹙一下眉头的拒绝。
<br/>　　“海棠，快先帮恩人准备房间，我们到时候在说”
<br/>　　陈高阳激动的说，海棠先扶着陈高阳进屋，然后细心的给涂小安收拾四合院中最好的房间。
<br/>　　如今的海棠不在是一只猫，而是一个人，这些事情自然不在话下。
<br/>　　随后，陈高阳不断的握着涂小安的手，表示天大的感谢，他一个瞎子真的不知道如何回报涂小安。
<br/>　　涂小安其实已经得到了很好的回报，但他跟海棠发生的事情，并不需要跟陈高阳讲。
<br/>　　他只是一个普通人，有些事情说了他也难以相信跟接受。
<br/>　　随便编了一个找回海棠的故事，也算给了陈高阳一个交待。
<br/>　　海棠房间收拾好了，涂小安很知趣，主动的说累了一天想去休息，给这对另类的情侣一个完整的相处空间。
<br/>　　让他们慢慢的腻歪，诉说彼此的相思之苦。
<br/>　　涂小安到很希望这对人妖恋都有一个很好的结局，他思念很开放的。
<br/>　　只要两人真心喜欢，猫精那又怎样，况且现在海棠不在是一只猫，更方便照顾陈高阳。
<br/>　　翌日...
<br/>　　涂小安早早的找了一个出去办事的借口从陈家大院离开，因为他有点受不了这对情侣的热情，简直快将他当神一样供着。
<br/>　　这段时间，涂小安应该都会住在陈家，一是真的没落脚的地方，二是担心白家的人会找过来。
<br/>　　毕竟蛇王石是猫精海棠从白家的手中抢来的。
<br/>　　昨晚实验室，白家再次失手，想必不会善罢甘休。
<br/>　　根据之前蛇博士跟他涂小安说，隐世家族白家，在温城的势力很大，他是无所谓，但是这对不容易的情侣，还是不要被人打扰的好。
<br/>　　这个雷，涂小安他接下来了。
<br/>　　离开陈家大院的时候，涂小安也不知道自己到底要去干嘛，忽然想起来，自己手中还有一张五百万的支票，决定找个银行先兑换，存起来。
<br/>　　这存，自然是要存到老姐涂小月的账户上。
<br/>　　从白镇离开，涂小安都没跟老姐好好的交代，因为化形了，新的身份也没在她面前出现过。
<br/>　　算是不告而别，仅电话联系了。
<br/>　　老姐肯定发狂的在暗暗的生他的气，以前涂小安时常不着家，一出去几天不见踪影。
<br/>　　现在好了，直接玩离家出走，到了别的城市。
<br/>　　涂小安想过，这五百万兑换，自己留一百万就好，其他的全部给老姐打过去，让她消消火。
<br/>　　钱对于涂小安现在来说，真的没什么感觉，他想赚钱也很容易，在说如今，他最重要的事情就是提升实力，尽快的证道，突破到三阶生灵。
<br/>　　到时候就能光明正大的回家，落叶归根。
