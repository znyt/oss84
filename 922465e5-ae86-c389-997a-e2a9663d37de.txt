<br/>    诸位实权亲王在大殿之中坐下，无眠者带着的面具是一个有着一只眼睛睁着，一只眼睛闭着的白色面具，当然，这个面具的意思不是睁只眼闭只眼，而是另外一个意思，星空中的修行者，便是取了不少的外号：独眼的监视者，无休止的巡视者，星空的守望者。
<br/>    
<br/>        这个意思便是永无休止的注视着星空。
<br/>    
<br/>        无眠者到底有多少位，在星空之中一直都是个秘，反正四大仙国和无眠者的摩擦，从来没有占过上风。
<br/>    
<br/>        其中虽然有着四大仙国从来不愿意齐心协力的缘故，但是却不得不承认一点，那就是无眠者和诸位亲王单打独斗，依旧是压着打！
<br/>    
<br/>        在陈明的眼中，其实无冕者议会更像是反抗军，天庭，瑶池，太虚，月宫把持着四棵仅有的英灵树，其他的凡人修行者便永世都没有出头之日，而无冕者议会，则是收纳众多不属于四大仙国的凡人天才的地方，为了能够和拥有英灵树的四大仙国对抗，无冕者议会只能用最为残酷的手段，培养出更强大的强者。
<br/>    
<br/>        光明被把持，所以无冕者议会只能信仰黑暗，同时成为这世间最大的黑暗。
<br/>    
<br/>        光明已经没有出路，唯有献身黑暗。
<br/>    
<br/>        陈明盘坐在一张竹席之上，在前方的小桌上，香炉之中升起袅袅云烟，宁静致远。
<br/>    
<br/>        至于这阁楼外面，已经是乱做了一团，众多前来看热闹的修行者不计其数，“那里面的就是燃罪上师吗？”
<br/>    
<br/>        “是顶楼的那个小白脸和尚吗？”
<br/>    
<br/>        “应该就是那个小白脸和尚！”
<br/>    
<br/>        “看起来年纪不大啊。”
<br/>    
<br/>        “额，这种未卜先知的修行者其实也是修行者，寿元和仙一样的，说不定，这就是个活了几千年的老怪物了！”
<br/>    
<br/>        “虽说燃罪上师佛法精深，但是未免太过猖狂了一点吧？”
<br/>    
<br/>        “你懂什么，这叫先声夺人！”
<br/>    
<br/>        外面吵吵嚷嚷，陈明却是懒得在意，很是悠闲地喝茶，人群之中让开一条路来，一辆黑色的马车缓缓行来，马车在阁楼前停下，一个手中杵着拐杖的老人下车，抬头看向陈明，只隐约看见陈
<br/>    
<br/>        明的模样，是个光头的年轻和尚。
<br/>    
<br/>        占天养拄着拐杖进去，无眠者当即起身，朝着占天养微微弯腰，占天养只是淡淡地笑了笑，并未还礼。
<br/>    
<br/>        周围的亲王们看见无眠者都这么尊敬占天养，心中不由得一惊，这个占天养，在无冕者议会身份这么高吗？
<br/>    
<br/>        只怕燃罪上师，也可能不是对手啊！
<br/>    
<br/>        占天养在众人的目光之中来到陈明的对面，盘腿坐下，“阁下，便是燃罪上师？”
<br/>    
<br/>        陈明微微点头，要是以他以前的脾气，估摸着上来就是一句，你这不是废话吗？“小僧恭候多时。”
<br/>    
<br/>        占天养微微一笑，如同树皮开裂，“阁下倒是好生年轻。”
<br/>    
<br/>        陈明道：“赶紧比，比完还得吃饭呢。”
<br/>    
<br/>        占天养的眉头微皱，没试探出啥来啊。
<br/>    
<br/>        占天养问道：“不知道，上师准备与我比试几局？我推荐一局，一局定输赢，总比什么五局三胜三局定了胜负要好看一些。”
<br/>    
<br/>        陈明明白占天养的意思，占天养就是说五局三胜制，占天养会直接赢三局，而自己三局全败。
<br/>    
<br/>        陈明点头道：“确实，这样，你也不至于身败名裂。”
<br/>    
<br/>        占天养道：“竟然要比的是卜算之道，不知燃罪上师想要比什么？”
<br/>    
<br/>        陈明伸手，点在茶碗之中，然后在桌面上落笔，写下四个字：星龙太子。
<br/>    
<br/>        占天养的目光一闪，这一次，遇到狠角色了，这个世上，卜算一道实际上并不是很难，卜算一道真正难的是最后一关，明明是错的，却要人承认是对的。
<br/>    
<br/>        说来说去，还是双方之间的一比交易，只是之前四大仙国的四位大祭司，提出的交易都足以摆平无冕者议会的百年大胜而已。
<br/>    
<br/>        陈明看向占天养，笑道：“他虽然身份不低，但是还是太过稚嫩，修为不够格，但是为了平息这一次的百年之战的结果，应该是够了吧。”
<br/>    
<br/>        占天养微微点头，然后道：“够了，绰绰有余。”
<br/>    
<br/>        身为无冕者议会的祭祀，而且身份不低，星龙太子是
<br/>    
<br/>        一个无面者，占天养自然是知道的。占天养甚至还暗自觉得这星龙太子着实聪明，若不是有着无面者的身份，无冕者议会可不会那么容易放任星龙太子在世间游走。
<br/>    
<br/>        无冕者议会对于星龙太子是一个无面者当做是星龙太子的表态，那就是他们是盟友。
<br/>    
<br/>        陈明笑道：“那么，我来算，你来确定对与不对？”
<br/>    
<br/>        占天养微微点头，即便是他，也无法算出星龙太子的位置，星龙太子，好像命格并不在星空之中，或者说，根本没有命星。
<br/>    
<br/>        陈明伸手在茶水中点了点，然后在桌面上落笔写下一句话：星龙太子不在南门观，万龙岭，万龙碑前。
<br/>    
<br/>        占天养有些吃惊的看了这一句话一眼，对方居然能够具体到这种程度。
<br/>    
<br/>        占天养当即看向场中款款而谈地九变蚕王一眼，叫道：“九变蚕王，还请上来一趟。”
<br/>    
<br/>        九变蚕王看得出来，即便是无眠者都对这占天养尊敬有加，也没有摆什么架子，直接上来，问道：“不知祭祀寻我何事？”
<br/>    
<br/>        占天养点了点桌面上陈明写的那句话，九变蚕王只是看了一眼，便是惊讶地后退了两步。
<br/>    
<br/>        占天养看到九变蚕王这般模样，已经猜到了七八分，这燃罪上师，绝对是算到了！
<br/>    
<br/>        占天养道：“九变蚕王，下楼吧。”
<br/>    
<br/>        九变蚕王也不说话，只是深深地朝着占天养和陈明拱手行了一礼。
<br/>    
<br/>        以前九变蚕王还觉得这些大祭司只是吃干饭的，现在看到这一场比试，才觉得后背发凉，这未免，也太可怕了一点，对方根本没有去过妖族南门观，但是却是知道的一清二楚！
<br/>    
<br/>        也不知道上面现在到底是什么情况。
<br/>    
<br/>        占天养看向陈明，问道：“燃罪上师，可否信守承诺？”
<br/>    
<br/>        占天养话里的意思很简单，那就是陈明既然拿出了这个筹码，必定不会说出来。星龙太子，在无冕者议会的算盘上，在他占天养的棋盘上，乃是至关重要的一颗棋子，此刻绝对不容有失。
<br/>    
<br/>        陈明笑道：“天知地知，你知我知。”
