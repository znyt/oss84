　　“大哥！”
<br/>　　“二弟！”
<br/>　　“大哥！”
<br/>　　“……”
<br/>　　两人本是没有什么交情，一番叫唤之后，两人的感情瞬间便剧增，达到交心的地步。
<br/>　　“二弟，为兄有一项光荣而艰巨的人物要交于你，希望你不要拒绝！”突然，伏羲语重心长地对着神农说道。
<br/>　　“什么事，大哥尽管吩咐，小弟自当竭尽所能，哪怕赴汤蹈火，也再所不辞！”神农大气道。
<br/>　　“那就好！”伏羲拍了怕神农的肩膀，有些如释重负道。
<br/>　　为什么我有点不好的预感呢？神农看着伏羲这幅表情，脑袋有点转不过来，他就是一个农夫，哪里懂的当政者的心思？
<br/>　　就这样，在伏羲的忽悠下，啊不，在伏羲的仔细劝导下，神农迷迷糊糊的答应了继任伏羲的人族共主一事。
<br/>　　……
<br/>　　十年后。
<br/>　　五百年之期圆满，人族共主伏羲准备禅让，证道人道天皇！
<br/>　　在这十年时间内，神农氏也逐渐熟悉了人族共主的工作。身为人族共主，自是需要做许多事情，简单来说，便是要多做于人族有利之事。
<br/>　　人族众多部落，对于伏羲要传位于神农之事，除了几个有野心的阴谋家之外，大多并没有什么反对的声音，毕竟神农氏的名字早已在人族传开了，他在人族的声望也不小。仅有的几个反对的声音，自然是无效的。
<br/>　　这日，便是人族伏羲禅位于神农的日子，整个人族都忙来起来，众多部落的首领都前来观礼。人族高层出动，事关人族三皇五帝之大事，关乎人族气运，此事由不得他们马虎。
<br/>　　禅位之地在泰山，当初伏羲封禅便是在泰山，定都在陈地，如今传位，在泰山进行，也是应当的。
<br/>　　午时三刻，伏羲拉着神农走上早已搭建好的祭台上，焚香祷告，念祭文，祈苍天，拜先贤，礼四方。
<br/>　　经过一段繁琐的礼仪之后，伏羲面色严肃，看向一众人族，大声喊道：
<br/>　　“吾伏羲，为人族共主五百年之期！衍创八卦，开启人族文化之源；作网渔猎，驯养家畜，提高人族生产力；变革婚姻，改血缘婚为族外婚，结束原始群婚状态；制礼乐，创歌谣，将音乐带入生活……
<br/>　　吾在位期间，一直殚精竭虑，为人族奉献一切，奈何，近来吾倍感力不从心，自揣能力有限，为人族能够有更好的发展，遂，决定禅让人族共主之位！
<br/>　　姜氏部落，姜姓烈山氏神农，年少有为，今已是转轮皇者，更有大德，播种五谷，解决人族温饱之难，贤明有德，可堪大任。故，吾决定，禅位于他！
<br/>　　浩浩上苍，冥冥大地，今吾伏羲禅位人族共主于神农，人神共鉴之！”
<br/>　　伏羲的声音宏大，嘹亮雄浑，更带着一种坚定，恢宏之音，传遍三界。
<br/>　　这一刹那，苍穹色变，大地轰鸣，无穷无尽的人道信念从四面八方席卷而来，无尽人族心中默念伏羲大帝之名，歌颂他的功德，人道气运汇聚而来，形成一道狂风，汇聚在穹顶之上。
<br/>　　“轰隆隆！”
<br/>　　这一刻，天地巨震，一道道浩大的声音，击在伏羲的心中，像是在对伏羲的内心进行拷问。
<br/>　　这是天皇证道必须要经历的过程，谁也不能帮助他。若是成功经受过拷问，那天地便会承认他的人道大帝天皇之位，若是不能通过，那便会被强大的天地之力反噬而死。
<br/>　　道音连响九道，在别人耳边，却是不能听出什么，但击在伏羲心中，却是每一道都掀起无边巨浪，击在他的心灵最深处。他这些年对人族的一举一动，都会被调了出来，对他进行拷问。
<br/>　　九音过后，伏羲依旧傲然挺立。
<br/>　　这时，天地似乎对他的考验已经完成了，似是承认了伏羲之言，所有的人道信念汇聚而起，顿时，滚滚功德金光，凝聚而出，无边金光，照亮了天地，璀璨无比，堪比太阳！
<br/>　　但那无边功德一直在九天之上轰响，却并不降落，就仿佛是，伏羲的考验，还没有完成一样。
<br/>　　“怎么回事？难道是考验还没有完成，还是伏羲做的还不够？”祭台不远处，伏羲之前的老师龟灵圣母抬头看着天上的功德，有些疑惑道。
<br/>　　就在三界诸神疑惑不解之时，忽然九天之上，仙音袅袅，紫气纵横，瑞气千条，霞光万丈，祥云万朵，金花乱坠，像是在恭迎什么。
<br/>　　却是洪荒诸圣来了。
<br/>　　人们并不知道，人道天皇证道，需要经过三重考验，第一重，自然便是人道考验，需要人族九成以上的同意了，他才有资格证道天皇。以伏羲对人族所做之事，人族无数信念的汇聚，这一道考验自然算是过了。
<br/>　　第二道考验是天地考验，便是之前的九道天音，专门为人内心设置的考验，承受住了，心境大增，问心无愧，自然也就通过了考验。
<br/>　　第三道考验，便是诸圣的考验，欲证道天皇，需要经过洪荒诸圣的同意才行。圣人执天之道，代天行事，圣人的同意，便是天道的同意。
<br/>　　三道考验，缺一不可！
<br/>　　“天皇伏羲，当证道否？”
<br/>　　九天之上，元始身为诸圣之首，天罚圣主，自然第一个出声问道。
<br/>　　闻言，太上道主、通天剑主、接引、准提圣人、女娲娘娘，俱皆纷纷点头称善，算是同意了伏羲证道天皇。
<br/>　　元始见后，微微点头，对此丝毫没有感到意外。女娲为伏羲的哥哥，自是最希望伏羲证道，太上欠女娲一份因果，也早已同意了伏羲的天皇之位；通天为伏羲的师祖，也算是同意了，至于西方二圣接引与准提，就算他们不同意也不行。
<br/>　　在知道各自的想法后，元始当即淡淡出声道：
<br/>　　“人族伏羲，当为天皇！
<br/>　　证道之后，当去火云洞，自此以后，镇压人族气运，非无量量劫至，不得外出！”
<br/>　　“轰隆隆！”
<br/>　　在得到诸位圣人的同意后，天上的无尽功德顿时不再迟疑，仿佛是找到了主人一样，迅速朝着伏羲轰然落下。
<br/>　　功德一分为二，九份落入伏羲头顶，一份没入龟灵圣母体内，功德入体，瞬间化为修为，龟灵圣母立刻便从大罗中期提升到大罗圆满。这一幕让九天上的通天教主满意的点了点头。
<br/>　　功德入体，伏羲蓦然仰天大吼，全身气势大盛，散发着无尽玄奥的巨大八卦图透体而出，遮蔽了他头顶的一片虚空，缓缓旋转，道法自生，无穷无尽。
<br/>　　“极道帝境，武道转轮，八卦轮，琴道轮，七绝轮，给我合！”伏羲脸色坚定，一声大吼。
<br/>　　他体内顿时飞出三道神轮，悬浮于他的脑后，缓缓汇聚在一起，想要合而为一。前世今生，诸多积累，如今更兼具无尽人道气运，天道功德，还有对于八卦的领悟，趁着证道天皇这一个契机，他在冲击极道帝境。
<br/>　　“轰！”
<br/>　　三道神轮在相撞，碰撞出无尽火花，道与道的碰撞，交织出无尽玄奥，道音自生，仙乐响起。
<br/>　　……
