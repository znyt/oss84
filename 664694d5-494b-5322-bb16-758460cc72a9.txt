　　叶尘自然不知道这些人的想法，他虽然灭了血天海，自己也受了不轻的伤，因而第一时间回到了灵溪所在的院落之中。
<br/>　　随手在周围布下了十几道禁制后，叶尘本想服用一些疗伤丹药，却发现丹药早已经被自己吃了精光。
<br/>　　自从他进入修真界以来，频频受伤，之前在天玄星炼制的海量丹药，早已经被他吃的干干净净。
<br/>　　以他如今的状态，当场炼制丹药显然已经来不及了，只好调动体内的万古长青珠，动用生命之力，来修复体内的伤势。
<br/>　　而就在这时，奥维斯的声音，突然在叶尘脑海之中响起，
<br/>　　“主人，我已经将噬血妖珠同化，不如我传一些血煞之力给你，血煞之力配合上您的吞天神功，对疗伤有奇效，还可以增进修为！”
<br/>　　叶尘顿时大喜，当即收起万古长青珠，转而催动吞天鼎，
<br/>　　“好！我来试试！”
<br/>　　呼呼呼！
<br/>　　一道道精纯的血煞之力，自吞天鼎中狂涌而出，冲入叶尘的经脉之中，
<br/>　　“啊！！”
<br/>　　以叶尘如今的身体强度，竟然都忍不住发出一声惨叫。
<br/>　　“好强的血煞之力！”
<br/>　　叶尘不由得感叹了一句，当即催动吞天神功，开始将那血煞之力缓缓炼化。
<br/>　　滋滋滋~
<br/>　　一转眼的功夫，叶尘体内损伤的经脉，竟然就已经恢复了七七八八！
<br/>　　居然比万古长青珠的疗伤效果，还有恐怖的多！
<br/>　　当然，也并不意味着，万古长青珠不如噬血妖珠。
<br/>　　毕竟，这吞天鼎，是吞天魔帝为自己量身打造的绝世神器，和吞天神功自然无比契合。
<br/>　　相比之下，万古长青珠则是润物细无声，能够在潜移默化中，改善宿主的身体乃至灵魂！
<br/>　　这两件法宝，一正一邪，如今却集于一人之身，也算是一种难得的缘分。
<br/>　　短短两三个小时，叶尘体内的伤势已经好了大半。
<br/>　　他并没有一鼓作气，继续炼化血煞之力，因为他发现，噬血妖珠的血煞之力，配合吞天神功，虽然有疗伤奇效，可如果吞噬的太多，似乎会影响心志，让人杀心大起！
<br/>　　“如今我的状态，自保应该绰绰有余了，可以用万古长青珠慢慢调养，倒也不急于一时……”
<br/>　　一念及此，叶尘收起吞天鼎，然后缓缓睁开双眼。
<br/>　　“叶尘哥哥，你终于醒啦！”
<br/>　　伴随着清脆动听的声音响起，一道娇小可爱的少女身影，冲到叶尘的跟前，还夹带着一股淡淡的清香。
<br/>　　却是灵溪小丫头，不知道何时已经醒来，看到叶尘坐在院子里修炼，也不敢打扰，已经在旁边等候多时。
<br/>　　叶尘冲小丫头微微一笑，在她身上打量了一番，点了点头道：
<br/>　　“不错不错！丫头，你如今全身经脉已经贯通，很快就能开始修炼了！”
<br/>　　小丫头顿时兴奋不已，
<br/>　　“真的吗？叶尘哥哥，我真的可以修炼了？那以后，是不是就再也不会再有说我没用了？”
<br/>　　叶尘哈哈一笑，揉了揉小丫头的脑袋，
<br/>　　“那是自然！谁若是敢说我们灵溪没用，看我不撕了他的嘴！”
<br/>　　小丫头感动的不行，眼眶微微一红，直接一头扑进叶尘的怀里，哭了起来，
<br/>　　“叶尘哥哥，你待我真好！呜呜~”
<br/>　　叶尘顿时有些束手无策，哄了好一会，小丫头这才从他的怀里缓缓起开。
<br/>　　叶尘随手将周围的禁制撤去，这才感应到院落之外，灵山、灵韵等人，似乎已经等候多时，于是朗声道：
<br/>　　“都进来吧！”
<br/>　　灵山等人在外面等了几个小时，不见任何动静，又不敢随意乱闯，早就急得不行，听到叶尘的传唤，顿时激动不已，当即便火急火燎的冲了进来。
<br/>　　噗通！噗通！噗通！
<br/>　　灵山带着黑岩城众人，一进入院落之中，当即便跪倒在地，其余众人见状，也纷纷跪倒。
<br/>　　“叶前辈！恳请您救救我黑岩城啊！”
<br/>　　灵山趴在地上，头磕的砰砰直响，连同整个地面都跟着一阵颤动。
<br/>　　叶尘见状，顿时眉头一皱，
<br/>　　他自然清楚灵山这些人的想法，如今他杀了血天海，必然会引起整个巨门星域震动，而血月宗也绝对不会善罢甘休！
<br/>　　但，这对于叶尘来说，并没有太大影响，
<br/>　　他有逍遥仙珠在手，随时都能拍屁股走人，到时候血月宗找不到他这个罪魁祸首，这笔账自然就会算在黑岩城的头上。
<br/>　　血月宗雄霸一域，相比之下，黑岩城根本没有任何抵抗之力！
<br/>　　叶尘沉默了片刻，方才缓缓道：
<br/>　　“那城主大人以为，我应该如何救你们？”
<br/>　　“这个……”
<br/>　　叶尘这话，顿时一下子把灵山给问住了。
<br/>　　他总不能说，让叶尘主动去血月宗说明情况，告诉血月宗的人，此事和他们黑岩城无关吧？
<br/>　　且不说，叶尘愿不愿意这么做，即便是他真的这么做了，血月宗的人会相信吗？
<br/>　　血无影和血天海，全都死在他们黑岩城，这是不争的事实！
<br/>　　以血月宗以往的行事作风，无论此事是否和黑岩城有关，恐怕都会将黑岩城屠城，来维护他们血月宗的威严。
<br/>　　“叶前辈！事到如今，晚辈也不知道该如何是好，但是恳请前辈，看在小女的面子的上，一定要救救我们啊！！”
<br/>　　灵山无奈之下，只好大声哀嚎，同时冲着灵溪频频使着眼色。
<br/>　　小丫头似乎也明白了事情的严重性，只好一脸可怜兮兮的望向叶尘，
<br/>　　“叶尘哥哥……”
<br/>　　叶尘顿时也郁闷不已，
<br/>　　杀了小的，来了老的，杀了老的，还有一个宗门！
<br/>　　难道非得逼他，将整个血月宗灭了不成？
<br/>　　且不说，以他如今的状态，独闯血月宗，根本没有什么胜算，即便他灭了血月宗，黑岩城如今已经惨败不堪，恐怕他离开之后，要不了多久，还是会被其他势力所灭……
<br/>　　弱小，在这个弱肉强食的修真界，本身就是一种错误，即便叶尘能帮得了他们一时，也帮不了他们一世。
<br/>　　就在叶尘一阵纠结之时，跪在灵山旁边的灵韵，忽然开口了，
<br/>　　“为今之计，恐怕我们只有离开巨门星域，才能逃脱血月宗的魔爪！”
