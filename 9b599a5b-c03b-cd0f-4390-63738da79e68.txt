陆寒霆勾起了薄唇，往她脸上亲去，“恩，看来我以后要更加努力一点，一边努力把肾给你，陪你造宝宝，一边还要努力赚钱，你和宝宝都归我养。”
<br/>
<br/>        夏夕绾闭了闭眼，在他低哑磁性的哄宠嗓音里，她双腿都发软了，陆先生真的会要人命啊。
<br/>
<br/>        这时“咳咳咳”的嗓音响起了，有人来了。
<br/>
<br/>        夏夕绾迅速将陆寒霆推开，这一次陆寒霆也顺势松开了她，李文清院长，陆子羡，陆茵茵都来了。
<br/>
<br/>        这三个人都看到了他们亲热的画面了，这声咳嗽是李文清院长发出来的。
<br/>
<br/>        陆子羡又恢复了往日里清冷淡漠的模样，那俊俏的面上并没有表露出什么，而陆茵茵的脸色很不好看，表情都是僵硬的。
<br/>
<br/>        李文清院长完全是吃瓜群众，还呵呵两声，“陆总，我多说一句，绾绾现在还小，正是努力拼事业的时候，孩子可以等等再要，不要太着急。”
<br/>
<br/>        陆寒霆伸手摸了摸夏夕绾的小脑袋，“恩，现在她自己还是孩子，需要人宠着，孩子等两年再做打算。”
<br/>
<br/>        李文清院长为自己跟陆寒霆达成了一致而鼓掌，“绾绾，子羡，我们不要再耽误时间了，厉老夫人那里可是要争分夺秒，我们进去商讨手术方案吧。”
<br/>
<br/>        李文清院长打开了商务套房的门，陆子羡走了进去，夏夕绾小脸红红的看向陆寒霆，“你要回公司吗？”
<br/>
<br/>        陆寒霆低眸问，“要我回去吗？”
<br/>
<br/>        夏夕绾摇头，像个粘人的小女孩，“工作重要还是我重要，你不能让严秘书把文件送到这里来办公吗，我想你陪着我。”
<br/>
<br/>        陆寒霆的眸里宠溺一片，“好。”
<br/>
<br/>        夏夕绾这才满意的挥了挥小手走进房间了。
<br/>
<br/>        房间门关上了，陆寒霆和陆茵茵站在外面，陆寒霆深邃的狭眸淡淡的扫了陆茵茵一眼，然后离开了。
<br/>
<br/>        陆茵茵自己一个人僵在了原地。
<br/>
<br/>        ……
<br/>
<br/>        两个小时后，三个人定下了厉老夫人的手术方案，夏夕绾打开房间门，没有看到陆寒霆的人。
<br/>
<br/>        这时陆茵茵走了过来，“夕绾，你过来，我有一件事想要跟你说。”
<br/>
<br/>        “姑姑，你想要跟我说什么？”夏夕绾问。
<br/>
<br/>        “夕绾，我知道你已经嫁给寒霆了，但是我想你并不了解寒霆，关于寒霆的过去，你一无所知，你并不知道他究竟是怎样的一个人。”
<br/>
<br/>        夏夕绾就想起前不久陆先生在沐浴间里捧着她的脸亲吻，边吻边用一种沙哑的嗓音跟她呢喃绾绾，不管陆茵茵跟你说什么，你都不要听。
<br/>
<br/>        原来陆先生早就猜到陆茵茵这个姑姑会把他最痛苦不堪的过去拿出来说事。
<br/>
<br/>        夏夕绾澄亮的翦瞳落在了陆茵茵的脸上，“所以姑姑，你现在要把陆先生的过去都告诉我？”
<br/>
<br/>        “是的夕绾，我告诉你，寒霆他有病，他…”
<br/>
<br/>        陆茵茵的话戛然而止，因为前方闯入了一道高大英挺的身影，陆寒霆来了。
