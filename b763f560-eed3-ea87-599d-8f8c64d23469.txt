　　张道长道：“少年人有志向是好的，我看你之前作法，应该是有方士牌位，对你这个年纪来说，也算是了不起了，潜心修炼，将来未尝不会有当上天师的一天。”
<br/>　　“我说，你还教训没完了是吧！”叶少阳实在有点不能让忍了，狠狠瞪了张道长一眼，刚要开口，刘老汉上来打圆场，以他长辈的身份，对张道长赔礼道歉，想要把他拉走。
<br/>　　张道长也没有太在意，看了看床上的孩子，对刘四夫妻说道：“我让你们带着孩子去道观作法，为什么没去？幸亏我担心出什么意外，带了弟子前来接你，不然今天让这位后生险些闯下大祸。”
<br/>　　刘四各种赔不是，表示是叶少阳的到来，耽误了时间。
<br/>　　张道长也不想听这些，摆摆手，让他赶紧收拾，随自己去道观。
<br/>　　“少阳，我们走吧……”刘老汉有些尴尬地跟刘四道歉，刘四生气不理，刘老汉只好过来拉叶少阳。
<br/>　　叶少阳让他稍等，自己上前问张道长：“张道长，你打算让他们一家走出千里之外，逃避这几个恶鬼的追杀？”
<br/>　　张道长叹了口气，道：“忏愧，我这也是没有办法。我刚晋升真人不久，以我只修为，对付不了三只恶鬼，也只能出此下策，让他们避过而已。”
<br/>　　叶少阳道：“他们走后，三只恶鬼还在这，将来不是会害别人？”
<br/>　　张道长道：“应该不会，那三只恶鬼纠缠婴儿不放，必有缘由，疑似报仇，应该与他人无害。”
<br/>　　“一个婴儿，怎么会得罪三个恶鬼，就算是追前世的债，也没有这么早，在婴儿时期就要弄死他的。”
<br/>　　“这个，我也想过。”张道长点头承认，“既然与婴儿无关，那一定与他父母或先祖有关，只能是这个缘由。”
<br/>　　这下轮到叶少阳点头，问道：“你没有调查清楚？”
<br/>　　张道长很不喜欢叶少阳用这种方式跟他对话，但本着长辈的姿态，也只能大度一点，回答他的提问。
<br/>　　“我尝试过跟那三个恶鬼通灵，但他们拒绝，刘四夫妇也想不起曾经得罪过什么人，这三个恶鬼为什么要伤人，却是无法得知。”
<br/>　　叶少阳道：“所以你对付不了这三个恶鬼，又想救这孩子，只能让他们搬出千里之外，躲避恶鬼追杀。”
<br/>　　张道长眉头皱了皱，道：“难道还有什么好办法？”
<br/>　　叶少阳道：“你让他们先在道观住一晚，在道观威严正气之下，那三个恶鬼不得不暂时离开婴儿的身体，之后你抹去他身上的印记，让恶鬼找不到，然后再让他们夫妻把孩子带走……听上去是不错，可是你想过没有，这三个恶鬼如果有办法找到婴儿呢？”
<br/>　　张道长一怔，说道：“怎么可能！”
<br/>　　“极有可能！”叶少阳道，“我之前在对付这三个恶鬼的时候，发现这三个恶鬼与婴儿之间，有一种神秘的联系，我相信，就算把婴儿送到千里之外，这几个恶鬼也有办法找过去。”
<br/>　　“你胡说八道什么！”有几个小道士听烦了，出口骂道。
<br/>　　张道长挥手让他不要说话，定定地望着叶少阳，说道：“什么神秘的联系？”
<br/>　　“是魂魄深处的一种联系，难以言说，我猜是魂之羁绊。”
<br/>　　“魂之羁绊……”张道长喃喃重复了一遍，这四个字对他来说有点陌生，但他也听说过。魂之羁绊，是一种无形的力量，将两个人的灵魂连在一起，在人活着的时候，因为魂魄与肉身是一体的，是感觉不到魂之羁绊的，但是人死后，就可以隐约感觉到它的存在。
<br/>　　这是一种冥冥之中的指引，无法用言语来形容，也无法用一般法术察觉到，一般只出现在亲人和深切的爱人之间，是一种灵魂深处的联系，魂之羁绊的力量并不强，在两个不同的灵魂身上，也无法造成任何特殊的现象，仅仅是让鬼魂能够隐约感知到对方的存在。
<br/>　　人间有些不在一起的至亲之人，如果一方遇难丧命，在那个瞬间，活着的那个会突然心慌悸动，或身边出现一些奇异的现象。科学上无法给出解释，只能称之为心灵感应。但是在法术界，却被成为是魂之羁绊。
<br/>　　除了父母子女这样的至亲之间，如果是缘定三生的夫妻或者情侣，彼此之间也会有这种魂之羁绊。真正的一见钟情，就是魂之羁绊在起作用。
<br/>　　但是在于法师来说，只有道门天师、佛门禅师以上等级的法师，才能够感知到魂之羁绊的存在。
<br/>　　“魂之羁绊！”
<br/>　　张道长又重复了一遍，望着叶少阳，嘴巴缓缓张开，说道：“你怎么可能感知到魂之羁绊的存在？”
<br/>　　叶少阳耸了耸肩，“我之前就说了，我是天师，说了你不信。”
<br/>　　“可你的法力，最多只是个方士。”
<br/>　　叶少阳不想跟他解释，因为根本解释不清，转脸对着刘四，说道：“你家里只有这一个儿子，在生他之前，还有没有生过别的孩子，但是早亡没有养大的？”
<br/>　　刘四夫妻听了这话，当场震惊。刘四睁大眼睛盯着叶少阳，失声道：“你、你你怎么知道？”
<br/>　　叶少阳笑了笑，对他来说，事情是明摆着的：
<br/>　　魂之羁绊，除了父母子女、三世情人之间，兄弟姐妹之间，有时候也会存在魂之羁绊。这婴儿才出生不到一年，自然不可能遇到什么三世情人，父母也健在，既然他感知到那三个恶鬼与婴儿之间有魂之羁绊，那只有一种可能：这几个恶鬼是他的兄弟姐妹。不过前提是死去的。
<br/>　　刘四跟妻子看了一眼，然后磕磕巴巴地讲述起来：原来，在这个儿子之前，两人年轻时候也曾生过孩子，一连生过三个，两男一女，结果无一例外都是养到不到一周岁就死了，当时也找先生看过，说是夫妻二人命犯七煞，注定无子，想要养活一个儿子，只有吃斋烧香，多做善事，积累阴德，十年之后或许有子。
<br/>　　（从桂林回来了，明天到家，努力恢复更新。）
