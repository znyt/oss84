　　第八二零章
<br/>　　“郁宁，你大爷的……”一群人类头土脸的从城门口附近的位置上跑出来，一边骂一边齐声大骂。
<br/>　　“郁宁，你大爷的……”弥漫着的烟雾被风吹散之后，骑在马上随时准备出击的席二愣和伍登亦是破口大骂，然后从马上跳下来，一声呼哨之后，冲了出去。
<br/>　　被众人骂的狗血淋头的郁宁尴尬的站在开炮的位置，目瞪口呆的看着完全倒塌的城门楼子，‘啪’的给了自己一个嘴巴：“你大爷的……怎么放了那么多火药！”
<br/>　　原来高耸的城门楼子现在已经不见了，原本城门位置爆破之后应该是空洞一片的地方，此时堆满了碎砖烂瓦，以及折断的木料。
<br/>　　而应该在极短时间就冲进城的两个‘獠牙’中队此时正在锋利的在瓦砾堆上爬着，当然，爬的时候嘴里少不得骂骂咧咧的数落郁宁不是个东西。
<br/>　　可是有什么办法呢，塌都塌了，谁知道龟兹城看着坚固，实际上就是银样蜡枪头，中看不中用，四包炸药就给炸成了这个样子，也不知道那帮混蛋到底把炸药放到了什么位置，怎么会有这样的效果。
<br/>　　西域军团爬瓦砾堆爬的郁闷，城里的龟兹军和于阗军则更是郁闷，原本挺好的城楼楼子，突然之间就没了，变成了一堆瓦砾，让他们本就有些惶恐的情绪雪上加霜，死了老子娘一样在营地里到处乱窜。
<br/>　　李元昌脸色惨白的带着五百左右的于阗禁军疯了一样的在王宫外面砸着宫门：“国主，大事不好了，快点开门，开门啊！”
<br/>　　“汉王殿下，国主说了任何人不见，再说现在外面乱的很，请恕臣不能给您开门。”龟兹禁宫统领还算是镇定，听着外面混乱的嘈杂的声音，心中虽然紧张，但依旧牢记着自己的使命。
<br/>　　“不见个鬼，再不见我们全都要人头落地了，你去告诉他，马上出来，否则别怪老子言之不预。”李元昌一边指示着手下疯狂的砸门，一边愤怒的吼着。
<br/>　　没办法，外面太乱了，所有人都在争相逃命，哭爹喊娘的什么声音都有，不大点声王宫里面的禁军统领根本就听不到。
<br/>　　古人大都迷信，西域这帮家伙更是如此，国王往往总是会通过一些神神鬼鬼的东西来迷惑和忽悠老百姓，久而久之，西域对鬼神之类的敬畏远胜中原大地。
<br/>　　所以一声惊天动地的巨响之后塌掉的城门楼子成了压倒龟兹国的最后一根稻草，不管是当兵的还是老百姓都相信，一定是他们和大唐对抗引发了天谴，所以发会降下神罚，把城门给砸烂掉了。
<br/>　　不过龟兹城中的乱局对李元昌来说是末日，但对于费了半天力气爬进城来的伍登与席二愣来说，那可就是天赐良机。
<br/>　　慌乱的人群中，两人借着机会各自抢到一匹马，也不管是不是能不用来作战，翻身骑上之后手中武器圆了就向着龟兹军队人多的地方冲了过去。
<br/>　　城中已经乱成了一团，根本没有时间和机会去问军营在什么位置，往人多的地方杀成了两人唯一的选择
<br/>　　“杀……，大唐万胜！”唯恐天下不乱的两个家伙一边屠杀着看到的一切目标，一边兴奋的喊着，把城中的混乱无限制的扩大着。
<br/>　　但凡是些名人，多多少少都有些人来疯的本质，越是混乱的情况，这帮家伙就越是疯，加上杀入敌国国都，屠城灭国的荣耀加身之下，这些大唐武疯子们更是疯的厉害。
<br/>　　跟在他们身后爬来的‘獠牙’有不少人也乘机抢了一些马匹，当然，也有一些抢到了牛，不过管它是什么呢，只要是四条腿能跑的，在屁股上划一刀之后，跑的都挺快。
<br/>　　乱，整个龟兹城除了乱字已经不需要其它的形容词，看着无数的大唐‘獠牙’杀进城中，城中乱局更甚：
<br/>　　“大军杀进城啦……快跑啊！”
<br/>　　“是那天的杀人狂魔，救命，不要让他过来！”
<br/>　　各种各样奇葩的喊声在城里蔓延，很快便传到了王宫宫门附近的位置。
<br/>　　“你大爷的，你自己守着门吧，明年今天老子会给你们国主上柱香的。”李元昌紧张的向着声音传来的看方向了一眼，再也顾不上砸门，也不再想见龟兹国主，带着身边的五百来人怒嗓着仓惶而去。
<br/>　　“怎么办？统领，要不要通知国主？”龟兹禁军统领身边的一个小兵兵，趴在宫墙上向外望着，看到李元昌离开不禁回头问道。
<br/>　　“要通知你去通知，老子不陪了。”禁军统领指挥着手下人将宫门打开，瞪了小兵兵一眼：“不想死的都跟老子走，趁着唐军还没有杀到，能逃一个是一个！”
<br/>　　敢情这段早就已经看出苗头不对，不想去通知龟兹国主只是想要找机会逃走而已，如跑去通知了龟兹国主，那岂不是走不了了。
<br/>　　漆黑的夜里，虽然说不上伸手不见五指，但是十余步之外想要看清是什么人却是一点可能都没有，在这样的情况下，就算是有火把照明，能见度也不会好上多少，到处乱窜的人群使得一些有心抵抗一下的龟兹军根本分辨不清那些是自己人，哪些是唐军。
<br/>　　况且兵败如山倒之下，那些拼命逃窜的散兵游勇哪里还管什么自己人不自己人，现在只要是个人敢阻拦他们逃命，立刻就是兜头一刀先砍死再说。命都快要保不住了，还哪有什么心思分辨是不是自己的，是不是军官，从另一个方向逃出城才是最要的。
<br/>　　龟兹国主实际上早就醒了，城门楼子被炸塌的动静实在太大，几乎全城都被惊醒，国主自然也不例外。
<br/>　　“去，去看看怎么回事，别外把禁军统领给孤叫来。”听着宫外隐隐传来的哭喊声，龟兹国主皱眉叫来了宫中侍卫吩咐道。
<br/>　　“国，国主，东面的城门楼子塌了，统领……统领也已经跑了，现在宫里乱成一团，所有人都在准备逃命。”侍卫苦着脸说道。
<br/>　　同时他也很想说，‘如果不是您叫我，我也走了’，不过在看到龟兹国主难看的脸色之后，他明智的闭上了嘴，决定还是不要自找麻烦。
<br/>(本章完)
