　　第245章
<br/>　　程大雷已被这一幕搞懵，眼前密密麻麻汇聚有四五百人，都是绿林豪横人物。
<br/>　　这些人手里握着兵器，一时吵得吐沫横飞，眼看就要当场动起手来。
<br/>　　“程当家自然要去我们风云寨，我们一切都已备好。”
<br/>　　“难道只有风云寨有准备，我们乱石寨就没有准备么。”
<br/>　　“呵呵，我们三狼寨都已经杀好猪了。”
<br/>　　程大雷和徐神机对视一眼，都摸不准眼前情况，不知这些人是好意还是恶意，他眼神的意思是，让徐神机找个借口，几人走为上策。
<br/>　　徐神机立刻领悟，冲众人压了压手。
<br/>　　“诸位静言，某正是那神机妙算徐神机，既然我们来了此地，各家各寨都是要去的，都要去的。”
<br/>　　“……”程大雷。
<br/>　　青牛山一战后，程大雷隐姓埋名，杨龙停被捧为帝国战神，当然，这属于官方消息。而在江湖道，对这个消息是不认可的。
<br/>　　当时，程大雷为躲避杨龙停的追杀，散去蛤蟆寨弟兄。这些人浪迹天涯，散落四方。如他们，不管到了那儿，都不可能幡然醒悟，开始做良民，依旧是做山贼的居多。
<br/>　　他们无论加入那个山寨，说来说去，总要提到青牛山一战，有青牛山一战，自然不会不提到程大雷。于是一传十，十传百，传遍北方各地，大家都知道，程大雷以一座破山寨，挡住戎族精兵，杀敌十万。
<br/>　　而这次程大雷往西北去，葫芦山的范桐也传下话来，说程当家要从西北道上过，各家各寨都要做好接待工作。
<br/>　　其实，在程大雷从葫芦山离开时，这一路上都有人应着，只是程大雷恰好没遇上而已。
<br/>　　一群人欢呼雀跃，恨不得敲锣打鼓，程大雷先去的还是风云寨，其他山寨扼腕叹息，看这架势，程大雷没有十天半个月，绝对走不出峡山。
<br/>　　而此时还处在云里雾里的程大雷并没有注意到，在不远处的山坳上，一棵大树遮蔽的阴影下站着一个人。
<br/>　　看到程大雷被迎上山这一幕，他咬着牙，嘴里骂了一句脏话。
<br/>　　“大人的命令，绝不能让此人活着走到琴川。”身边一人道。
<br/>　　“明白。”此人眼里浮现一道狠意。
<br/>　　在他脚立的山坡下，伏着一排人，黑衣黑甲，静若山石。
<br/>　　……
<br/>　　青牛山蛤蟆寨的主人，击败北蛮部的胜利者，屠杀帝国战神的凶手程寨主大雷，在峡山受到热烈欢迎，与峡山各寨举行了多方会谈，会谈在友好活泼的气氛中进行。
<br/>　　在会谈后的晚宴上，程寨主对大家的热情招待表示了谢意，并对峡山的特色小吃赞不绝口。
<br/>　　次日，在风云寨寨主黄飞虎的陪同下，程寨主参观了风云寨的演武场，厨房，铁匠铺，并提出宝贵意见。
<br/>　　乱石寨，三狼寨等山寨都希望程寨主可以到他们山寨参观，把先进经验留下来，把艰苦奋斗的精神传下去。
<br/>　　屋子里，徐神机几个大眼瞪小眼，程大雷手托下巴，露出生无可恋的表情。
<br/>　　自从程大雷来到峡山后，可以说几乎没闲着，各家各寨争着宴请程大雷，一开始是满足了程大雷的虚荣心，但一天一天接着如此过，程大雷难免有些掩卷。
<br/>　　可不去还不行，你吃了某某的，不吃我的，莫非是看不起我么，还是不拿我当兄弟。
<br/>　　除此之外，程大雷还应邀参观了各家山寨，程大雷也是抱着学习的目的，看山寨的防御有什么自己可以借鉴的。过程中，这些人难免就会说：程寨主和戎族打过仗，对于程寨主的本事，我们都是服的，不如程寨主看看，我们这山寨建的哪里有问题……
<br/>　　程大雷还真能回答上来。他守城三个月，每天十二个时辰，想的就是一个字：守。
<br/>　　日日绞尽脑汁，废寝忘食，都琢磨着这个字，不知不觉，程大雷已成为擅守的高手。
<br/>　　还莫说亲自走过去看看，程大雷只需站在高处扫一眼，便能点出问题所在，皆是由浅入深，入木三分。
<br/>　　程大雷深刻体会到作为一个偶像，粉丝太多的烦恼。
<br/>　　如此一来，这些人对程大雷便更加佩服：果不愧能击退戎族，名不虚传，名不虚传呐。
<br/>　　“程寨主，歇着呢，正好您有空，快去我山寨看看，给我们指点指点。”胡八刀走进屋子。
<br/>　　程大雷满脸委屈，最后却还是无可奈何的站起。
<br/>　　“走吧。”
<br/>　　程大雷带着徐神机几人，峡山几家山寨的寨主都跟在后面。
<br/>　　“程当家，等去了乱世寨，也得去我们山寨看看。”
<br/>　　“上次程当家发现我们山寨后山有条小道，我正让人堵呢，幸亏被程当家发现，不然官府打过来我们就遭殃了。”
<br/>　　“诸位，我们已经在这里待了七八天了，是不是有什么事要程某人做？”程大雷骑在马上，有些没精打采。
<br/>　　“哈哈，实在瞒不过程寨主，眼前有件买卖，想让程寨主打头。”
<br/>　　“什么买卖啊，太小的我可不做。”此时大家正通过一道峡谷。
<br/>　　“抢亲！”
<br/>　　“这算的什么大买卖……”
<br/>　　程大雷忽地一顿，就看到峡谷上方一块大石被推动，轰然砸下来。
<br/>　　“散开！”
<br/>　　程大雷从马上跃起，附身在山壁上。幸亏程大雷提醒得早，其他几人才狼狈躲在峡谷底部的凹处。
<br/>　　秦蛮左手拎着徐灵儿，右肩扛着徐神机，大步离开落石处。
<br/>　　程大雷抬头往上看，只见峡谷上有几道人影，正将滚石往下推。
<br/>　　“XXX”
<br/>　　程大雷爆出一句国骂，身子忽然跃起，身无彩凤双飞翼使出，脚在山壁上一点，便是三五丈距离，此时他长袍飘动，翩然若仙。
<br/>　　峡谷下抱头鼠窜的几人呆呆看着这一幕，嘴里难免发出一声惊叹。
<br/>　　“果不愧是程当家呐。”
<br/>　　等落石静下来时，诸人都没有大碍，可有几匹马都被砸成肉酱，诸人倒抽一口凉气，若非程大雷警觉，后果不堪设想。
<br/>　　等几人绕路来到悬崖上时，看到上面的人已经被程大雷解决了。
<br/>　　他手里握着一柄剑，剑尖仍在滴血，面前倒了几具尸体，却还有两人活着，只是下巴都被程大雷摘了。
<br/>(本章完)
