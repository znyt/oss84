　　林峰跟着大部队回到了基地。
<br/>　　虽然这一次计划失败了，但也不是没有任何收获，至少也算是保留了革命火种，其次就是知道了许多军部之人，原本不知道的事情。
<br/>　　陆勇回到基地第一件事就是大刀阔斧的抓叛徒。
<br/>　　而可能是察觉到了风声，张家和王家的人居然都离开了特等区。
<br/>　　至于他们去了哪里，没有人知道，陆勇带着人来时，只是看到了人去楼空的大别墅。
<br/>　　上面产生了一场规模不小的地震，而下面的老百姓还是如往日一样，为一口吃的而折腰。
<br/>　　男的做苦力赚钱，给人当奴隶当狗，挥之即来呼之即去，有姿色的女的就不惜出卖自己的肉体，二十多岁的女人让七八十岁的老头子任意的冲刺，还要装出一副满足的笑容来让对方愉悦，这一切，为的只是一点吃的罢了。
<br/>　　至于林峰的店——新开的店，生意也极其的火爆。
<br/>　　不过店在今天被人给砸了，一群人把店内东西给砸的稀巴烂，带头的还是一个安保队的巡逻小队的小队长。
<br/>　　转眼，三天时间过去，林峰在基地内修炼，刚用大量晶石让自己实力突破到八阶巅峰就听到了这个坏消息，来到这里时候，吴三也被人打成猪头三了。
<br/>　　吴三实力可不算弱，四阶，在这个基地也是中上的水平，可是被人打的这么惨，很显然砸店的人不简单。
<br/>　　“你就是这家店的老板？你们店内有的东西是不准贩卖的，违反了市场管理法，所以你的店要被封了，或者你可以交三千颗晶石来花钱消灾。”
<br/>　　小队长扫了眼林峰呵呵一笑道。
<br/>　　“你先说说，我这店内有什么东西不准贩卖？”林峰淡然问道。
<br/>　　“毒品！”
<br/>　　小队长拿出了一小包的粉末。
<br/>　　“毒品！不管是现在还是之前都是不准贩卖的东西，封你店还是轻的。”小队长冷笑道。
<br/>　　“林峰，我绝对没有卖那个东西，这东西现在根本不赚钱，现在大家饭都吃不饱，谁特么去买这个玩意啊。”吴三立即辩解道。
<br/>　　林峰扫了对方一眼，的确，对方说的很有道理，饭都吃不饱的情况下，除了那些瘾君子谁会去想这买这东西，而且现在随随便便一个怪物皮毛赚的钱都很多了，没必要卖这些不入流的东西。虽然黑市上，毒品好像价格很高，这东西不仅仅可以当做毒品还可以当作医疗用品。
<br/>　　“哦？你的意思是我们王队长陷害你们了？”一个汉子忽然冷笑道。
<br/>　　“你又是谁？”
<br/>　　林峰问道。
<br/>　　“隔壁那家无敌小店就是我开的！！我叫李德！！”这穿着西装的年轻人冷笑道。
<br/>　　然后他上前一步小声道：“小子，实话告诉你吧，你这个店这几天抢了老子的生意，老子很不爽！所以要弄死你！别怀疑！别犹豫！老子有这个能力！因为我爹是巡逻小队第三小队的正队长！”
<br/>　　说着这汉子得意洋洋的看着林峰。他还特别强调了一个正字！
<br/>　　林峰总算是搞明白了，原来是个军二代啊。这军二代也开了个店。
<br/>　　“你做你的生意，我做我的生意，大家凭本事赚钱，没必要这样搞吧。”
<br/>　　林峰淡然道。
<br/>　　“呵呵，敢抢老子的生意，老子就是要让你店面倒闭！”李德冷笑道。
<br/>　　“让我倒闭？就你那个小队长的爹还有这能耐？”
<br/>　　林峰一副压根不在意的样子。
<br/>　　“难道这小子大有来历？”
<br/>　　李德看林峰有持无惧的样子，他也皱起眉头，不过不对啊，这家店开店时，他还特意注意过，没有来什么大人物啊，要知道现在末世了，但很多事情文化风俗还都是留着以前的那一套。
<br/>　　比如说开店了，会有人送花篮，上面写着谁谁谁送的，这都是秀自己多牛逼的一种体现。
<br/>　　像他这个店，因为他爹是个小队长的缘故，开业那天，几十个大头兵过来了，一些大帮派的领头人也都来送花篮。
<br/>　　这一看就知道是牛逼人物的地盘。
<br/>　　可是这个店开业时，什么没有，这很显然就是寻常人开的了。无人问津。
<br/>　　“和他废话什么，证据确凿，直接给他查封。”
<br/>　　王队长大手一挥立即几个衣服上写着宪兵二字的小兵就要来封店。
<br/>　　“我看谁敢！”
<br/>　　林峰上前一步拦住了这群宪兵。
<br/>　　“呦呵，你还敢抗拒执法不成？！”
<br/>　　王队长刷的一下拔出了手枪对准了林峰。
<br/>　　其余宪兵也都拿出了武器虎视眈眈的看着林峰。
<br/>　　林峰闻言目光微微眯起，什么地方都有败类，哪怕是军部也不例外。
<br/>　　“你在找死你知道吗？”
<br/>　　“呦呵？还威胁我？怎么，你莫非还敢打我不成？知不知道，在基地内对一个现役军官动手是什么后果？枪毙你都是轻的！”
<br/>　　王队长得意洋洋道。他身上的这一件衣服，那就是最好的保护伞。为了搞到这件衣服，他付出了多少心血，拍了多少人的马匹。
<br/>　　“小子，有本事你就打啊。”
<br/>　　“哈哈，怂包一个，有能耐揍我啊。”
<br/>　　王队长撅着屁股在那里得意洋洋道。几个宪兵也都哈哈大笑。
<br/>　　砰！
<br/>　　然而下一刻。
<br/>　　他们就笑不出来了。
<br/>　　他们的王队长直接被林峰一脚踹飞了，从店里面踹到了店外面在街道上面趴着来了一招狗吃屎。
<br/>　　“你敢对队长动手？”
<br/>　　两个宪兵满脸诧然。他们万万没有想到敢有人在基地内对他们出手，这是非常不可思议的事情，也是死罪。
<br/>　　砰砰。
<br/>　　然而下一刻他们两个也飞了出去。
<br/>　　“你，你死定了，你敢对王队长出手，你小子完蛋了！”
<br/>　　李德也是面色一变。
<br/>　　然后下一刻他也被丢了出去。几个人就好像沙包一样并列的扒在地上。极其的狼狈。
<br/>　　“要不是老子这店刚开一天，不想见血，想要图个吉利，你们今天几个人的脑袋就得搁在这里了。”
<br/>　　林峰冷笑道。
<br/>　　“你……”
<br/>　　王队长又气又怒的站起身看到林峰眼神他吓了一跳。
<br/>　　“你给我等着！走！”
<br/>　　他立即灰溜溜的离开了。
<br/>　　“小子，你死定了！”
<br/>　　李德留下一句狠话立即跑了。
<br/>　　“不会有麻烦吧？”
<br/>　　吴三道。
<br/>　　“没事，只要你小子真的没有卖这些垃圾东西，那么就屁事没有，我们是站在理这一边，他要是再敢来，就没这么简单了。”
<br/>　　林峰压根不在意，区区一个巡逻队副队长能够把他林峰的店面给整倒闭了？那他林峰这几天给军部忙活的事情也算是白忙活了。
<br/>　　“林峰你放心吧，虽然我末世前算是个游走在底线附近的商人，开过洗头房也开过网吧，网吧虽然说不让未成年进入，但我也依然让一些初中生来上网，但是毒品这些东西我是从来不碰的。”吴三一本正经的说道。
<br/>　　林峰点了点头，正在说话的时候。
<br/>　　外面传来了运输车的声音。
<br/>　　然后稀里哗啦就是二十多个宪兵全副武装的从车子上跳下来。
<br/>　　顿时吸引了一群看热闹的人。
