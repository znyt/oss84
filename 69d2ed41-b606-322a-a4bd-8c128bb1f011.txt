　　慕容飞还是到了要返回部队的时间，心中有再多的不舍，也必须要分离。
<br/>　　杭山市国际机场登机安检口前，慕容飞和佟千雪依依不舍的分别。
<br/>　　“这次回去还有行动吗？”佟千雪对慕容飞问道。
<br/>　　慕容飞为难的咧嘴道：“你这个问题实在太难回答了，这有没有行动也不是我说得算。”
<br/>　　“不管你是去沙漠、去极地还是去危机四伏的南美洲雨林，你都要记住，你现在可不是一个人了，一定要保护好自己。”佟千雪替慕容飞整了整衣服，说道：“平时记得刮胡子，剃须刀给你放包里了，别没事拿刀去刮，要是把脸划破，破了相，小心我就不要你了。”
<br/>　　“咱们现在可是军婚，你要不要，这可不是你说得算。”慕容飞嘿嘿一笑说道。
<br/>　　“我的确是没办法主动提出离婚，但是我可以扼杀你那些坏想法……”佟千雪对着慕容飞露出一个警告的眼神。
<br/>　　慕容飞瞬间就怂了，他现在能明白为什么罗昊看到云熙，就跟老鼠见到猫似得了。这女人的杀手锏都是一样一样的啊。
<br/>　　慕容飞看了眼时间，距离过安检登机还有四十五分钟，低头在佟千雪额头上亲了一下后，转身朝着安检口走去。慕容飞来的时候，两手空空，什么东西都没有带，回去的时候，佟千雪给他准备了一个背包的东西，里面都是一些日用品和吃的东西。尽管龙隐部队的伙食标准都比得上飞行员的伙食标准了，可佟千雪还是固执的给慕容飞准备了很多东西。
<br/>　　佟千雪站在安检口外，一直看着慕容飞的身影从自己视野中消失才转身离开。
<br/>　　虽然自己每次叮嘱慕容飞在外行动时要小心，慕容飞每次都告诉自己，他是狙击手那是距离敌人最远的战斗位置；但是佟千雪何尝不知道，这是慕容飞在安慰自己。
<br/>　　狙击手的确是距离敌人最远的战斗单位，可同样的，也是最容易遭到炮火覆盖的战斗单位。从狙击步枪枪响，到敌人的炮弹落下，中间的时间间隔可能只有几秒钟的时间。
<br/>　　……
<br/>　　龙隐部队，教官宿舍楼。
<br/>　　云熙正在子午的房间里整理着他的遗物。
<br/>　　子午的葬礼在赵天宇等人返回后就进行了，是赵国邦亲自主持的，墓碑上的字也是赵国邦亲自题写的，子午在K国被炸得尸骨无存，所以在他的墓穴里只放了一套他平时很少穿的常服，立下了一个衣冠冢。
<br/>　　子午的宿舍一直被锁着，里面的东西也保持着原来的样子没有人进去动过，赵天宇把子午宿舍门的钥匙交给了云熙，让她去整理子午的遗物。
<br/>　　云熙好几次拿着钥匙走到了子午宿舍前，想到自己刚来龙隐部队时，一直缠着子午教自己各种新技术，而子午被自己缠得无可奈何最终只能选择妥协的画面，眼眶发红，失去了用钥匙打开宿舍门的勇气，转身离开。那段时间，是自己哥哥离开后，云熙最快乐的时间，她可以像一个孩子一样跟子午无理取闹，子午满是无奈但是却又不忍拒绝，就像是一个父亲面对自己淘气调皮的女儿一样。
<br/>　　就这样，子午的宿舍就这么被关着，关了有两个礼拜，云熙在罗昊的陪伴下，用赵天宇交给她的钥匙打开了子午宿舍的门。
<br/>　　子午宿舍中的摆设，还是跟云熙第一次来到他宿舍时一样，几乎没有任何的改变，书桌上摆放着子午平时所使用的电脑，键盘上的字母已经被磨花看不清楚。
<br/>　　“键盘就是黑客战斗的武器。”云熙走到键盘前，用手轻轻抚摸过键盘对罗昊说道：“这样一块看上去已经用了好几年的键盘，实际上，最多只用了不超过两个月的时间，我曾经看到过老师一个礼拜敲坏过两块键盘。以前这里有一个纸箱子，里面全都是老师用坏的键盘。”
<br/>　　很久没有来子午的宿舍，可是云熙还是能清楚的记得子午东西摆放的位置。
<br/>　　罗昊把折叠起来的纸箱打开，底部贴上两层封箱带，然后从云熙手里接过她递给自己的各种专业书籍，整齐的摆放在纸箱中。
<br/>　　云熙伸手去拿书架最上层的书籍时，触碰到了桌子上的鼠标，电脑机箱发出一阵轻响，原本暗着的显示器突然亮了起来。
<br/>　　子午桌子上这台电脑并没有被关闭，而是处于休眠状态，因为被云熙触碰到鼠标的关系，使得电脑又重新工作。
<br/>　　电脑界面被锁住了，需要密码才能打开。云熙站在桌子前想了想，在键盘上敲下一组密码，顺利通过了验证。
<br/>　　“老师的密码是他进入龙隐部队的时间。”云熙拉开椅子坐了下来，移动着鼠标对站在旁边的罗昊说道：“老师说，从他加入龙隐部队的那天前，曾经的GHOST就‘死了’，取而代之的是子午。过去，GHOST是以自己为中心，但子午却是以国家为中心。”
<br/>　　云熙移动着鼠标，不断点击着，这时她脸上突然露出诧异之色，低头朝着放在桌子下的机箱看去，在一个USB接口上发现了一个硬盘！
<br/>　　这个硬盘云熙熟悉，罗昊也同样熟悉！这是罗昊从“暗月”在火山岛上那个试验基地中逃出来时，从仙岛夫电脑上拆下来的硬盘。因为仙岛夫在离开时启动删除程序，所以硬盘中的大部分数据都遭到了破坏，虽然罗昊最终没有让删除程序把硬盘中的数据彻底粉碎，但是原来存储在硬盘中的东西，还是被破坏的很厉害。
<br/>　　罗昊被蛰龙从火山岛上救出来，陷入昏迷的这段时间中，蛰龙把这个硬盘寄回了国内，到了子午的手上。
<br/>　　子午恢复了硬盘中的一些东西，可是硬盘中最重要的那部分数据始终没有被恢复，时间一长，大家都忘记了这个硬盘的存在。云熙没想到，自己的老师还在尝试着对硬盘中遭到破坏的数据进行恢复，而且还有了突破性的进展。
<br/>　　云熙盯着电脑，双手飞快的在键盘上敲打着，一时间，子午宿舍中只剩下了键盘敲打的声响。
